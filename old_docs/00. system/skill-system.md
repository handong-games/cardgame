## Part 1: 왜 만드는가 (Why)

### 1.1 기획 배경

동전 던지기로 획득한 코스트를 소비하는 행동 수단이 필요하다. 카드 시스템은 드로우 과정이 추가되어 턴 시작이 느려지고, 드로우 운까지 겹치면 이중 확률로 행동 불가 상황이 발생한다. 스킬은 항상 접근 가능하여 동전 결과만으로 즉시 판단할 수 있고, 획득/교체를 통해 자신만의 전략을 구성하는 성장 경험을 제공한다.

### 1.2 해결할 문제

| 문제 | 현재 상황 | 해결 방향 |
|------|----------|----------|
| **코인 소비 수단 부재** | 동전 던지기로 코스트를 얻지만 사용할 행동이 없음 | 스킬로 코인을 소비하는 행동 수단 제공 |
| **카드 드로우의 이중 확률** | 코인 확률 + 드로우 확률이 겹쳐 행동 불가 발생 | 스킬은 항상 접근 가능, 코인 결과만으로 즉시 판단 |
| **전략 구성 경험** | 고정된 행동 세트로 플레이어 개성 부재 | 스킬 획득/교체를 통해 자신만의 전략 구성 |

> 핵심: 스킬 시스템은 플레이어가 코인을 얻었을 때 **행동하는 수단**을 제공하며, 그 행동은 플레이어 **전략의 바탕**이 된다.

### 1.3 기대 효과 & 리스크

| 구분 | 내용 | 대응 방안 |
|------|------|----------|
| **기대 효과** | 코스트로 스킬을 사용하며 전투 행동의 즉각적 피드백 | - |
| **기대 효과** | 다양한 몬스터에 맞춰 스킬을 습득하고 슬롯에 배치하는 빌드 구성 경험 | - |
| **기대 효과** | 코인 확률에 대비한 스킬 조합 완성으로 전략적 성장감 | - |

## Part 2: 무엇을 만드는가 (What)

### 2.1 한 줄 정의

> 스킬 시스템은 **코인 소비를 통한 전투 행동**을 담당하며, **클래스별 고유 스킬과 슬롯 기반 빌드 구성**을 통해 **몬스터에 대응하는 전략적 성장**을 제공한다.

### 2.2 감정 설계

| 순간 | 목표 감정 | 설계 요소 |
|------|----------|----------|
| 코인 결과 확인 후 | **즉각적 판단** | 사용 가능/불가 스킬을 시각적으로 즉시 구분 표시 |
| 사용 가능 스킬 호버 | **선택의 기대감** | 스킬 아이콘이 위로 올라오는 애니메이션 (카드 호버처럼), 불가 스킬은 무반응 |
| 스킬 아이콘 클릭 | **전략적 확신** | 프리뷰 시스템으로 예상 효과 표시 (예: 적 HP바에 예상 데미지 감소 표시). 프리뷰 시스템 상세는 별도 기획 |
| 대상 지목 | **행동의 몰입감** | 스킬 아이콘(시작점) → 마우스(끝점) 화살표 표시, 몬스터 호버 시 타겟 하이라이트 UI |

### 2.3 시스템 구조

```
코인 던지기 결과 (앞면/뒷면 개수 확정)
    |
[스킬 활성화 판정] 각 스킬의 코스트 vs 보유 코인 비교
    |
[스킬 선택] 사용 가능한 스킬 중 선택
    |
[대상 지목] 스킬 대상 유형에 따라 분기
    |   ├─ 단일 대상: 타겟 선택 (화살표 UI)
    |   ├─ 아군 전체 / 적 전체 / 전체: 자동 적용 (추후 확장)
    |
[효과 적용] 데미지/방어/버프 등 처리
    |
[코인 차감] 사용한 코스트만큼 차감
    |
[반복] 남은 코인으로 추가 스킬 사용 가능
    |
[턴 종료] 잔여 코인 소멸
```

### 2.4 핵심 규칙

| 규칙 | 내용 | 예외 |
|------|------|------|
| **코스트 유형** | 앞면/뒷면/복합(앞+뒤) 코스트 가능 | - |
| **기본 스킬** | 클래스별 3개 제공 (공격, 방어, 뒷면 스킬). 교체 가능 | - |
| **코스트 방향** | 전사=앞면 위주, 도적=뒷면 위주, 마법사=복합(앞+뒤) | - |
| **기본 뒷면 스킬** | 전사: 피 회복. 다른 클래스는 추후 설계 | - |
| **슬롯 시작** | 4개로 시작 (기본 스킬 3개 + 빈 슬롯 1개) | - |
| **슬롯 확장** | 마을/상점에서 구매 또는 조건 달성으로 최대 6개까지 확장 | - |
| **턴당 사용 횟수** | 스킬별로 턴당 사용 횟수가 다름 | - |
| **스킬 교체** | 보상으로 획득한 스킬이 기존 스킬 대체 | 보상 시스템 기획서에서 상세 |
| **0코스트 없음** | 모든 스킬은 최소 1개 이상의 코인 필요 | - |

### 2.5 주요 수치 기준

| 항목 | 기준값 | 범위 | 비고 |
|------|--------|------|------|
| 기본 스킬 수 | 3개 | - | 공격, 방어, 뒷면 스킬 (전사: 피 회복) |
| 시작 슬롯 | 4개 | - | 기본 스킬 3개 + 빈 슬롯 1개 |
| 최대 슬롯 | 6개 | - | 마을/상점에서 확장 |
| 스킬 코스트 | 1~5개 | - | 앞면/뒷면/복합 |
| 시작 코인 | 3개 | - | 동전 던지기 기획서 참조 |
