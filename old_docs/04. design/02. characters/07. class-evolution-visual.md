# 클래스 전직 비주얼 가이드

## 개요

이 문서는 캐릭터 전직(Class Evolution) 시의 시각적 연출을 정의합니다. 전직은 게임의 중요한 이벤트로, 기억에 남는 특별한 연출이 필요합니다.

---

## 전직 시스템 개요

### 전직 트리
```
전직 경로:
├── 전사 (Warrior)
│   ├── → 버서커 (Berserker) [공격 특화]
│   └── → 검사 (Swordsman) [기술 특화]
├── 팔라딘 (Paladin)
│   ├── → 성기사 (Holy Knight) [방어 특화]
│   └── → 사제 (Priest) [힐링 특화]
└── (추가 클래스 예정)
```

### 전직 조건
| 조건 유형 | 내용 |
|-----------|------|
| 레벨 요구 | Lv. 15+ |
| 특수 조건 | 전직별 퀘스트 완료 |
| 선택 | 2개 경로 중 1개 선택 |

---

## 전직 연출 시퀀스

### 전체 타임라인 (5초)
```
전직 연출 구조:
├── Phase 1: 집중 (0-1000ms)
├── Phase 2: 고양 (1000-2000ms)
├── Phase 3: 변신 (2000-3000ms)
├── Phase 4: 등장 (3000-4000ms)
└── Phase 5: 완료 (4000-5000ms)
```

---

## Phase 1: 집중 (Concentration)

### 시간: 0-1000ms

### 시각 효과
```
집중 페이즈:
├── 배경:
│   ├── 서서히 어두워짐 (100% → 70%)
│   ├── 주변 배경 블러 처리
│   └── 캐릭터에 스포트라이트
├── 캐릭터:
│   ├── 중앙으로 이동 (300ms)
│   ├── 눈 감고 집중하는 포즈
│   ├── 미세하게 떠오름 (5px)
│   └── 몸 주변 작은 반짝임 시작
└── 파티클:
    ├── 바깥에서 안으로 모이는 빛 (20개)
    ├── 색상: 클래스 테마 색상
    └── 속도: 천천히 → 빠르게
```

### 이펙트 상세
| 요소 | 시작 | 끝 | 이징 |
|------|------|-----|------|
| 배경 밝기 | 100% | 70% | Ease Out |
| 블러 정도 | 0px | 8px | Ease Out |
| 캐릭터 위치 | 원래 | 중앙 | Ease Out Back |
| 부유 높이 | 0px | 5px | Ease In Out |

### 사운드 큐
| 시간 | 사운드 | 설명 |
|------|--------|------|
| 0ms | 차임 시작 | 낮은 종소리 |
| 500ms | 바람 소리 | 바람이 모이는 느낌 |

---

## Phase 2: 고양 (Ascension)

### 시간: 1000-2000ms

### 시각 효과
```
고양 페이즈:
├── 배경:
│   ├── 더 어두워짐 (70% → 50%)
│   └── 색조 변화 (클래스 테마색 틴트)
├── 캐릭터:
│   ├── 더 높이 떠오름 (5px → 20px)
│   ├── 몸에서 빛 발산 시작
│   ├── 의상/장비 빛나기 시작
│   └── 머리카락 위로 흔들림
├── 파티클:
│   ├── 모인 빛이 몸으로 흡수
│   ├── 에너지 링 확장 (3회)
│   └── 주변에 떠다니는 별 증가
└── 이펙트:
    ├── 캐릭터 실루엣 글로우
    └── 바닥에 마법진 나타남
```

### 마법진 디자인
```
전직 마법진:
├── 형태: 원형 + 문양
├── 크기: 캐릭터 기준 200% 너비
├── 색상: 클래스 테마 색상
├── 회전: 시계 방향 천천히
├── 투명도: 0% → 80%
└── 스타일: 손그림 느낌의 선
```

### 사운드 큐
| 시간 | 사운드 | 설명 |
|------|--------|------|
| 1000ms | 에너지 모임 | 상승하는 사운드 |
| 1500ms | 마법진 등장 | 신비로운 종소리 |

---

## Phase 3: 변신 (Transformation)

### 시간: 2000-3000ms

### 시각 효과
```
변신 페이즈:
├── 플래시:
│   ├── 0ms: 강한 빛 폭발 (화면 80% 밝기)
│   ├── 색상: 클래스 테마 + 흰색
│   └── 지속: 300ms
├── 캐릭터:
│   ├── 플래시 동안 실루엣만 보임
│   ├── 실루엣 내에서 의상 변화
│   └── 새 장비/의상 형태로 변환
├── 파티클:
│   ├── 폭발적 별/빛 파티클 (50개)
│   ├── 방사형으로 퍼짐
│   └── 새 클래스 색상
└── 마법진:
    ├── 최고 밝기
    └── 회전 속도 증가
```

### 실루엣 변환 상세
```
실루엣 변환 시퀀스:
├── 0-100ms: 강한 빛으로 형태 가림
├── 100-200ms: 실루엣 안에서 변화 시작
│   ├── 갑옷 형태 변화
│   ├── 무기 변화
│   └── 머리카락/장식 변화
├── 200-500ms: 점진적 변화 완료
└── 500-1000ms: 빛 감소, 새 모습 공개 준비
```

### 사운드 큐
| 시간 | 사운드 | 설명 |
|------|--------|------|
| 2000ms | 빛 폭발 | 밝고 따뜻한 폭발음 |
| 2200ms | 변신 소리 | 마법 변환 소리 |
| 2800ms | 완료 차임 | 높은 종소리 |

---

## Phase 4: 등장 (Reveal)

### 시간: 3000-4000ms

### 시각 효과
```
등장 페이즈:
├── 배경:
│   ├── 빛 감소 (80% → 정상)
│   └── 블러 해제
├── 캐릭터:
│   ├── 새 모습 공개
│   ├── 영웅적 포즈 (팔 벌리거나 무기 들기)
│   ├── 천천히 착지 (20px → 0px)
│   └── 망토/머리카락 펄럭임
├── 파티클:
│   ├── 주변에 떠다니는 반짝임
│   ├── 바닥에서 올라오는 빛
│   └── 새 클래스 특유 파티클
└── 오라:
    ├── 새 클래스 테마 오라 상시
    └── 투명도: 0% → 50%
```

### 포즈 옵션
| 클래스 | 등장 포즈 | 특징 |
|--------|----------|------|
| 버서커 | 도끼 들어올림 | 힘차고 역동적 |
| 검사 | 검 앞으로 | 우아하고 기품 있는 |
| 성기사 | 방패+검 | 수호자다운 당당함 |
| 사제 | 지팡이 들기 | 평화롭고 신성한 |

### 사운드 큐
| 시간 | 사운드 | 설명 |
|------|--------|------|
| 3000ms | 착지 소리 | 부드러운 착지 |
| 3500ms | 무기 소리 | 무기 휘두르는 소리 |

---

## Phase 5: 완료 (Complete)

### 시간: 4000-5000ms

### 시각 효과
```
완료 페이즈:
├── 배경:
│   ├── 완전히 정상화
│   └── 전투 화면으로 복귀 준비
├── 캐릭터:
│   ├── 기본 대기 포즈로 전환
│   ├── 새 오라 유지
│   └── 미소 또는 자신감 표정
├── UI:
│   ├── "전직 완료!" 텍스트 팝업
│   ├── 새 클래스 이름 표시
│   └── 획득 스킬 미리보기 (선택)
└── 파티클:
    ├── 축하 반짝임 마무리
    └── 점점 감소하여 종료
```

### 텍스트 연출
```
"전직 완료!" 텍스트:
├── 위치: 화면 중앙 상단
├── 폰트: 제목 폰트, Bold
├── 크기: 48px → 64px → 48px
├── 색상: Warm Gold (#FFD700)
├── 외곽선: Dark Brown (#3A2818)
├── 애니메이션:
│   ├── 스케일 바운스 (Ease Out Elastic)
│   └── 글로우 펄스
└── 지속: 1500ms 후 페이드 아웃
```

### 사운드 큐
| 시간 | 사운드 | 설명 |
|------|--------|------|
| 4000ms | 팡파레 | 축하 사운드 |
| 4500ms | 승리 테마 | 짧은 멜로디 |

## 클래스별 고유 연출

### 전사 → 버서커
```
버서커 전직 고유 연출:
├── 테마 색상: Deep Red (#B22222)
├── 파티클: 불꽃 (따뜻한 주황)
├── 마법진: 불꽃 문양
├── 플래시 색상: 붉은 오렌지
├── 등장 포즈: 도끼 위로 들어올리며 포효
├── 특수 효과:
│   ├── 변신 중 눈이 붉게 빛남
│   ├── 화염 오라 상승
│   └── 야수같은 이빨 살짝 보임
└── 사운드: 야수 포효 + 화염 소리
```

### 전사 → 검사
```
검사 전직 고유 연출:
├── 테마 색상: Silver (#C0C0C0)
├── 파티클: 검기 (은빛 바람)
├── 마법진: 검 문양
├── 플래시 색상: 은빛 + 파랑
├── 등장 포즈: 우아하게 검 휘두름
├── 특수 효과:
│   ├── 검에서 빛 방출
│   ├── 공기 중 검기 궤적
│   └── 정돈된 머리카락
└── 사운드: 검 휘두르는 소리 + 바람
```

### 팔라딘 → 성기사
```
성기사 전직 고유 연출:
├── 테마 색상: Bright Gold (#FFD700)
├── 파티클: 신성 빛 (금색 + 흰색)
├── 마법진: 태양 문양
├── 플래시 색상: 순백 + 금색
├── 등장 포즈: 검+방패 들고 수호 자세
├── 특수 효과:
│   ├── 머리 위 후광
│   ├── 작은 날개 이펙트
│   └── 신성한 바람
└── 사운드: 천사 합창 + 종소리
```

### 팔라딘 → 사제
```
사제 전직 고유 연출:
├── 테마 색상: Cream + Green (#90EE90)
├── 파티클: 치유 빛 (녹색 + 분홍)
├── 마법진: 생명 문양 (나뭇잎)
├── 플래시 색상: 녹색 + 금색
├── 등장 포즈: 지팡이 들고 축복 자세
├── 특수 효과:
│   ├── 주변에 꽃잎/잎 흩날림
│   ├── 치유 오라 상승
│   └── 평화로운 표정
└── 사운드: 자연 소리 + 차분한 멜로디
```

## 전후 비교 화면

### 비교 UI 레이아웃
```
전직 비교 화면:
┌─────────────────────────────────────────┐
│         [ 전직 완료! ]                   │
├───────────────────┬─────────────────────┤
│                   │                     │
│    [이전 모습]    │    [새로운 모습]    │
│    (흑백/희미)    │    (컬러/빛나는)    │
│                   │                     │
├───────────────────┴─────────────────────┤
│  전사 ────────→ 버서커                  │
├─────────────────────────────────────────┤
│  새로운 능력:                            │
│  • 광폭화 (분노 시 데미지 증가)         │
│  • 피의 일격 (체력 소모하여 강공격)     │
├─────────────────────────────────────────┤
│           [ 확인 ]                       │
└─────────────────────────────────────────┘
```

### 비교 애니메이션
```
비교 화면 애니메이션:
├── 이전 모습: 왼쪽에서 슬라이드 인
├── 화살표: 가운데서 펄스 + 이동
├── 새 모습: 오른쪽에서 슬라이드 인 + 빛남
├── 텍스트: 순차적 페이드 인
└── 버튼: 마지막에 바운스 등장
```

## 기술적 고려사항

### 성능 최적화
```
최적화 규칙:
├── 파티클 최대 100개
├── 블러는 GPU 가속 사용
├── 배경 어둡게 처리로 렌더링 부하 감소
├── 전직 연출 중 다른 UI 비활성화
└── 저사양 옵션: 파티클 50%, 블러 제거
```

### 스킵 옵션
```
스킵 처리:
├── 터치/클릭으로 스킵 가능
├── 스킵 시 Phase 5로 바로 이동
├── 필수 정보는 스킵 후에도 표시
└── 첫 번째 전직은 스킵 불가 (권장)
```
