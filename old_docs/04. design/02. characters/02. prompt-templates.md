# 캐릭터 AI 프롬프트 템플릿

## 개요

이 문서는 Gemini API를 사용하여 캐릭터 이미지를 생성하기 위한 검증된 프롬프트 템플릿을 제공합니다.

---

## 프롬프트 구조

### 4레이어 시스템
```
[Layer 1] 스타일 베이스 (모든 캐릭터 공통)
    ↓
[Layer 2] 클래스 특성 (전사/팔라딘/...)
    ↓
[Layer 3] 상태 수정자 (기본/부상/버프)
    ↓
[Layer 4] 포즈 수정자 (서기/전투/방어)
    ↓
[Negative] 제외 요소
```

---

## Layer 1: 스타일 베이스

### 모든 캐릭터 공통
```
storybook illustration style character portrait,
hand-painted watercolor and gouache texture,
dark fairytale medieval fantasy atmosphere,
stylized proportions with slightly large head,
expressive eyes and clear readable silhouette,
warm soft lighting from top-left direction,
aged parchment texture background with vignette,
single character centered, card game art,
vertical portrait format 2:3 aspect ratio
```

---

## Layer 2: 클래스 특성

### 전사 (Warrior)
```
[WARRIOR]
warrior class fighter, heavy steel plate armor,
battle-worn and scarred veteran appearance,
broad muscular shoulders, sturdy strong build,
wielding large two-handed greatsword,
or sword and kite shield combination,
steel red and iron gray color scheme,
golden decorative trim accents,
fierce determined warrior expression,
dark brown or black hair, tanned skin
```

### 팔라딘 (Paladin)
```
[PALADIN]
paladin holy knight, ornate golden plate armor,
divine holy symbols and sun emblems engraved,
protective radiant aura glowing softly,
wielding blessed longsword and tower shield,
shield bearing golden sun motif,
white gold and royal blue color scheme,
righteous noble compassionate expression,
blonde or silver hair, fair skin,
ethereal holy light particles
```

### 전사 → 팔라딘 전직
```
[WARRIOR_TO_PALADIN]
warrior turned paladin, transitioning knight,
mix of battle-worn steel and new golden elements,
emerging holy aura beginning to glow,
old scars beneath new divine blessing,
steel red fading into holy gold accents,
expression shifting from fierce to righteous,
blade beginning to shine with holy light
```

## Layer 3: 상태 수정자

### 기본 상태 (Default)
```
[STATE_DEFAULT]
healthy full vitality condition,
armor intact and well-maintained,
alert but relaxed neutral stance,
normal color saturation,
no visible damage or effects
```

### 부상 상태 (Wounded)
```
[STATE_WOUNDED_LIGHT] (HP 50-75%)
minor scratches on armor surface,
slightly tense expression,
subtle fatigue showing

[STATE_WOUNDED_MEDIUM] (HP 25-50%)
visible dents and cracks in armor,
hunched defensive posture,
pained but determined expression,
slightly desaturated colors,
thin red scratches visible

[STATE_WOUNDED_SEVERE] (HP <25%)
heavily damaged broken armor pieces,
kneeling or struggling stance,
gritting teeth with fierce resolve,
desaturated colors with red tint,
blood stains and deep wounds visible
```

### 버프 상태 (Buffed)
```
[STATE_BUFF_STRENGTH]
red fiery aura surrounding body,
empowered aggressive stance,
glowing red particles floating,
intensified fierce expression

[STATE_BUFF_DEFENSE]
blue protective shield aura,
sturdy grounded defensive stance,
blue energy barrier shimmer,
calm confident expression

[STATE_BUFF_HOLY] (팔라딘 전용)
golden divine light emanating,
ethereal holy halo above head,
blessed radiant aura intensified,
serene righteous expression
```

### 디버프 상태 (Debuffed)
```
[STATE_DEBUFF_WEAK]
purple sickly aura clinging,
slouched weakened posture,
strained struggling expression,
desaturated muted colors

[STATE_DEBUFF_POISON]
green toxic mist surrounding,
visibly uncomfortable expression,
greenish skin tint undertone,
dripping poison effect
```

## Layer 4: 포즈 수정자

### 방향 규칙 (필수)
```
[DIRECTION - 모든 포즈에 적용]
body facing right diagonal direction,
three-quarter view angled to right,
character looking toward front-right
```

> **중요**: 모든 캐릭터 카드는 몸이 우측 대각선 방향을 향하도록 생성되어야 합니다.

### 기본 서기 (Idle)
```
[POSE_IDLE]
standing relaxed but ready pose,
weapon held at rest position,
body facing right diagonal 3/4 view,
eyes looking toward front-right,
weight evenly distributed
```

### 전투 준비 (Combat Ready)
```
[POSE_COMBAT]
dynamic combat ready stance,
weapon raised in attack position,
leaning slightly forward,
intense focused gaze,
one foot forward ready to charge
```

### 방어 자세 (Defensive)
```
[POSE_DEFEND]
protective defensive crouch,
shield raised in front,
peeking over shield edge,
braced solid stance,
ready to block incoming attack
```

### 승리 (Victory)
```
[POSE_VICTORY]
triumphant victorious pose,
weapon raised high in air,
proud accomplished expression,
chest out shoulders back,
slight upward camera angle
```

### 피격 (Hit Reaction)
```
[POSE_HIT]
recoiling from impact,
staggered off-balance stance,
grimacing pain expression,
motion blur on edges,
impact effect suggestion
```

## 생성 팁

### 일관성 유지
```
1. 동일 캐릭터는 Layer 1, 2를 고정
2. 상태/포즈만 Layer 3, 4에서 변경
3. 네거티브 프롬프트는 항상 포함
4. 시드 값 기록하여 유사 결과 재현
```

### 품질 향상
```
1. 구체적 형용사 사용 (large → broad muscular)
2. 색상 명시 (red → steel red, crimson)
3. 방향 명시 (light → top-left warm light)
4. 분위기 강조 (dark fairytale, mysterious)
```

### 흔한 문제 해결
```
너무 사실적: "stylized", "illustration" 강조
애니메이션 느낌: "hand-painted", "traditional" 추가
배경이 복잡: "simple background", "vignette" 추가
색상 과포화: "muted colors", "desaturated" 추가
```
