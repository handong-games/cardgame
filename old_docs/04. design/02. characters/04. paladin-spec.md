# 팔라딘 (Paladin) 캐릭터 스펙시트

## 개요

팔라딘은 신성한 빛의 수호자로, 동료를 보호하고 치유하는 탱커/서포터 역할입니다. 고귀하면서도 따뜻하고 귀여운 느낌을 주는 것이 핵심입니다.

---

## 캐릭터 정체성

### 기본 정보
| 항목 | 내용 |
|------|------|
| 클래스명 | 팔라딘 (Paladin) |
| 역할 | 탱커/서포터/힐러 |
| 성격 | 고귀, 온화, 헌신적, 정의로운 |
| 나이대 느낌 | 청년 (성숙하지만 귀엽게) |
| 테마 | 빛, 보호, 치유, 신성 |

### 캐릭터 컨셉
```
팔라딘의 핵심:
├── 빛을 다루는 신성한 수호자
├── 동료를 지키는 따뜻한 마음
├── 정의롭지만 온화한 성품
├── 고귀하지만 친근한 분위기
└── 강인함과 부드러움의 조화
```

## SD 비율 스펙

### 기본 비율
```
팔라딘 SD 비율:
├── 전체 등신: 2.5 등신
├── 머리 비율: 전체의 40%
├── 몸통 비율: 전체의 30%
├── 다리 비율: 전체의 30%
├── 어깨: 머리 너비의 1.4배 (균형잡힌)
└── 손/발: 작고 둥글게
```

### 실제 치수 (512x768 기준)
| 부위 | 높이 | 너비 | 비고 |
|------|------|------|------|
| 머리 | 280px | 230px | 동그란 형태 |
| 몸통 | 220px | 180px | 갑옷으로 약간 넓게 |
| 다리 | 180px | 150px | 짧고 안정적 |
| 방패 | 250px | 180px | 몸과 비슷한 크기 |

## 얼굴 디자인

### 기본 얼굴 특징
```
귀여운 팔라딘 얼굴:
├── 눈: 크고 동그랗게, 부드러운 눈매
│   ├── 눈동자: 파란색 또는 금색
│   ├── 하이라이트: 크게 2개, 별처럼 반짝
│   └── 표정: 온화하고 자애로운
├── 코: 매우 작게, 점 또는 U자
├── 입: 작고 부드러운 미소
│   └── 기본: 온화한 미소
├── 볼 홍조: 양쪽 필수 (#FFB6C1)
├── 눈썹: 부드러운 곡선, 온화한 모양
└── 머리카락: 금발 또는 밝은 갈색, 정돈됨
```

### 표정 가이드
| 표정 | 눈 | 입 | 볼 홍조 | 용도 |
|------|-----|-----|---------|------|
| 기본 | 온화한 눈 | 미소 | 연하게 | 기본 상태 |
| 수호 | 결의 + 온화 | 다문 미소 | 보통 | 방어 시 |
| 치유 | 눈 감고 미소 | 부드러운 미소 | 진하게 | 힐링 시 |
| 기쁨 | 눈웃음 | 환하게 웃음 | 진하게 | 승리 |
| 염려 | 살짝 찌푸림 | 걱정 | 보통 | 동료 부상 |

## 의상 & 장비 디자인

### 갑옷 (Holy Armor)

#### 상체 갑옷
```
팔라딘 갑옷 (상체):
├── 형태: 우아한 흉갑 + 어깨 보호대
├── 소재 느낌: 빛나는 금속, 곡선미
├── 주 색상: Warm Gold (#FFD700)
├── 안감: Cream White (#FFFDD0)
├── 장식:
│   ├── 가슴에 태양/별 문양
│   ├── 어깨에 하늘색 보석
│   └── 금색 테두리 전체
└── 특징: 빛이 나는 듯한 느낌
```

#### 하체 갑옷
```
팔라딘 갑옷 (하체):
├── 형태: 금빛 갑옷 치마
├── 레이어: 금색 + 크림색 안감
├── 색상: Warm Gold + Cream White
├── 가죽 벨트: 흰색, 금색 버클
└── 부츠: 금빛 장식의 흰색 부츠
```

### 망토 (Holy Cape)
```
팔라딘 망토:
├── 형태: 어깨에서 발목까지 긴 망토
├── 외부 색상: Cream White (#FFFDD0)
├── 내부 색상: Sky Blue (#87CEEB)
├── 테두리: 금색 자수 (#FFD700)
├── 움직임: 우아하게 펄럭임
└── 특징: 빛에 반짝이는 느낌
```

### 무기 (Weapon)

#### 성스러운 철퇴 (Holy Mace)
```
팔라딘 철퇴:
├── 크기: 캐릭터와 비슷 (SD 과장)
├── 형태: 둥근 머리의 철퇴
├── 색상:
│   ├── 머리: Warm Gold (#FFD700)
│   ├── 손잡이: Cream White
│   └── 보석: Sky Blue (#87CEEB)
├── 장식: 태양 문양, 빛 이펙트
└── 느낌: 부드럽고 신성한
```

#### 성스러운 방패 (Holy Shield)
```
팔라딘 방패:
├── 크기: 캐릭터 몸통 크기
├── 형태: 우아한 카이트 실드
├── 색상:
│   ├── 주색: Warm Gold (#FFD700)
│   ├── 테두리: Cream White
│   └── 보석: Sky Blue (#87CEEB)
├── 문양: 태양 또는 별 (중앙)
└── 효과: 빛나는 오라
```

## 등급별 시각 변화

### 기본 (Basic) - Lv. 1~10
```
기본 팔라딘:
├── 갑옷: 단순한 금빛 갑옷
├── 색상: 채도 80%
├── 장식: 기본 태양 문양
├── 무기: 기본 철퇴+방패
├── 오라: 미세한 빛
└── 파티클: 가끔 반짝임
```

### 숙련 (Advanced) - Lv. 11~20
```
숙련 팔라딘:
├── 갑옷: 더 정교한 디자인
├── 색상: 채도 90%
├── 장식: 보석 추가, 문양 정교
├── 무기: 빛나는 무기
├── 오라: 상시 금빛 오라
└── 파티클: 주변 반짝임
```

### 마스터 (Master) - Lv. 21~30
```
마스터 팔라딘:
├── 갑옷: 천사 날개 모티브 추가
├── 색상: 채도 100%, 더 밝음
├── 장식: 정교한 자수, 큰 보석
├── 무기: 강한 빛 이펙트
├── 오라: 강한 금빛+하늘빛 오라
└── 파티클: 상시 빛 파티클
```

### 전설 (Legendary) - Lv. 31+
```
전설 팔라딘:
├── 갑옷: 빛의 갑옷, 작은 날개
├── 색상: 눈부시게 빛나는
├── 장식: 모든 곳에 빛 이펙트
├── 무기: 빛의 무기 (반투명)
├── 오라: 완전한 빛 오라 + 후광
└── 파티클: 상시 빛+별 파티클
```

## 상태별 시각 변화

### 버프 상태 (축복)
```
축복 상태:
├── 오라: 금빛 + 녹색 이중 글로우
├── 아이콘: 머리 위 작은 후광
├── 색상: 더 밝게 빛남
└── 파티클: 위로 올라가는 빛
```

### 수호 상태 (보호막)
```
보호막 상태:
├── 오라: 하늘색 방어막
├── 이펙트: 방패 문양 투영
├── 색상: 하늘색 강조
└── 파티클: 주변 회전 빛
```

### 부상 상태
```
부상 팔라딘:
├── 빛: 약간 어두워짐 (너무 어둡지 않게)
├── 표정: 걱정되는 표정
├── 갑옷: 약간 먼지/흠집
└── 효과: 작은 치유 빛 주변에
```

## 전직 프리뷰

### 팔라딘 → 성기사 (Holy Knight)
```
성기사 특징:
├── 테마: 빛의 기사, 정의의 심판자
├── 색상: 더 밝은 금색 + 흰색
├── 갑옷: 천사 날개 장식
├── 무기: 빛의 대검
├── 특징: 작은 날개 장식
└── 오라: 강한 신성 오라
```

### 팔라딘 → 사제 (Priest)
```
사제 특징:
├── 테마: 치유의 달인, 빛의 축복자
├── 색상: 크림 화이트 + 금색
├── 의상: 로브 스타일 (갑옷 감소)
├── 무기: 성스러운 지팡이
├── 특징: 후광, 평화로운 분위기
└── 오라: 치유의 녹색+금색 오라
```
