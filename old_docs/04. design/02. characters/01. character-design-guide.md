# 캐릭터 디자인 가이드

## 개요

이 문서는 플레이어 캐릭터의 시각적 디자인 원칙을 정의합니다. 모든 캐릭터는 귀여운 SD 비율의 카드 형태로 표현됩니다.

---

## 캐릭터 디자인 철학

### 핵심 원칙
```
1. 귀여움: SD 비율 (2-2.5 등신)의 사랑스러운 캐릭터
2. 따뜻함: 볼 홍조, 친근한 표정
3. 손그림 느낌: 연필 스케치 + 수채화 채색
4. 클래스 정체성: 실루엣으로 클래스 구분 가능
```

### SD (Super Deformed) 비율
```
귀여운 SD 캐릭터 비율:
├── 전체 등신: 2~2.5 등신
├── 머리: 전체의 40-50% (매우 큼!)
├── 몸통: 전체의 25-30%
├── 다리: 전체의 20-25% (짧고 귀엽게)
└── 손/발: 작고 동그랗게 단순화
```

---

## 얼굴 표현

### 얼굴 비율
```
귀여운 얼굴:
├── 눈: 얼굴의 40-50% (크고 동그랗게!)
├── 코: 매우 작게 또는 점으로
├── 입: 작고 귀엽게 (미소)
├── 볼: 항상 볼 홍조 추가! (필수)
└── 표정: 따뜻하고 친근하게
```

### 표정 가이드
| 상황 | 표정 | 설명 |
|------|------|------|
| 기본 | 부드러운 미소 | 따뜻하고 친근한 |
| 전투 | 용감한 미소 | 자신감 있지만 귀여운 |
| 승리 | 환한 웃음 | 뿌듯하고 기쁜 |
| 부상 | 살짝 걱정 | 무섭지 않게, 귀엽게 |

### 볼 홍조 (필수!)
```
볼 홍조 규칙:
├── 위치: 양 볼 광대뼈 아래
├── 색상: Rosy Pink #FFB6C1
├── 형태: 둥글거나 약간 타원형
├── 크기: 눈 크기의 30-40%
└── 적용: 모든 캐릭터 필수!
```

## 카드 프레임 내 배치

### 프레임 구조 (빈티지 양피지 스타일)
```
   ╭───────────────────────────╮  ← 둥근 사각형
  ╱│                           │╲ ← 닳고 해진 모서리
 │ │                           │ │
 │ │       ┌─────────┐         │ │
 │ │       │ 캐릭터   │         │ │ ← 질감 있는 베이지 종이
 │ │       │ (SD비율) │         │ │    수채화 종이 결
 │ │       │ 볼홍조!  │         │ │    오래된 종이 질감
 │ │       └─────────┘         │ │    미세한 얼룩
 │ ├───────────────────────────┤ │ ← 얇은 갈색 구분선 (금색 X)
 │ │    ❤️❤️❤️  HP 영역       │ │ ← HP 표시 (하단)
  ╲│                           │╱ ← 닳고 해진 모서리
   ╰───────────────────────────╯  ← 둥근 사각형
```

### 프레임/배경 스타일
| 요소 | 스타일 |
|------|--------|
| 테두리 | 낡은 양피지 테두리 (나무 프레임 X) |
| 형태 | 둥근 사각형 |
| 모서리 | 빈티지한 닳고 해진 종이 |
| 배경색 | 질감 있는 베이지색 |
| 배경 질감 | 수채화 종이 결 |
| 분위기 | 오래된 종이, 미세한 얼룩/낡은 느낌 |

### 배치 규칙
| 영역 | 비율 | 설명 |
|------|------|------|
| 상단 여백 | 5% | 양피지 테두리 |
| 캐릭터 영역 | 75% | SD 캐릭터 중앙 배치 |
| HP 영역 | 15% | 하트 아이콘 + HP |
| 하단 여백 | 5% | 양피지 테두리 |

## 포즈 가이드

### 방향 규칙 (필수!)
```
캐릭터 방향:
├── 몸 방향: 우측 대각선 방향 (필수!)
├── 시선: 우측 전방 대각선
├── 각도: 3/4 뷰 (약간 측면, 대각선)
└── 이유: 카드 레이아웃 일관성 유지
```

> **중요**: 모든 캐릭터 카드는 몸이 우측 대각선 방향을 향하도록 생성되어야 합니다.

### 권장 포즈
| 포즈 | 용도 | 설명 |
|------|------|------|
| 정면 서기 | 기본 카드 | 양손 앞에, 3/4 각도로 우측 대각선 향함 |
| 손 흔들기 | 친근함 | 한 손 들어 인사, 우측 대각선 향함 |
| 무기 들기 | 전투 | 무기 들고 용감하게, 우측 대각선 향함 |
| 기뻐하기 | 승리 | 양손 들고 환호, 우측 대각선 향함 |

### 피해야 할 포즈
- 좌측을 향하는 포즈 (규칙 위반)
- 무서운/위협적인 포즈
- 너무 다이나믹한 액션 (가독성 저하)
- 뒤돌아 있는 포즈 (표정 안 보임)

## 장비 디자인

### 무기 스타일
```
귀여운 무기:
├── 크기: 캐릭터 대비 살짝 크게 (귀여움 포인트)
├── 형태: 단순하고 둥글둥글하게
├── 디테일: 최소한으로
└── 색상: 클래스 악센트 색상 적용
```

### 갑옷 스타일
```
심플한 갑옷:
├── 형태: 단순화된 플레이트
├── 디테일: 과하지 않게
├── 빛 표현: 수채화 하이라이트
└── 장식: 금색 포인트만
```
