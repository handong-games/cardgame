# 배경 디자인 가이드

## 개요

이 문서는 게임 배경의 시각적 디자인 원칙을 정의합니다. 배경은 전투 화면의 따뜻하고 포근한 분위기를 설정하고 세계관을 강화합니다.

---

## 배경 디자인 철학

### 핵심 원칙
```
1. 분위기 설정: 따뜻하고 포근한 분위기 전달
2. 비침습적: 게임플레이를 방해하지 않음
3. 레이어 구조: 깊이감 있는 평면적 표현
4. 세계관 통합: 따뜻한 동화 스타일 유지
```

### 배경의 역할
| 역할 | 설명 |
|------|------|
| 맥락 제공 | 현재 위치/상황 전달 |
| 분위기 조성 | 따뜻함/설렘 등 감정 유도 |
| 시각적 기반 | UI와 캐릭터의 기반 레이어 |
| 세계관 확장 | 동화책 같은 스토리텔링 |

---

## 기술 사양

### 해상도 및 비율
| 항목 | 사양 |
|------|------|
| 기본 해상도 | 1920 x 1080 px |
| 비율 | 16:9 |
| 안전 영역 | 중앙 1280 x 720 (주요 요소) |
| 파일 형식 | PNG (투명도 지원 시) / JPG |

### 레이어 구조
```
배경 레이어 구조:
├── Layer 0: 하늘/최원경 (가장 뒤)
│   └── 색상 그라데이션, 구름, 달/태양
├── Layer 1: 원경 (Far Background)
│   └── 산, 성, 먼 숲 실루엣
├── Layer 2: 중경 (Mid Background)
│   └── 건물, 나무, 주요 환경 요소
├── Layer 3: 근경 (Near Background)
│   └── 앞쪽 장식 요소, 프레임
└── Layer 4: 오버레이 (Optional)
    └── 안개, 파티클, 빛줄기
```

### 색상 처리
```
깊이에 따른 색상 변화:
├── 원경: 밝고 부드러운 파스텔 톤
├── 중경: 따뜻한 기본 색상
├── 근경: 따뜻한 색상, 디테일 증가
└── 전체: 따뜻한 동화 분위기 유지
```

## 구도 가이드라인

### 기본 구도
```
전투 화면 구도:
┌─────────────────────────────────────┐
│         하늘/원경 (20%)             │
├─────────────────────────────────────┤
│                                     │
│         중경 (40%)                  │
│    (적 캐릭터 배치 영역)             │
│                                     │
├─────────────────────────────────────┤
│                                     │
│         전경 (40%)                  │
│    (플레이어 캐릭터 배치 영역)        │
│                                     │
└─────────────────────────────────────┘
```

### 시선 유도
```
주의 사항:
├── 중앙 영역은 캐릭터용으로 비워둠
├── 디테일은 가장자리에 배치
├── 밝은 영역으로 시선 유도 가능
└── 대비가 강한 요소는 가장자리에
```

### 프레임 요소
```
측면 프레임 (Optional):
├── 왼쪽: 나무, 기둥, 커튼 등
├── 오른쪽: 대칭 또는 변형
├── 상단: 아치, 가지, 처마
└── 하단: 풀, 돌, 바닥 요소
```

## 조명 가이드라인

### 기본 조명 설정
```
주광원 방향: 좌상단 45도 (일관성 유지)
분위기 조명: 따뜻하고 부드럽게

숲: 나무 사이로 스며드는 황금빛 햇살
던전: 따뜻한 횃불의 오렌지빛
성: 따뜻한 촛불/스테인드글라스 빛
```

### 시간대별 조명
| 시간대 | 색온도 | 분위기 |
|--------|--------|--------|
| 새벽 | 부드러운 핑크 | 설렘 |
| 낮 | 따뜻한 황색 | 평화로움 |
| 황혼 | 따뜻한 오렌지 | 포근함 |
| 밤 | 따뜻한 보라/파랑 | 신비로움 |

### 조명 효과
```
빛줄기 (God Rays):
├── 사용: 숲, 성당 내부
├── 방향: 위에서 아래로
├── 색상: 따뜻한 황금빛
└── 투명도: 20-40%

부드러운 안개:
├── 사용: 숲, 던전 (은은하게)
├── 위치: 하단
├── 색상: 밝은 크림색
└── 투명도: 20-30%

파티클:
├── 먼지: 따뜻한 황금빛 (느리게 떠다님)
├── 반딧불: 숲 (따뜻한 노란빛)
├── 불씨: 성 (주황빛, 위로 상승)
```

## 애니메이션 가이드라인

### 미세 움직임 (Ambient Animation)
```
권장 요소:
├── 구름 이동 (매우 느리게)
├── 잎 흔들림 (바람 효과)
├── 물 반짝임 (웨이브)
├── 횃불 흔들림 (불꽃)
├── 안개 흐름 (순환)
└── 파티클 부유
```

### 애니메이션 속도
| 요소 | 주기 | 설명 |
|------|------|------|
| 구름 | 60초+ | 거의 인지 안 됨 |
| 바람 | 2-4초 | 자연스러운 흔들림 |
| 물결 | 1-2초 | 잔잔한 움직임 |
| 불꽃 | 0.5-1초 | 빠른 깜빡임 |
| 파티클 | 3-5초 | 천천히 부유 |

## 생성 팁

### 일관성 유지
```
같은 지역 배경 생성 시:
1. 따뜻한 스타일 베이스 고정
2. 지역 수정자 동일하게 유지
3. 시간대/분위기만 변경
4. 따뜻한 색상 팔레트 엄격히 준수
```

### 캐릭터 공간 확보
```
중앙 확보를 위한 프롬프트:
+ "empty center area for characters"
+ "space for foreground elements"
+ "no characters no people"
+ "background environment only"
```

### 따뜻한 분위기 유지
```
분위기 강화 프롬프트:
+ "warm cozy atmosphere"
+ "golden sunlight filtering"
+ "friendly welcoming environment"
+ "children's picture book background"
```
