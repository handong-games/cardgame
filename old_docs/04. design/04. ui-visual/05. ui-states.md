# UI 상태 가이드

## 개요

이 문서는 게임 내 모든 UI 요소의 상태(State) 변화를 정의합니다. 모든 상태 변화는 따뜻하고 부드러운 느낌을 유지하며, 사용자에게 명확한 피드백을 제공합니다.

---

## UI 상태 정의

### 기본 상태 유형

| 상태 | 영문 | 설명 |
|------|------|------|
| 기본 | Normal | 아무 상호작용 없는 기본 상태 |
| 호버 | Hover | 마우스 올림/터치 준비 상태 |
| 눌림 | Active/Pressed | 클릭/터치 중인 상태 |
| 포커스 | Focus | 키보드/게임패드 선택 상태 |
| 비활성 | Disabled | 사용 불가능한 상태 |
| 로딩 | Loading | 처리 중인 상태 |
| 성공 | Success | 성공적으로 완료된 상태 |
| 오류 | Error | 문제가 발생한 상태 |

## 버튼 상태

### 기본 버튼 (Primary Button)

#### 상태별 스타일
| 상태 | 배경색 | 테두리 | 텍스트 | 효과 |
|------|--------|--------|--------|------|
| Normal | Warm Brown (#8B4513) | Soft Gold (#DAA520) 2px | Cream (#FFFDD0) | 없음 |
| Hover | Light Wood (#DEB887) | Warm Gold (#FFD700) 3px | Cream (#FFFDD0) | 살짝 확대 (1.05x), 글로우 |
| Active | Dark Wood (#5D3A1A) | Soft Gold (#DAA520) 2px | Cream (#FFFDD0) | 축소 (0.95x), 아래로 2px |
| Focus | Warm Brown (#8B4513) | Warm Gold (#FFD700) 3px | Cream (#FFFDD0) | 펄스 글로우 |
| Disabled | Light Gray (#D3D3D3) | Gray (#A9A9A9) 1px | Gray (#A9A9A9) | 투명도 60% |
| Loading | Warm Brown (#8B4513) | 회전 테두리 | 로딩 스피너 | 스피너 회전 |

#### 전환 애니메이션
```
상태 전환:
├── Normal → Hover: 150ms, Ease Out Quad
├── Hover → Active: 50ms, Linear
├── Active → Normal: 200ms, Ease Out Back
├── Any → Disabled: 200ms, Ease Out
└── Any → Loading: 150ms, Ease Out
```

| Disabled | 투명 | Gray (#A9A9A9) 1px | Gray (#A9A9A9) | 투명도 50% |

### 위험 버튼 (Danger Button)

#### 상태별 스타일

|------|--------|--------|--------|------|

## 카드 상태

### 핸드 카드 상태

#### 상태별 스타일
| 상태 | 위치 | 스케일 | 테두리/글로우 | 효과 |
|------|------|--------|--------------|------|
| Normal | 핸드 영역 | 1.0 | 기본 나무 프레임 | 없음 |
| Hover | 위로 15px | 1.1 | 금색 글로우 | 살짝 회전 정렬 |
| Selected | 위로 30px | 1.15 | 밝은 금색 글로우 | 반짝임 파티클 |
| Dragging | 마우스/터치 위치 | 1.1 | 강한 금색 글로우 | 그림자 강화 |
| Playable | 핸드 영역 | 1.0 | 녹색 펄스 (#90EE90) | 미세 펄스 |
| Unplayable | 핸드 영역 | 1.0 | 회색 테두리 | 채도 50% 감소 |
| Discarding | 버림 더미로 이동 | 0.8 → 0 | 페이드 아웃 | 회전 15° |

```
카드 상태 전환:
├── Normal → Hover: 200ms, Ease Out Back
├── Hover → Selected: 150ms, Ease Out
├── Selected → Dragging: 즉시
├── Dragging → Drop: 300ms, Ease Out Back
├── Any → Discarding: 300ms, Ease In
└── Normal ↔ Playable: 400ms, 펄스 루프
```

### 에너지 부족 시 카드 표시
```
에너지 부족 카드:
├── 테두리: 회색 (#A9A9A9)
├── 채도: 50% 감소
├── 에너지 표시: 빨간색으로 변경
├── 호버 시: 툴팁 "에너지 부족"
└── 분위기: 슬프지만 귀여운 느낌
```

### 덱/버림더미 카드 스택
| 상태 | 외관 | 효과 |
|------|------|------|
| Normal | 뒷면 스택 | 없음 |
| Hover | 살짝 밝아짐 | 카드 수 표시 |
| Empty | 빈 프레임 | 점선 테두리 |

## 입력 필드 상태

### 텍스트 입력 필드

#### 상태별 스타일
| 상태 | 배경 | 테두리 | 플레이스홀더 | 효과 |
|------|------|--------|-------------|------|
| Normal | Cream (#FFFDD0) | Warm Brown (#8B4513) 1px | Soft Brown (#6B5344) | 없음 |
| Focus | Warm Parchment (#FFF5E6) | Soft Gold (#DAA520) 2px | 사라짐 | 살짝 확대 |
| Filled | Cream (#FFFDD0) | Warm Brown (#8B4513) 1px | 텍스트 표시 | 체크 아이콘 |
| Error | Rose Light (#FFE4E4) | Coral Red (#FF6B6B) 2px | 에러 메시지 | 흔들림 |
| Disabled | Light Gray (#D3D3D3) | Gray (#A9A9A9) 1px | 회색 텍스트 | 투명도 60% |

```
입력 필드 전환:
├── Normal → Focus: 150ms, Ease Out
├── Focus → Filled: 즉시
├── Any → Error: 200ms + 흔들림 300ms
└── Error → Normal: 200ms, Ease Out
```

## 체크박스 & 라디오 버튼

### 체크박스

#### 상태별 스타일
| 상태 | 배경 | 테두리 | 체크 마크 | 효과 |
|------|------|--------|----------|------|
| Unchecked | Cream (#FFFDD0) | Warm Brown (#8B4513) 2px | 없음 | 없음 |
| Unchecked + Hover | Warm Parchment (#FFF5E6) | Soft Gold (#DAA520) 2px | 없음 | 살짝 확대 |
| Checked | Soft Gold (#DAA520) | Warm Brown (#8B4513) 2px | Cream (#FFFDD0) ✓ | 바운스 등장 |
| Checked + Hover | Warm Gold (#FFD700) | Soft Gold (#DAA520) 2px | Cream (#FFFDD0) ✓ | 글로우 |
| Disabled | Light Gray (#D3D3D3) | Gray (#A9A9A9) 1px | 회색 ✓ (있으면) | 투명도 50% |

```
체크 애니메이션:
├── Unchecked → Checked: 200ms
│   ├── 배경: 색상 전환
│   └── 체크마크: 스케일 0 → 1.2 → 1.0 (Ease Out Back)
└── Checked → Unchecked: 150ms
    └── 체크마크: 스케일 1 → 0 (Ease In)
```

### 토글 스위치

#### 상태별 스타일
| 상태 | 트랙 (OFF) | 트랙 (ON) | 핸들 | 효과 |
|------|-----------|----------|------|------|
| OFF | Warm Beige (#F5DEB3) | - | Warm Brown (#8B4513), 왼쪽 | 없음 |
| ON | - | Soft Green (#90EE90) | Warm Brown (#8B4513), 오른쪽 | 없음 |
| Hover (OFF) | Light Wood (#DEB887) | - | 확대 | 글로우 |
| Hover (ON) | - | Soft Green 밝게 | 확대 | 글로우 |
| Disabled | Light Gray (#D3D3D3) | Light Gray (#D3D3D3) | Gray | 투명도 50% |

```
토글 애니메이션:
├── OFF → ON: 250ms
│   ├── 핸들: 왼쪽 → 오른쪽 (Ease Out Back)
│   └── 트랙: 색상 전환
└── ON → OFF: 250ms
    └── 핸들: 오른쪽 → 왼쪽 (Ease Out Back)
```

|------|------|--------|--------|------|
| Inactive | 투명 | Soft Brown (#6B5344) | 없음 | 없음 |

```
탭 전환 애니메이션:
├── 하단 바: 이전 탭 → 새 탭 슬라이드 (300ms, Ease Out)
├── 텍스트 색상: 150ms, Ease Out
└── 콘텐츠: 페이드 전환 (200ms)
```

## 알림 & 토스트

### 알림 유형별 스타일

#### 성공 알림
```
├── 배경: Soft Green (#90EE90) 20%
├── 테두리: Soft Green (#90EE90) 왼쪽 4px
├── 아이콘: 체크마크, 녹색
├── 텍스트: Warm Brown (#4A3728)
└── 애니메이션: 오른쪽에서 슬라이드 인
```

#### 정보 알림
```
├── 배경: Sky Blue (#87CEEB) 20%
├── 테두리: Sky Blue (#87CEEB) 왼쪽 4px
├── 아이콘: 정보 i, 파란색
├── 텍스트: Warm Brown (#4A3728)
└── 애니메이션: 위에서 슬라이드 인
```

#### 경고 알림
```
├── 배경: Warm Amber (#FFBF00) 20%
├── 테두리: Warm Amber (#FFBF00) 왼쪽 4px
├── 아이콘: 삼각형 !, 주황색
├── 텍스트: Warm Brown (#4A3728)
└── 애니메이션: 흔들림 + 등장
```

#### 오류 알림
```
├── 배경: Rose Pink (#FFB6C1) 20%
├── 테두리: Coral Red (#FF6B6B) 왼쪽 4px
├── 아이콘: X 마크, 빨간색 (귀엽게)
├── 텍스트: Warm Brown (#4A3728)
└── 애니메이션: 흔들림 + 등장
```

**주의**: 오류 알림도 따뜻하고 친근하게 (무섭지 않게)

### 토스트 애니메이션
```
토스트 라이프사이클:
├── 등장 (0-300ms): 위/아래에서 슬라이드, Ease Out Back
├── 표시 (300ms-3000ms): 정지
├── 사라짐 (3000ms-3300ms): 페이드 + 위로 슬라이드, Ease In
└── 자동 닫기: 3초 (설정 가능)
```

## 로딩 상태

### 스피너
```
로딩 스피너:
├── 형태: 귀여운 원형
├── 색상: Soft Gold (#DAA520) → Warm Gold (#FFD700) 그라데이션
├── 크기: 24px (소), 40px (중), 64px (대)
├── 애니메이션: 360° 회전, 1000ms, Linear 루프
└── 스타일: 동화책 느낌의 부드러운 선
```

### 프로그레스 바
```
진행 바:
├── 트랙: Warm Beige (#F5DEB3)
├── 채움: Soft Gold (#DAA520) → Warm Gold (#FFD700) 그라데이션
├── 테두리: Warm Brown (#8B4513) 1px, 둥근 모서리
├── 애니메이션: 채움 확장 (Ease Out)
└── 완료 시: 반짝임 효과 + 색상 녹색으로
```

### 스켈레톤 로딩
```
스켈레톤 화면:
├── 배경: Warm Beige (#F5DEB3) 50%
├── 애니메이션: 좌 → 우 반짝임 웨이브
├── 웨이브 색상: Cream (#FFFDD0) 70%
├── 웨이브 속도: 1500ms 루프
└── 형태: 실제 콘텐츠와 유사한 둥근 형태
```

## 터치/클릭 피드백

### 터치 리플 효과
```
리플 효과:
├── 시작: 터치 위치에서
├── 형태: 원형 확장
├── 색상: Cream (#FFFDD0) 40%
├── 크기: 0px → 60px
├── 지속시간: 300ms
├── 이징: Ease Out
└── 종료: 투명도 0%로 페이드
```

### 햅틱 피드백 연동
| 상호작용 | 햅틱 유형 | 강도 |
|---------|----------|------|
| 버튼 탭 | Light | 약함 |
| 카드 선택 | Medium | 중간 |
| 에러 발생 | Notification | 짧은 진동 |
| 레벨업/보상 | Heavy | 강함 |
| 전직 완료 | Success | 특수 패턴 |

## 접근성 고려사항

### 포커스 표시
```
키보드/게임패드 포커스:
├── 테두리: Warm Gold (#FFD700) 3px
├── 오프셋: 요소 바깥 2px
├── 애니메이션: 펄스 글로우 (1500ms 루프)
└── 대비: 배경과 충분한 대비 확보
```

### 색상 대비 기준
| 용도 | 최소 대비 비율 | 권장 |
|------|---------------|------|
| 본문 텍스트 | 4.5:1 | 7:1 |
| 대형 텍스트 | 3:1 | 4.5:1 |
| UI 컴포넌트 | 3:1 | 4.5:1 |
| 그래픽 요소 | 3:1 | - |

### 색맹 대응
```
색맹 친화적 디자인:
├── 색상만으로 상태 구분하지 않음
├── 아이콘/패턴으로 추가 구분
├── 밝기 대비 충분히 확보
└── 테스트: 색맹 시뮬레이터로 검증
```
