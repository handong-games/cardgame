# VFX 카탈로그

## 개요

이 문서는 게임 내 모든 시각 효과(VFX)를 카탈로그화합니다. 각 효과는 따뜻한 스토리북 스타일을 유지하며 귀엽고 친근한 느낌을 줍니다.

---

## 전투 이펙트

### 물리 공격 이펙트

#### 베기 (Slash)
```
용도: 검, 도끼 등 근접 무기 공격
├── 형태: 부드러운 호(arc) 형태
├── 색상: Cream (#FFFDD0) + Soft Gold (#DAA520)
├── 지속시간: 200ms
├── 크기: 80px ~ 120px
├── 애니메이션: 호 궤적을 따라 빠르게 스윕
├── 잔상: 2~3개의 페이드 아웃 잔상
└── 파티클: 작은 반짝임 3~5개
```

**프롬프트**
```
soft curved slash effect, warm cream and gold colors,
hand-drawn swoosh motion, storybook illustration style,
gentle trailing sparkles, not aggressive or sharp
```

#### 찌르기 (Stab)
```
용도: 창, 단검 등 관통 공격
├── 형태: 직선 모션 블러
├── 색상: Light Silver (#D3D3D3) + Soft Blue (#87CEEB)
├── 지속시간: 150ms
├── 크기: 60px ~ 100px
├── 애니메이션: 직선으로 빠르게 이동
├── 임팩트: 작은 별 모양 플래시
└── 파티클: 임팩트 지점에서 방사형 반짝임
```

#### 타격 (Impact)
```
용도: 주먹, 방패 등 충격 공격
├── 형태: 별 모양 임팩트
├── 색상: Warm Yellow (#FFD93D) + Warm Orange (#FFA500)
├── 지속시간: 250ms
├── 크기: 40px ~ 80px
├── 애니메이션: 확장 후 페이드
├── 반짝임: 중앙에서 바깥으로 퍼지는 링
└── 파티클: 별 파티클 5~8개
```

### 마법 공격 이펙트

#### 화염 마법 (Fire Magic)
```
용도: 화염 계열 스킬
├── 형태: 부드러운 불꽃 뭉치
├── 색상: Warm Orange (#FFA500) + Soft Coral (#FF6B6B)
├── 지속시간: 400ms
├── 크기: 60px ~ 100px
├── 애니메이션: 타오르며 위로 상승
├── 파티클: 불꽃 파티클 10~15개
└── 글로우: 주변 따뜻한 빛 확산
```

**주의**: 불꽃도 귀엽게! 무섭지 않게!

**프롬프트**
```
cute warm fire magic effect, soft orange and coral,
friendly flame burst, storybook illustration,
watercolor fire texture, not scary or threatening
```

#### 얼음 마법 (Ice Magic)
```
용도: 빙결 계열 스킬
├── 형태: 반짝이는 결정
├── 색상: Sky Blue (#87CEEB) + Light Silver (#D3D3D3)
├── 지속시간: 350ms
├── 크기: 50px ~ 90px
├── 애니메이션: 결정이 나타나며 반짝임
├── 파티클: 눈송이/반짝임 8~12개
└── 느낌: 차갑지만 아름다운 느낌
```

**주의**: 차가운 느낌이지만 따뜻한 색감 유지

#### 번개 마법 (Lightning Magic)
```
용도: 전격 계열 스킬
├── 형태: 부드러운 지그재그 선
├── 색상: Sunny Yellow (#FFD93D) + Cream (#FFFDD0)
├── 지속시간: 200ms
├── 크기: 전체 경로
├── 애니메이션: 빠른 플래시 + 잔상
├── 파티클: 별 반짝임 10~15개
└── 글로우: 밝은 황금색 글로우
```

#### 치유 마법 (Healing Magic)
```
용도: 회복 스킬
├── 형태: 상승하는 빛 기둥 + 잎/꽃잎
├── 색상: Soft Green (#90EE90) + Rose Pink (#FFB6C1)
├── 지속시간: 600ms
├── 크기: 캐릭터 전체 커버
├── 애니메이션: 아래에서 위로 빛 상승
├── 파티클: 잎/꽃잎 8~12개, 반짝임 10개
└── 숫자: +HP 팝업
```

**프롬프트**
```
gentle healing magic effect, soft green and pink,
rising flower petals and sparkles, warm glow,
storybook fairy tale healing, comforting atmosphere
```

## 상태 이펙트

### 데미지 표시

#### 물리 데미지
```
요소 구성:
├── 숫자: 빨간색 (#CD5C5C), 위로 튀어오름
├── 폰트: 굵은 둥근 폰트, 외곽선 있음
├── 크기: 24px ~ 36px
├── 애니메이션:
│   ├── 0-100ms: 위로 튀어오름 (easeOutBack)
│   ├── 100-300ms: 정지
│   └── 300-500ms: 페이드 아웃
└── 파티클: 작은 임팩트 반짝임 3개
```

#### 마법 데미지
```
요소 구성:
├── 숫자: 보라색 (#9370DB), 위로 떠오름
├── 폰트: 굵은 둥근 폰트, 마법 글로우
├── 크기: 24px ~ 36px
├── 애니메이션: 부드럽게 위로 떠오름
└── 파티클: 마법 반짝임 5개
```

#### 크리티컬 데미지
```
요소 구성:
├── 숫자: 금색 (#FFD700), 더 크게
├── 폰트: 굵은 폰트, 두꺼운 외곽선
├── 크기: 36px ~ 48px
├── 텍스트: "CRITICAL!" 또는 "치명타!"
├── 애니메이션:
│   ├── 확대 후 정상 크기로
│   ├── 흔들림 효과
│   └── 별 파티클 방출
└── 파티클: 별 8~12개
```

### 힐 표시

#### 체력 회복
```
요소 구성:
├── 숫자: 녹색 (#90EE90), 위로 떠오름
├── 폰트: 둥근 폰트, 부드러운 글로우
├── 크기: 24px ~ 32px
├── 접두사: "+" 기호
├── 애니메이션: 부드럽게 위로 (easeOutQuad)
└── 파티클: 하트 또는 잎 파티클 3~5개
```

### 버프/디버프 표시

#### 버프 적용
```
효과 구성:
├── 아이콘 글로우: 금색 (#FFD700) 테두리 글로우
├── 상승 화살표: 녹색 (#90EE90)
├── 파티클: 위로 올라가는 반짝임 5개
├── 지속시간: 400ms
└── 사운드 큐: 긍정적 차임
```

#### 디버프 적용
```
효과 구성:
├── 아이콘 글로우: 연보라 (#E6E6FA) 테두리
├── 하강 화살표: 연보라 (#DDA0DD)
├── 파티클: 아래로 내려가는 먼지 5개
├── 지속시간: 400ms
└── 분위기: 슬프지만 귀여운 느낌
```

## UI 이펙트

### 글로우 효과 (Glow)

#### 버튼 호버 글로우
```
├── 색상: Soft Gold (#DAA520)
├── 크기: 버튼 + 8px 확장
├── 투명도: 0% → 40% → 0%
├── 지속시간: 300ms (in/out)
└── 애니메이션: 부드러운 페이드
```

#### 선택 가능 카드 글로우
```
├── 색상: Warm Gold (#FFD700)
├── 크기: 카드 + 6px 확장
├── 투명도: 20% ~ 50% (펄스)
├── 펄스 주기: 1000ms
└── 애니메이션: 부드러운 펄스
```

#### 선택된 카드 글로우
```
├── 색상: Bright Gold (#FFF8DC)
├── 크기: 카드 + 10px 확장
├── 투명도: 60%
├── 파티클: 주변에 떠다니는 반짝임 3개
└── 애니메이션: 살짝 위로 떠오름
```

### 파동 효과 (Ripple)

#### 터치 피드백
```
├── 형태: 원형 파동
├── 색상: Cream (#FFFDD0)
├── 시작 크기: 20px
├── 최종 크기: 60px
├── 투명도: 40% → 0%
├── 지속시간: 300ms
└── 애니메이션: 확장하며 페이드
```

#### 중요 알림 파동
```
├── 형태: 원형 파동 (3회 반복)
├── 색상: Warm Gold (#FFD700)
├── 크기: 40px → 80px
├── 투명도: 50% → 0%
├── 간격: 200ms
└── 총 지속시간: 600ms
```

### 반짝임 효과 (Sparkle)

#### 새로운 아이템 반짝임
```
├── 형태: 별 모양 x 3~5개
├── 색상: Warm Gold (#FFD700)
├── 크기: 8px ~ 16px
├── 위치: 아이템 주변 랜덤
├── 주기: 500ms마다 반짝임
└── 지속시간: 아이템 확인까지 지속
```

#### 업그레이드 가능 반짝임
```
├── 형태: 작은 별 순환
├── 색상: Sunny Yellow (#FFD93D)
├── 크기: 6px ~ 12px
├── 위치: 버튼/아이콘 모서리
├── 회전: 시계 방향 순환
└── 주기: 1500ms 순환
```

## 특수 이펙트

### 전직 연출 이펙트

#### Phase 1: 집중 (0-1000ms)
```
├── 캐릭터 주변 빛 수렴
├── 파티클: 바깥에서 안으로 모이는 별 20개
├── 색상: Soft Gold (#DAA520)
├── 배경: 점점 밝아짐
└── 캐릭터: 살짝 빛남
```

#### Phase 2: 변신 (1000-2000ms)
```
├── 강한 빛 플래시
├── 캐릭터 실루엣만 보임
├── 파티클: 방사형 폭발 별 30개
├── 색상: Warm Gold (#FFD700) + Cream (#FFFDD0)
└── 사운드: 업그레이드 차임
```

#### Phase 3: 등장 (2000-3000ms)
```
├── 빛이 사라지며 새 모습 공개
├── 캐릭터 포즈: 영웅적 등장 포즈
├── 파티클: 주변에 떠다니는 반짝임
├── 글로우: 금색 오라 유지 (3초)
└── 텍스트: "전직 완료!" 팝업
```

### 보스 등장 이펙트
```
연출 시퀀스:
├── 0-500ms: 화면 살짝 어두워짐 (밝기 80%)
├── 500-1000ms: 보스 위치에 빛 기둥
├── 1000-1500ms: 보스 실루엣 등장
├── 1500-2000ms: 빛 사라지며 보스 공개
├── 파티클: 위엄있는 반짝임 (금색, 은색)
└── 느낌: 위엄있지만 귀여운 보스
```

### 승리/패배 이펙트

#### 승리 연출
```
요소 구성:
├── 배경: 황금빛으로 밝아짐
├── 텍스트: "승리!" 크게 표시
│   ├── 색상: Warm Gold (#FFD700)
│   ├── 크기: 64px → 80px → 64px
│   └── 애니메이션: 확대 후 안정
├── 파티클:
│   ├── 별 폭죽 30개
│   ├── 꽃가루/종이 조각 20개
│   └── 반짝임 지속
└── 분위기: 축제처럼 밝고 기쁜 느낌
```

#### 패배 연출
```
요소 구성:
├── 배경: 살짝 어두워짐 (너무 어둡지 않게, 80%)
├── 텍스트: "다시 도전!" (패배보다 긍정적)
│   ├── 색상: Soft Purple (#9370DB)
│   ├── 크기: 48px
│   └── 애니메이션: 부드럽게 등장
├── 파티클: 작은 별 몇 개 (위로하는 느낌)
└── 분위기: 슬프지만 희망적인 느낌
```

## 코인 플립 이펙트

### 코인 회전 중
```
├── 코인 회전: Y축 360° x 3~5회
├── 빛 반사: 회전마다 반짝임
├── 파티클: 궤적에 반짝임 흔적
├── 색상: Gold + Silver 번갈아
├── 지속시간: 1500ms ~ 2500ms
└── 사운드: 동전 회전 소리
```

### 앞면 결과 (Heads - 유리)
```
├── 코인 정지: 앞면 보이며 빛남
├── 플래시: 금색 빛 확산
├── 파티클: 금색 별 폭발 15개
├── 텍스트: "앞면!" + 효과 설명
├── 글로우: 따뜻한 황금빛 오라
└── 분위기: 기쁘고 행운의 느낌
```

### 뒷면 결과 (Tails - 불리)
```
├── 코인 정지: 뒷면 보이며 은빛
├── 플래시: 은색 빛 (밝게)
├── 파티클: 은색 반짝임 10개
├── 텍스트: "뒷면!" + 효과 설명
├── 글로우: 은색 오라 (슬프지 않게)
└── 분위기: 아쉽지만 다음을 기대
```

## 이펙트 레이어 순서

### 렌더링 우선순위
```
레이어 순서 (뒤에서 앞):
├── Layer 0: 배경 파티클 (먼지, 앰비언트)
├── Layer 1: 배경
├── Layer 2: 지면 이펙트 (그림자)
├── Layer 3: 캐릭터/몬스터
├── Layer 4: 캐릭터 이펙트 (오라, 버프)
├── Layer 5: 공격/스킬 이펙트
├── Layer 6: 데미지 숫자
├── Layer 7: UI 요소
└── Layer 8: UI 이펙트 (글로우, 알림)
```
