# 애니메이션 가이드

## 개요

이 문서는 게임 내 모든 애니메이션의 원칙, 타이밍, 이징 함수를 정의합니다. 모든 애니메이션은 따뜻하고 부드러운 느낌을 주며, 동화책 일러스트의 분위기를 유지합니다.

---

## 애니메이션 기본 원칙

### 디즈니 12원칙 (간소화 적용)

#### 1. 스쿼시 & 스트레치 (Squash & Stretch)
```
귀여운 캐릭터에 생명력 부여:
├── 점프/착지 시 압축과 늘어남
├── 공격 모션의 과장된 움직임
├── 표정 변화의 탄력적 움직임
└── 적용 비율: 원본의 80% ~ 120%
```

#### 2. 예비 동작 (Anticipation)
```
동작 전 살짝 반대로:
├── 공격 전: 뒤로 살짝 물러남
├── 점프 전: 아래로 웅크림
├── 던지기 전: 팔을 뒤로
└── 지속시간: 전체 동작의 15-25%
```

#### 3. 과장 (Exaggeration)
```
SD 캐릭터의 귀여운 과장:
├── 표정: 더 크게, 더 귀엽게
├── 동작: 더 역동적으로
├── 반응: 더 명확하게
└── 이펙트: 더 화려하게 (따뜻하게)
```

#### 4. 느린 시작/끝 (Ease In/Out)
```
자연스러운 가속과 감속:
├── 모든 움직임에 이징 적용
├── 급작스러운 시작/끝 금지
├── 부드러운 곡선 움직임
└── 이징 함수 참조
```

## 이징 함수 라이브러리

### 기본 이징

#### Linear (선형)
```
용도: 로딩 바, 카운트다운
곡선: 일정한 속도
CSS: linear
코드: t
사용: 제한적으로만 사용
```

#### Ease Out Quad (감속)
```
용도: 대부분의 UI 애니메이션
곡선: 빠르게 시작, 천천히 끝
CSS: ease-out
코드: 1 - (1 - t) * (1 - t)
사용: 가장 많이 사용
```

#### Ease In Out Quad (가속 후 감속)
```
용도: 화면 전환, 모달
곡선: 천천히 시작, 중간 빠름, 천천히 끝
CSS: ease-in-out
코드: t < 0.5 ? 2 * t * t : 1 - pow(-2 * t + 2, 2) / 2
사용: 화면 전환에 필수
```

### 탄성 이징

#### Ease Out Back (오버슈트)
```
용도: 카드 등장, 버튼 눌림, 팝업
곡선: 목표 지점을 살짝 넘었다가 돌아옴
특징: 탄력 있는 느낌
코드: 1 + 2.70158 * pow(t - 1, 3) + 1.70158 * pow(t - 1, 2)
사용: 귀여운 등장 효과에 적합
```

#### Ease Out Elastic (스프링)
```
용도: 중요한 알림, 레벨업, 보상
곡선: 목표 지점에서 튕기듯 진동
특징: 주목도 높음
진동: 2~3회 감쇄 진동
사용: 특별한 순간에만 사용
```

#### Ease Out Bounce (바운스)
```
용도: 아이템 드롭, 착지, 충격 후
곡선: 땅에 떨어져 튀는 느낌
특징: 경쾌하고 재미있음
바운스: 3~4회 감쇄 바운스
사용: 물리적 충격 표현
```

### 이징 선택 가이드

| 상황 | 권장 이징 | 이유 |
|------|-----------|------|
| 일반 UI 전환 | Ease Out Quad | 자연스럽고 빠른 반응 |
| 화면 전환 | Ease In Out Quad | 부드러운 전환 |
| 카드 등장 | Ease Out Back | 탄력 있는 귀여운 느낌 |
| 알림/팝업 | Ease Out Back | 주목성 확보 |
| 레벨업/보상 | Ease Out Elastic | 특별한 느낌 강조 |
| 아이템 드롭 | Ease Out Bounce | 물리적 재미 |
| 사라짐 | Ease In Quad | 빠르게 사라짐 |

## 타이밍 가이드

### 지속시간 기준

| 유형 | 시간 | 용도 |
|------|------|------|
| 순간 피드백 | 100ms | 버튼 눌림, 터치 피드백 |
| 빠른 전환 | 150ms | 호버, 작은 상태 변화 |
| 표준 전환 | 200ms | 일반 UI 애니메이션 |
| 의미 있는 전환 | 300ms | 카드 이동, 패널 열림 |
| 강조 전환 | 500ms | 중요한 상태 변화 |
| 연출 효과 | 800ms | 스킬 이펙트, 공격 |
| 드라마틱 연출 | 1200ms | 전직, 보스 등장 |
| 컷씬 | 2000ms+ | 스토리 연출 |

### 타이밍 원칙
```
타이밍 규칙:
├── 사용자 입력 피드백: 100ms 이내
├── 연속 동작 간격: 50ms 이상
├── 읽기 필요 텍스트: 최소 1500ms 노출
├── 중요 정보: 최소 2000ms 노출
└── 스킵 가능 연출: 터치로 스킵 지원
```

## 캐릭터 애니메이션

### 대기 (Idle)
```
기본 대기 루프:
├── 지속시간: 2000ms (루프)
├── 호흡: 위아래 2px 움직임
│   ├── 0-1000ms: 살짝 위로 (2px)
│   └── 1000-2000ms: 원위치로
├── 눈 깜빡임: 3000ms마다
│   ├── 눈 감기: 50ms
│   └── 눈 뜨기: 100ms
├── 이징: Ease In Out Sine
└── 분위기: 평화롭고 귀여운 느낌
```

### 공격 (Attack)
```
공격 모션:
├── 총 지속시간: 600ms
├── Phase 1 - 예비 (0-150ms):
│   ├── 뒤로 물러남 (5px)
│   ├── 무기 들어올림
│   └── 이징: Ease Out Quad
├── Phase 2 - 공격 (150-350ms):
│   ├── 앞으로 돌진 (15px)
│   ├── 무기 휘두름
│   └── 이징: Ease In Quad (빠르게)
├── Phase 3 - 임팩트 (350-400ms):
│   ├── 잠시 멈춤
│   ├── 임팩트 이펙트
│   └── 화면 흔들림 (2px, 50ms)
└── Phase 4 - 복귀 (400-600ms):
    ├── 원위치로
    └── 이징: Ease Out Quad
```

### 피격 (Hit)
```
피격 모션:
├── 총 지속시간: 400ms
├── Phase 1 - 충격 (0-100ms):
│   ├── 뒤로 밀림 (8px)
│   ├── 몸 떨림
│   └── 이징: Linear (빠른 충격)
├── Phase 2 - 표정 (100-200ms):
│   ├── 눈 X자 또는 찡그림
│   ├── 입 벌어짐
│   └── 붉은 깜빡임 (1회)
├── Phase 3 - 복귀 (200-400ms):
│   ├── 원위치로
│   ├── 표정 정상화
│   └── 이징: Ease Out Quad
└── 분위기: 아파도 귀여운 느낌
```

### 사망 (Death)
```
사망 모션 (귀엽게):
├── 총 지속시간: 1000ms
├── Phase 1 - 충격 (0-200ms):
│   ├── 눈 빙글빙글 또는 X자
│   ├── 별 파티클 머리 위로 3개
│   └── 살짝 흔들림
├── Phase 2 - 쓰러짐 (200-600ms):
│   ├── 옆으로 기울어짐 (30도)
│   ├── 천천히 투명해짐 (80%)
│   └── 이징: Ease Out Quad
├── Phase 3 - 소멸 (600-1000ms):
│   ├── 반짝임 파티클과 함께 사라짐
│   ├── 투명도 80% → 0%
│   └── 작은 별/하트 파티클 5개
└── 분위기: 슬프지만 폭력적이지 않게
```

### 승리 포즈 (Victory)
```
승리 모션:
├── 총 지속시간: 1500ms
├── Phase 1 - 기쁨 (0-500ms):
│   ├── 팔 들어올리기
│   ├── 위로 점프 (10px)
│   └── 이징: Ease Out Back
├── Phase 2 - 환호 (500-1000ms):
│   ├── 빙글 회전 (360도)
│   ├── 별 파티클 10개
│   └── 이징: Ease Out Quad
├── Phase 3 - 포즈 (1000-1500ms):
│   ├── 영웅적 포즈 유지
│   ├── 반짝반짝 글로우
│   └── 루프
└── 분위기: 기쁘고 자랑스러운 느낌
```

## 카드 애니메이션

### 카드 뽑기 (Draw)
```
카드 뽑기:
├── 총 지속시간: 400ms
├── 시작 위치: 덱 위치 (화면 밖)
├── 끝 위치: 핸드 영역
├── 스케일: 0.5 → 1.0
├── 회전: -30도 → 0도
├── 이징: Ease Out Back (탄력)
└── 파티클: 카드 따라 반짝임 3개
```

### 카드 선택 (Select)
```
카드 선택:
├── 총 지속시간: 200ms
├── 위로 이동: 20px
├── 스케일: 1.0 → 1.1
├── 글로우: 금색 테두리 활성화
├── 이징: Ease Out Back
└── 사운드: 부드러운 팝 소리
```

### 카드 사용 (Play)
```
카드 사용:
├── 총 지속시간: 500ms
├── Phase 1 - 이동 (0-200ms):
│   ├── 타겟 방향으로 이동
│   └── 이징: Ease Out Quad
├── Phase 2 - 효과 (200-400ms):
│   ├── 빛 폭발
│   ├── 효과 적용
│   └── 카드 빛남
├── Phase 3 - 소멸 (400-500ms):
│   ├── 페이드 아웃
│   └── 반짝임과 함께 사라짐
└── 이징: Ease In Quad
```

### 카드 버리기 (Discard)
```
카드 버리기:
├── 총 지속시간: 300ms
├── 이동: 버림 더미 방향
├── 회전: 0도 → 15도
├── 스케일: 1.0 → 0.8
├── 투명도: 100% → 0%
├── 이징: Ease In Quad
└── 파티클: 없음 (빠르게 처리)
```

## 화면 전환 애니메이션

### 페이드 전환
```
기본 화면 전환:
├── 페이드 아웃: 200ms, 0% → 100% 검정
├── 로딩 (필요시): 변동
├── 페이드 인: 300ms, 100% → 0% 검정
├── 이징: Ease In Out Quad
└── 총 시간: 500ms + 로딩
```

### 슬라이드 전환
```
좌우 슬라이드:
├── 현재 화면: 왼쪽으로 -100%
├── 새 화면: 오른쪽 100% → 0%
├── 지속시간: 400ms
├── 이징: Ease Out Quad
└── 용도: 메뉴 이동, 목록 스크롤
```

### 전투 시작 전환
```
전투 진입:
├── Phase 1 (0-500ms): 화면 흔들림 + 빛 플래시
├── Phase 2 (500-1000ms): 원형 와이프 전환
├── Phase 3 (1000-1500ms): 전투 화면 등장
├── Phase 4 (1500-2000ms): 적 등장 연출
└── 총 시간: 2000ms
```
