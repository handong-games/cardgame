# 프롬프트 라이브러리

## 개요

이 문서는 검증된 프롬프트 모음을 제공합니다. 따뜻한 스토리북 스타일의 귀여운 SD 캐릭터를 생성하기 위한 프롬프트입니다.

## 실패 사례 및 교훈

### 사례 1: SD 비율 실패
```
실패 프롬프트:
"cute warrior character illustration"

문제점:
- SD 비율 미명시
- 결과: 일반적인 비율의 캐릭터

수정:
+ "chibi SD proportions" 명시
+ "big head small body 2-2.5 ratio" 추가
+ "large expressive round eyes" 추가
```

### 사례 2: 볼 홍조 누락
```
실패 프롬프트:
"cute chibi character with smile"

문제점:
- 볼 홍조 미언급
- 결과: 홍조 없는 캐릭터

수정:
+ "rosy blushing cheeks on both sides" 필수 추가
+ "pink blush" 명시
+ 네거티브에 "no blush" 추가
```

### 사례 3: 어두운 분위기
```
실패 프롬프트:
"fantasy goblin monster in forest"

문제점:
- 따뜻한 분위기 미강조
- 결과: 어둡고 무서운 고블린

수정:
+ "warm cozy friendly" 강조
+ "cute adorable mischievous" 추가
+ "playful" 사용
+ 네거티브에 "dark scary horror" 강화
```

### 사례 4: 복잡한 프레임
```
실패 프롬프트:
"fantasy card frame with decorations"

문제점:
- 심플함 미강조
- 결과: 복잡한 장식 프레임

수정:
+ "simple wooden frame" 강조
+ "minimal decoration" 추가
+ "brown tones" 명시
+ 네거티브에 "complex ornate" 추가
```
