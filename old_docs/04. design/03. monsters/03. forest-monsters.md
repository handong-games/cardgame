# 숲 지역 몬스터 스펙시트

## 개요

이 문서는 숲(Forest) 지역에 등장하는 모든 몬스터의 상세 스펙을 정의합니다. 모든 몬스터는 **악당/적대적 스타일**로 디자인됩니다.

---

## 지역 개요

### 숲 지역 테마
```
숲 지역 분위기:
├── 환경: 어둡고 위험한 숲속
├── 색조: 어두운 녹색, 탁한 갈색, 불길한 빛
├── 느낌: 포식자가 잠복한 위협적 공간
├── 시간대: 황혼/새벽 (그림자 강조)
└── 특징: 어두운 동화 속 숲, 야생적 생물들
```

### 지역 색상 팔레트
| 용도 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| 주색 | Dark Forest Green | #228B22 | 나뭇잎, 풀 |
| 보조 | Murky Brown | #5C4033 | 나무, 땅 |
| 눈 악센트 | Glowing Yellow | #FFD700 | 빛나는 눈 |

## 공통 스타일 가이드

### 모든 몬스터 필수 요소
```
필수 디자인 요소:
├── 위협적 비율 (과장된 특징)
├── 빛나는/날카로운 눈
├── 날카로운 이빨/발톱 표현
├── 각지고 위협적인 실루엣
├── 어두운 색상 팔레트
├── 손그림 스토리북 느낌
└── 악당답고 적대적인 외형
```

### 티어별 크기 비교
| 티어 | 크기 (캐릭터 대비) | 디테일 수준 |
|------|------------------|------------|
| Tier 1 | 45-70% | 단순, 야생적 |
| Tier 2 | 80-120% | 중간, 위협적 장식 |
| Tier 3 | 130-180% | 복잡, 압도적 장식 |

### 상태 변화 공통
| 상태 | 시각 변화 |
|------|----------|
| 버프 | 어두운 붉은/검은 글로우 |
| 디버프 | 독성 녹색 연기 |
| 분노 | 붉은 화염 + 뿔 강화 |
| 약화 | 균열/상처 증가 |
