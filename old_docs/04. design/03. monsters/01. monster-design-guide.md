# 몬스터 디자인 가이드

## 개요

이 문서는 게임 내 적 몬스터의 시각적 디자인 원칙을 정의합니다. 몬스터는 캐릭터와 대비되는 **악당/적대적 스타일**의 카드 형태로 표현됩니다.

---

## 몬스터 디자인 철학

### 핵심 원칙
```
1. 위협적: 악당답고 적대적인 분위기의 몬스터
2. 세계관 통합: 어두운 동화 (dark fairytale) 분위기
3. 가독성: 전투 중 빠른 인식 가능
4. 개성: 각 몬스터만의 악당다운 특징
```

### 악당 몬스터 접근
```
특징:
├── 위협적 실루엣: 날카로운 특징 강조 (teeth, claws, horns)
├── 과장된 비율: 위협 요소 강조 (큰 이빨, 날카로운 눈)
├── 표정: 적대적이고 악당스러운 표정 (menacing, feral)
├── 방향: 몸이 좌측 대각선을 향함
└── 세계관 소속감: 어두운 동화 세계의 적
```

### 방향 규칙 (필수!)
```
몬스터 방향:
├── 몸 방향: 좌측 대각선 방향 (필수!)
├── 시선: 좌측 전방 대각선
├── 각도: 3/4 뷰 (약간 측면, 대각선)
└── 이유: 캐릭터(우측 대각선)와 대칭 구도
```

> **중요**: 모든 몬스터 카드는 몸이 좌측 대각선 방향을 향하도록 생성되어야 합니다.

## 몬스터 티어 시스템

### 티어별 위협 표현

#### Tier 1: 일반 몬스터 (Normal)
| 요소 | 특징 |
|------|------|
| 크기 | 캐릭터보다 작음 |
| 색상 | 어두운 자연색 (탁한 녹색, 갈색) |
| 디테일 | 단순하지만 날카로운 특징 |
| 실루엣 | 야생적이고 위협적 형태 |
| 표정 | 야생적, 악의적 (feral, malicious) |

```
예시: 사악한 고블린, 독슬라임, 광폭한 늑대
색상: Dark Green #2E8B57, Murky Brown #8B7355
눈 악센트: Glowing Yellow #FFD700
```

#### Tier 2: 정예 몬스터 (Elite)
| 요소 | 특징 |
|------|------|
| 크기 | 캐릭터와 비슷한 크기 |
| 색상 | 더 진하고 어두운 색조 |
| 디테일 | 위협적인 장식 요소 |
| 실루엣 | 각지고 날카로운 형태 |
| 표정 | 교활하고 계산적 (calculating, cunning) |

```
예시: 흉악한 오크, 파괴 골렘, 원한의 유령
색상: Dark Purple #4B0082, Dark Teal #008080
눈 악센트: Piercing Red #FF4500
```

#### Tier 3: 보스 몬스터 (Boss)
| 요소 | 특징 |
|------|------|
| 크기 | 캐릭터보다 큼 |
| 색상 | 강렬하고 어두운 색상 |
| 디테일 | 위압적인 장식, 왕관/뿔 등 |
| 실루엣 | 압도적이고 지배적 형태 |
| 표정 | 압도적, 지배적 (dominating, overwhelming) |

```
예시: 사악한 드래곤, 타락한 마왕, 암흑 기사
색상: Blood Red #8B0000, Cursed Gold #B8860B, Dark Purple #301934
눈 악센트: Glowing Crimson #DC143C
```

## 지역별 몬스터 테마

### 숲 지역 (Forest)

#### 디자인 테마
```
숲 몬스터 특징:
├── 야생적이고 원시적 생물 기반
├── 어두운 녹색/갈색 팔레트
├── 이끼, 덩굴, 뿌리 장식
├── 그림자 속 포식자 느낌
└── 덤불에서 기습하듯 등장
```

## 몬스터 카드 구조

### 카드 레이아웃 (빈티지 양피지 스타일)
```
   ╭───────────────────────╮  ← 둥근 사각형
  ╱│                       │╲ ← 닳고 해진 모서리
 │ │   질감 있는 베이지    │ │
 │ │   수채화 종이 결      │ │
 │ │   ┌─────────┐        │ │
 │ │   │ 악당 몬스터│        │ │ ← 몬스터 일러스트
 │ │   │ (위협적)  │        │ │    (빛나는 눈!)
 │ │   │ (날카로움)│        │ │
 │ │   └─────────┘        │ │
 │ ├───────────────────────┤ │ ← 얇은 갈색 구분선
 │ │  ❤️❤️❤️  [의도]      │ │ ← HP, 의도 (하단)
  ╲│                       │╱ ← 닳고 해진 모서리
   ╰───────────────────────╯  ← 둥근 사각형
```

### 프레임/배경 스타일 (몬스터 전용)
| 요소 | 스타일 |
|------|--------|
| 테두리 | 낡은 양피지 테두리 + 덩굴/나뭇잎 장식 |
| 테두리 색상 | 어두운 갈색 (#6B5642) + 녹색 포인트 |
| 형태 | 둥근 사각형 |
| 모서리 장식 | 덩굴/나뭇잎 무늬 (캐릭터: 우아한 곡선) |
| 모서리 | 빈티지한 닳고 해진 종이 |
| 배경색 | 약간 어두운 베이지색 |
| 배경 질감 | 수채화 종이 결, 숲속 느낌 |
| 분위기 | 오래된 종이, 자연적/야생적 느낌 |

### 캐릭터 vs 몬스터 테두리 차이
| 요소 | 캐릭터 카드 | 몬스터 카드 |
|------|------------|------------|
| 모서리 장식 | 우아한 곡선 스크롤 | 덩굴/나뭇잎 무늬 |
| 테두리 톤 | 밝은 갈색 | 어두운 갈색+녹색 |
| 전체 느낌 | 영웅/모험가 | 숲속/야생 생물 |

### 의도 표시 시스템
```
공격 의도:
├── 아이콘: 귀여운 검 (따뜻한 빨강)
├── 데미지 숫자 표시
└── 작은 별 반짝임 효과

방어 의도:
├── 아이콘: 귀여운 방패 (하늘색)
├── 블록 숫자 표시
└── 부드러운 빛 효과

버프 의도:
├── 아이콘: 귀여운 화살표 (초록)
├── 버프 아이콘 미리보기
└── 반짝이는 별 효과

특수 의도:
├── 아이콘: 귀여운 물음표 (보라)
├── 알 수 없는 행동
└── 신비로운 반짝임
```

## 표현 가이드라인

### 몬스터 표정/동작
```
일반 몬스터 (Tier 1):
├── 야생적이고 본능적 표정 (feral, wild)
├── 공격 준비 자세
├── 빛나는 눈 (glowing eyes)
└── 날카로운 이빨/발톱 노출

정예 몬스터 (Tier 2):
├── 교활하고 계산적 표정 (cunning, calculating)
├── 위협적인 포즈
├── 날카롭게 빛나는 눈
└── 잔인한 미소

보스 몬스터 (Tier 3):
├── 압도적이고 지배적 표정 (dominating, overwhelming)
├── 군림하는 포즈
├── 강렬하게 빛나는 눈
└── 사악한 카리스마
```

### 크기 대비
```
화면 내 상대적 크기:
├── Tier 1: 플레이어 카드의 80-100%
├── Tier 2: 플레이어 카드의 100-120%
└── Tier 3: 플레이어 카드의 120-150%
```
