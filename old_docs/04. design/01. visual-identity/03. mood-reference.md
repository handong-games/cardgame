# 무드보드 및 참조

## 개요

이 문서는 게임의 시각적 방향성을 설정하기 위한 참조 자료와 무드보드를 정리합니다.

---

## 핵심 무드 키워드

### 분위기 정의
```
Warm Cozy Fairytale (따뜻한 동화)
├── 포근하고 아기자기한 세계
├── 손그림 느낌의 따뜻함
├── 귀여운 캐릭터들의 모험
└── 어린이 그림책 같은 친근함
```

### 감정 목표
| 게임 상황 | 목표 감정 | 시각적 표현 |
|-----------|-----------|-------------|
| 탐험 중 | 설렘 + 호기심 | 따뜻한 톤, 부드러운 빛 |
| 전투 시작 | 긴장 + 용기 | 밝은 색상 유지, 살짝 강조 |
| 코인 뒤집기 | 두근두근 기대 | 황금/은색 반짝임 |
| 승리 | 뿌듯함 | 밝은 황금빛, 파티클 |
| 패배 | 아쉬움 | 살짝 채도 감소 (어둡지 않게) |

## 주요 참조: Dice & Fold

### 채택할 핵심 요소
```
Dice & Fold 스타일:
├── 손그림 스토리북 느낌
├── 과장된 비율 (큰 머리)
├── 심플한 나무/갈색 프레임
├── 하단 HP 표시 영역
├── 따뜻하고 아기자기한 느낌
└── 캐릭터들의 친근한 표정
```

### 스타일 분석
| 요소 | Dice & Fold 특징 | 우리 게임 적용 |
|------|-----------------|----------------|
| 선 | 부드러운 연필 스케치 | 동일 적용 |
| 채색 | 수채화 느낌 | 동일 적용 |
| 비율 | SD 비율 (큰 머리) | 동일 적용 |
| 프레임 | 심플한 나무 | 동일 적용 |
| 분위기 | 따뜻하고 귀여움 | 동일 적용 |
| HP 표시 | 하단 별도 영역 | 동일 적용 |

## 보조 참조

### 어린이 그림책 일러스트
```
참조 특징:
├── 귀엽고 친근한 캐릭터
├── 볼에 홍조
├── 큰 눈, 따뜻한 표정
├── 부드러운 수채화 채색
└── 심플하고 명확한 형태
```

### 손그림 카드 게임
```
참조 요소:
├── 양피지 느낌의 배경
├── 나무 프레임
├── 금색 테두리 장식
├── 가독성 좋은 정보 배치
└── 따뜻한 전체 색조
```

---

## 시각적 모티프

### 반복 모티프
| 모티프 | 의미 | 표현 방식 |
|--------|------|-----------|
| 태양 | 희망, 행운 | 따뜻한 황금빛, 둥글게 |
| 달 | 도전, 미지 | 부드러운 은빛, 초승달 |
| 하트 | 체력, 생명 | 귀여운 하트 모양 |
| 별 | 보상, 희망 | 반짝이는 별 모양 |
| 잎/꽃 | 자연, 성장 | 부드러운 곡선 |

### 프레임 장식
```
심플한 장식:
├── 나무 질감 (간결하게)
├── 모서리 작은 잎/덩굴 (선택적)
├── 금색 얇은 테두리
└── 과하지 않은 심플함 유지
```

## 조명 분위기

### 기본 조명 설정
```
따뜻한 자연광:
├── 색온도: 따뜻한 황색 (4000-5000K)
├── 방향: 좌상단 부드럽게
├── 강도: 부드럽고 포근하게
└── 그림자: 짧고 따뜻하게
```

### 상황별 조명 변화
| 상황 | 조명 특성 | 효과 |
|------|-----------|------|
| 평화로운 탐험 | 따뜻하고 밝음 | 안정감 |
| 전투 진입 | 살짝 생동감 추가 | 긴장감 (밝음 유지) |
| 승리 순간 | 황금빛 확산 | 성취감 |
| 코인 뒤집기 | 중앙 집중 조명 | 집중 |

## 세계관 비주얼

### 전체 분위기
```
따뜻한 동화 세계:
├── 친근하고 포근한 분위기
├── 위험도 귀여운 방식으로 표현
├── 밝은 색조 유지
└── 어둡거나 무서운 요소 최소화
```

### 지역별 분위기 (따뜻하게 유지)
```
숲 지역 (Forest):
├── 밝은 초록/갈색 톤
├── 햇살 비치는 숲
├── 귀여운 동물 느낌 몬스터
└── 무드: 상쾌하고 모험적

던전 지역 (Dungeon):
├── 따뜻한 갈색/주황 (횃불)
├── 신비로운 분위기 (무섭지 않게)
├── 보물상자, 열쇠 등
└── 무드: 탐험적, 호기심

성 지역 (Castle):
├── 따뜻한 금색/붉은색
├── 화려하지만 친근하게
├── 왕관, 보석 등
└── 무드: 장엄하지만 따뜻하게
```
