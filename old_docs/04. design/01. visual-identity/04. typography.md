# 타이포그래피 가이드

## 개요

이 문서는 게임의 폰트 시스템과 텍스트 스타일을 정의합니다. 모든 텍스트는 동화책 느낌의 따뜻하고 친근한 분위기를 유지합니다.

---

## 폰트 선정 원칙

### 스타일 방향
```
폰트 선정 기준:
├── 따뜻하고 친근한 느낌
├── 동화책/그림책 분위기
├── 손글씨 또는 라운드 형태
├── 가독성 우선
├── 다국어 지원 (한글/영문)
└── 라이선스 확인 필수
```

### 피해야 할 폰트 스타일
```
금지 폰트 유형:
├── 고딕/산세리프 딱딱한 폰트
├── 공포/호러 느낌의 폰트
├── 기계적이고 차가운 폰트
├── 지나치게 장식적인 폰트
└── 가독성 낮은 필기체
```

## 텍스트 스타일

### 굵기 (Font Weight)

| 굵기 | 값 | 용도 |
|------|-----|------|
| Regular | 400 | 본문, 설명 |
| Medium | 500 | 강조, 라벨 |
| SemiBold | 600 | 버튼, 제목 |
| Bold | 700 | 중요 정보, 숫자 |
| ExtraBold | 800 | 매우 중요한 제목 |

### 자간 (Letter Spacing)

| 용도 | 자간 | 설명 |
|------|------|------|
| 제목 (큰) | -0.02em | 살짝 좁게 |
| 제목 (작은) | -0.01em | 약간 좁게 |
| 본문 | 0em | 기본 |
| 버튼/라벨 | 0.02em | 살짝 넓게 |
| 강조 숫자 | 0.05em | 넓게 |

### 줄 간격 (Line Height)

| 용도 | 줄 간격 | 설명 |
|------|---------|------|
| 제목 | 1.2 - 1.3 | 조밀하게 |
| 본문 (짧은) | 1.4 | 중간 |
| 본문 (긴) | 1.6 | 여유롭게 |
| 버튼 | 1.0 | 한 줄 |

## 텍스트 효과

### 외곽선 (Outline)

#### 카드 이름/중요 텍스트
```
외곽선 스타일:
├── 색상: Warm Brown (#4A3728)
├── 두께: 2px
├── 스타일: 부드러운 둥근 외곽선
└── 용도: 카드 이름, HP, 데미지 숫자
```

#### 데미지 숫자
```
데미지 숫자 외곽선:
├── 내부: Warm Red (#CD5C5C)
├── 외곽선 1: Cream (#FFFDD0) 3px
├── 외곽선 2: Dark Wood (#3A2818) 2px
└── 효과: 그림자 (2px, 2px, Dark Brown)
```

### 그림자 (Shadow)

#### 제목용 그림자
```
부드러운 제목 그림자:
├── X 오프셋: 2px
├── Y 오프셋: 2px
├── 블러: 4px
├── 색상: Warm Brown (#4A3728) 30%
└── 용도: 큰 제목, 게임 로고
```

#### UI 텍스트 그림자
```
미세한 UI 그림자:
├── X 오프셋: 1px
├── Y 오프셋: 1px
├── 블러: 2px
├── 색상: Warm Brown (#4A3728) 20%
└── 용도: 버튼 텍스트, 라벨
```

### 글로우 (Glow)

#### 마법/특수 텍스트
```
따뜻한 글로우:
├── 색상: Warm Gold (#FFD700)
├── 블러: 8px
├── 확산: 4px
├── 애니메이션: 부드러운 펄스 (2초 주기)
└── 용도: 레벨업, 전직, 보상
```

#### 긍정적 효과
```
녹색 글로우:
├── 색상: Soft Green (#90EE90)
├── 블러: 6px
├── 확산: 2px
└── 용도: 힐링, 버프 획득
```

## 숫자 표현

### 데미지 숫자
```
데미지 표시:
├── 폰트: DNF Bit Bit 또는 굵은 제목 폰트
├── 크기: 32px ~ 48px
├── 색상: Warm Red (#CD5C5C)
├── 외곽선: Cream 3px + Dark Wood 2px
├── 애니메이션: 위로 튀어오름 + 페이드
└── 크리티컬: 1.5x 크기 + 금색 + 흔들림
```

### 힐 숫자
```
회복 표시:
├── 폰트: 동일
├── 크기: 28px ~ 40px
├── 색상: Soft Green (#90EE90)
├── 외곽선: Cream 2px + Dark Wood 1px
├── 애니메이션: 부드럽게 위로 떠오름
└── 파티클: 작은 하트/잎 동반
```

### HP/에너지 바 숫자
```
상태 바 숫자:
├── 폰트: Bold 본문 폰트
├── 크기: 14px ~ 18px
├── 형식: "현재/최대" (예: 85/100)
├── 색상: Cream (바 위) 또는 Warm Brown (바 옆)
└── 업데이트: 숫자 롤링 애니메이션
```

### 코스트/가격 숫자
```
비용 표시:
├── 폰트: Bold 제목 폰트
├── 크기: 20px ~ 24px
├── 색상: Warm Gold (#FFD700)
├── 위치: 카드 좌상단 원 안
└── 부족 시: Gray + 취소선
```

```
├── 폰트: 제목용 한글 폰트

├── 굵기: Bold (700)

├── 외곽선: 있음 (배경 대비)

```

### 카드 설명
```
├── 폰트: 본문용 폰트
├── 크기: 12px ~ 14px
├── 굵기: Regular (400)
├── 색상: Warm Brown (#4A3728)
├── 정렬: 중앙 또는 왼쪽
├── 행간: 1.4
└── 최대 줄: 3줄 권장
```

### 카드 수치
```
├── 데미지/힐: Bold, 색상 강조
├── 키워드: SemiBold, 밑줄 또는 색상
├── 숫자: Bold, 크기 살짝 크게
└── 예: "3 데미지를 입히고 2 블록을 얻습니다"
```

## 접근성 고려

### 최소 폰트 크기
| 플랫폼 | 본문 최소 | 캡션 최소 |
|--------|----------|----------|
| 모바일 | 14px | 10px |
| 태블릿 | 16px | 12px |
| 데스크톱 | 16px | 12px |

### 대비 비율
| 용도 | 최소 비율 | 권장 비율 |
|------|----------|----------|
| 본문 텍스트 | 4.5:1 | 7:1 |
| 대형 텍스트 (24px+) | 3:1 | 4.5:1 |
| UI 컴포넌트 | 3:1 | 4.5:1 |

### 가독성 체크
```
가독성 확인:
├── 배경과 텍스트 대비 충분한가?
├── 폰트 크기가 읽기 편한가?
├── 줄 간격이 답답하지 않은가?
├── 줄 길이가 너무 길지 않은가? (최대 80자)
└── 중요 정보가 강조되어 있는가?
```

## 폰트 로딩 전략

### 웹/앱 폰트 로딩
```
폰트 로딩 우선순위:
├── 1순위: 시스템 폰트 (즉시 표시)
├── 2순위: 본문 폰트 (중요)
├── 3순위: 제목 폰트
├── 4순위: 특수 폰트
└── 전략: FOUT (Flash of Unstyled Text) 허용
```

### 폰트 서브셋
```
용량 최적화:
├── 한글: 자주 사용하는 2,350자
├── 영문: 기본 문자셋
├── 숫자: 0-9
├── 특수문자: 게임에서 사용하는 것만
└── 예상 용량: 폰트당 100-300KB
```
