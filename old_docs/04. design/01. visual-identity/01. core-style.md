# 핵심 스타일 가이드

## 개요

이 문서는 게임의 핵심 비주얼 스타일을 정의합니다. 모든 에셋은 이 가이드라인을 기반으로 제작됩니다.

---

## 스타일 정의

### 그림체: 손그림 스토리북 일러스트레이션

```
핵심 특징:
├── 연필 스케치 기반 (Pencil sketch base)
├── 수채화 채색 (Watercolor coloring)
├── 손으로 그린 느낌 (Hand-drawn feel)
├── 따뜻하고 부드러운 선 (Warm soft lines)
└── 동화책 일러스트 분위기 (Storybook illustration)
```

### 참조 스타일
- **Dice & Fold**: 따뜻한 스토리북 카드 게임
- **전통 동화책 일러스트**: 포근하고 아기자기한 느낌
- **어린이 그림책**: 귀엽고 친근한 캐릭터

---

## 선 (Line Work)

### 연필 스케치 스타일
| 요소 | 특징 |
|------|------|
| 선 질감 | 연필로 그린 듯한 부드러운 질감 |
| 선 색상 | 순수 검정 대신 따뜻한 갈색/세피아 톤 |
| 선 두께 | 균일하지 않은 자연스러운 변화 |
| 마무리 | 약간 거친 끝, 완벽하지 않은 느낌 |

### 윤곽선 규칙
| 요소 | 선 두께 | 특징 |
|------|---------|------|
| 캐릭터 외곽 | 중간 (2-3px) | 부드럽고 따뜻한 갈색 |
| 내부 디테일 | 얇음 (1px) | 섬세하지만 과하지 않게 |
| 프레임 | 굵음 (3-4px) | 심플한 나무 질감 |

## 채색 (Coloring)

### 수채화 스타일
```
수채화 채색 특징:
├── 부드러운 그라데이션: 색상 경계가 자연스럽게 번짐
├── 투명한 레이어: 겹쳐 칠한 느낌
├── 따뜻한 색조: 전체적으로 따뜻한 톤
├── 하이라이트: 흰색을 남겨둔 듯한 밝은 부분
└── 볼 홍조: 캐릭터에 따뜻한 볼 홍조 추가
```

### 채색 우선순위
1. **캐릭터 피부**: 따뜻한 살구색 + 볼 홍조
2. **의상/장비**: 클래스별 색상 + 수채화 텍스처
3. **배경**: 양피지 톤 + 부드러운 그라데이션
4. **프레임**: 나무/갈색 + 금색 하이라이트

---

## 비율 (Proportions)

### SD (Super Deformed) 비율
```
귀여운 SD 캐릭터 비율:
├── 전체 등신: 2~2.5 등신
├── 머리: 전체의 40-50% (매우 큼)
├── 몸통: 전체의 25-30%
├── 다리: 전체의 20-25% (짧게)
└── 손/발: 작고 귀엽게 단순화
```

### 얼굴 비율
```
귀여운 얼굴 표현:
├── 눈: 얼굴의 40-50% (크고 동그랗게)
├── 코: 매우 작게 또는 점으로
├── 입: 작고 귀엽게
├── 볼: 항상 볼 홍조 추가
└── 표정: 따뜻하고 친근한 표정
```

## 명암 처리 (Lighting & Shading)

### 조명 방향
- **기본 광원**: 좌상단 45도 (←↖)
- **부드러운 조명**: 강한 대비 없이 따뜻하게

### 명암 단계
```
부드러운 2-3단계 명암:
├── 하이라이트 (Highlight): 흰색/밝은 부분, 15-20%
├── 미드톤 (Midtone): 기본 색상, 60-70%
└── 그림자 (Shadow): 따뜻한 그림자, 15-20%
```

### 그림자 스타일
- **부드러운 경계**: 수채화처럼 자연스럽게 번짐
- **따뜻한 그림자**: 차가운 색 대신 따뜻한 갈색/보라 사용
- **투명한 느낌**: 그림자도 살짝 투명하게

## 프레임 디자인

### 카드 프레임
```
심플한 나무 프레임:
├── 재질: 따뜻한 나무 질감
├── 색상: 갈색 (#8B4513) 베이스
├── 장식: 최소한, 심플하게
├── 모서리: 부드럽게 둥근 모서리
└── 하단: HP 표시 영역 확보
```

### 프레임 구조
```
┌─────────────────────────┐
│ ╔═══════════════════╗  │ ← 나무 프레임 상단
│ ║                   ║  │
│ ║    캐릭터         ║  │ ← 양피지 배경
│ ║    일러스트       ║  │
│ ║                   ║  │
│ ╠═══════════════════╣  │ ← 구분선
│ ║   HP: ❤️❤️❤️     ║  │ ← HP 표시 영역
│ ╚═══════════════════╝  │ ← 나무 프레임 하단
└─────────────────────────┘
```
