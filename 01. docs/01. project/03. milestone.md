# 마일스톤

## 개요

| 항목 | 내용 |
|------|------|
| **개발 기간** | 4주 |
| **주간 투입 시간** | 10-20시간 (파트타임) |
| **총 예상 시간** | 40-80시간 |
| **검증 목표** | 전직 시스템이 덱빌딩에 전략적 변화를 주는가? |

---

## 주간 마일스톤

### 1주차: 기본 전투 루프

**목표**: 카드 플레이 → 적 공격 → 턴 종료

**산출물**:
- 카드 데이터 구조 (ScriptableObject)
- 덱 시스템 (드로우, 셔플, 버리기)
- 에너지 시스템 (턴당 3 에너지)
- HP 시스템
- 기본 전투 UI

**검증 기준**:
- [ ] 카드를 플레이할 수 있음
- [ ] 적이 공격을 수행함
- [ ] 턴 종료가 정상 동작함

---

### 2주차: 라운드 진행

**목표**: 고정 전투 + 카드 선택 라운드

**산출물**:
- 라운드 1-4 진행 시스템
- 카드 선택 UI (3장 중 1장)
- 적 2종 (일반몬, 엘리트)
- 기본 카드 5-7장

**검증 기준**:
- [ ] 라운드 1-4까지 진행 가능
- [ ] 카드 선택 후 덱에 추가됨
- [ ] 선택한 카드에 따라 다음 라운드 내용 결정

---

### 3주차: 전직 시스템

**목표**: 팔라딘 오라 + 고유 카드

> 📋 전직 시스템 상세 스펙은 [스코프 보드](./02.%20scope-board.md#전직-시스템-상세-스코프) 참조

**검증 기준**:
- [ ] 전직 후 오라가 발동됨
- [ ] 팔라딘 카드 사용 가능
- [ ] 전직 전후로 플레이 느낌이 다름

---

### 4주차: 통합 및 보스

**목표**: 5라운드 전체 플로우, 보스전

**산출물**:
- 보스 (복합 패턴, 높은 HP)
- 5라운드 전체 시퀀스
- 승리/패배 화면

**검증 기준**:
- [ ] 5라운드 전체 플로우 완주 가능
- [ ] 보스전이 도전적임
- [ ] 전직이 전략적 변화를 주는지 확인 가능
