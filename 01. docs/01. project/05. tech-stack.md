## 2. 핵심 기술 스택

### 2.1 빌드 도구: Vite `5.x`

개발 서버 시작 속도가 빠르고, TypeScript 기본 지원. 설정 거의 불필요.

### 2.2 UI 라이브러리: React 18 `18.x`

컴포넌트 기반 UI → 카드, HP바 등 재사용 용이. 게임 상태 변경 시 화면 자동 갱신.

### 2.3 타입 시스템: TypeScript `5.x`

카드/게임 상태 등 복잡한 데이터 구조를 안전하게 관리. IDE 자동완성 지원.

---

## 3. 상태 관리

### Zustand `4.x` + Immer `10.x`

**Zustand**: Redux보다 간단하고, React 외부에서도 상태 접근 가능 (게임 로직 분리에 유리)

**Immer**: 중첩된 상태 업데이트를 간결하게 작성

```typescript
// Immer 없이 (복잡)
setState({ ...state, battle: { ...state.battle, player: { ...state.battle.player, hp: hp - 10 }}});

// Immer 사용 (간결)
setState(draft => { draft.battle.player.hp -= 10; });
```

## 4. 스타일링

### Tailwind CSS `3.x`

유틸리티 클래스 기반 → CSS 파일 없이 빠른 프로토타이핑. Claude가 클래스 작성 지원.

```tsx
// 카드 컴포넌트 예시
<div className="w-32 h-44 rounded-lg border-2 border-red-500 bg-red-50 p-2">
  카드
</div>
```

## 8. 프로젝트 생성 명령어

```bash
# 프로젝트 생성 및 이동
npm create vite@latest cardgame-web -- --template react-ts
cd cardgame-web

# 의존성 설치
npm install zustand immer framer-motion
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# 개발 서버 실행
npm run dev
```

> localhost:5173 접속 시 Vite 기본 화면이 보이면 성공
