# 코인 영역 UI

## 개요

전투 화면 하단에 위치하는 코인 자원 UI 레이아웃 정의

---

## 해상도별 높이

| 해상도 | SafeArea 높이 | 코인 영역 높이 | 비율 |
|--------|--------------|---------------|------|
| 640×480 | 480px | 128px | 약 27% |
| 1920×1080 | 1080px | 288px | 약 27% |
| 2560×1440 | 1440px | 384px | 약 27% |
| 3440×1440 | 1440px | 384px | 약 27% |

---

## 레이아웃 구조

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│                  앞면: {값} / 뒷면: {값}                   │  ← 상단 중앙
│                                                         │
│       (비움)      [동전 애니메이션 영역]      [버튼]        │  ← 코인 영역
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

## 구성 요소

| 위치 | 요소 |
|-----|------|
| 상단 중앙 | 코인 가치 UI |
| 중앙 | 동전 애니메이션 영역 |
| 좌측 | (비움) |
| 우측 | 토글 버튼 |

---

## 코인 가치 UI

- 형식: `앞면: {앞면 값어치} / 뒷면: {뒷면 값어치}`
- 예시: "앞면: 3 / 뒷면: 5"
- 위치: 상단 중앙

---

## 토글 버튼

- 토스 전: "코인 토스" 버튼
- 토스 후: "턴 종료" 버튼으로 변경
- 토스 후 스킬 사용 없이 바로 턴 종료 가능
- 위치: 우측
