# 스킬 툴팁 UI

## 개요

스킬 슬롯 호버 시 표시되는 상세 정보 패널 정의

---

## 트리거

- 마우스 호버 (0.3초 딜레이)
- 모바일: 롱 프레스

---

## 위치

- 스킬 슬롯 상단에 표시
- 화면 경계 감지하여 좌/우 조정
- 슬롯과 8px 간격

---

## 레이아웃 구조

```
┌────────────────────────────┐
│ [아이콘] 스킬명             │
├────────────────────────────┤
│ 설명 텍스트                 │
│ (여러 줄 가능)              │
├────────────────────────────┤
│ 🪙 비용: 3                  │
│ 🔄 턴당: 2회                │
│ ⏱️ 쿨다운: 없음             │
│ 🎯 타겟: 적 1명             │
├────────────────────────────┤
│ ⚔️ 피해: 8                  │
└────────────────────────────┘
```

---

## 표시 항목

### 헤더

| 요소 | 설명 |
|-----|------|
| 아이콘 | 스킬 아이콘 (32×32px) |
| 스킬명 | 굵은 텍스트, 14pt |

### 설명

- 스킬 효과 설명
- 최대 3줄, 초과 시 말줄임

### 속성

| 항목 | 표시 형식 |
|-----|----------|
| 비용 | 🪙 비용: N |
| 사용 제한 | 🔄 턴당: N회 |
| 쿨다운 | ⏱️ 쿨다운: N턴 (없으면 "없음") |
| 타겟 | 🎯 타겟: 자신/적 1명/모든 적 |

### 수치 (해당 시)

| 항목 | 표시 형식 |
|-----|----------|
| 피해 | ⚔️ 피해: N |
| 방어 | 🛡️ 방어: N |
| 회복 | 💚 회복: N |

---

## 스타일

| 속성 | 값 |
|-----|-----|
| 배경 | 반투명 검정 (rgba 0,0,0,0.85) |
| 테두리 | 1px 밝은 회색 |
| 모서리 | 8px 라운드 |
| 패딩 | 12px |
| 최대 너비 | 240px |
