# 스킬 시스템

## 개요

카드 덱빌딩 시스템을 대체하는 스킬 기반 액션 시스템

## 핵심 개념

### 카드 → 스킬 전환

| 기존 (카드) | 변경 (스킬) |
|------------|------------|
| 덱에서 드로우 | 모든 스킬 항상 사용 가능 |
| 에너지 비용 | 코인 비용 |
| 덱/손패/버리기 | 슬롯 기반 (제한 없음) |

### 스킬 특성

- **기본 스킬**: 클래스별 고정 스킬 세트로 시작
- **스킬 획득**: 전투 보상 선택 + 상점 구매
- **비용**: 코인 토스로 획득한 코인 소비
- **제한**: 턴당 사용 횟수 제한 (스킬별), 쿨다운 (옵션)
- **슬롯**: 무제한 - 획득한 모든 스킬 사용 가능

## 게임 흐름

```
턴 시작 → 코인 토스 → 스킬 사용 (반복) → 턴 종료
```

## 클래스별 설계 방향

| 클래스 | 기본 스킬 | 특징 |
|--------|----------|------|
| 워리어 | 공격, 방어, 강타 | 균형잡힌 공방 |
| 팔라딘 | 공격, 방어, 치유 | 회복 특화 |
