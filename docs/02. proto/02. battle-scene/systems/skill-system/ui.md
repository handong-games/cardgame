# 스킬 UI

## 개요

스킬 바 레이아웃과 스킬 슬롯의 시각적 표현 정의

---

## 스킬 바 위치

- 코인 영역 바로 위 (전투 영역 내 하단)
- 수평 중앙 정렬
- 코인 영역과 8~16px 간격

---

## 스킬 바 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│                      전투 영역                            │
├─────────────────────────────────────────────────────────┤
│       [스킬][스킬][스킬][스킬]... (수평 중앙, 최대 2줄)      │  ← 스킬 바
├── 8~16px 간격 ──────────────────────────────────────────┤
│                      코인 영역                            │
└─────────────────────────────────────────────────────────┘
```

| 속성 | 값 |
|-----|-----|
| 배치 | 수평 일렬, 중앙 정렬 |
| 오버플로우 | 2줄로 자동 확장 |

---

## 스킬 슬롯 구조

```
┌─────────────────────┐
│    ┌───────────┐    │
│    │           │    │
│    │   아이콘   │    │
│    │           │    │
│    └───────────┘    │
│      [코인 비용]      │
└─────────────────────┘
```

---

## 크기

| 해상도 | 슬롯 크기 | 아이콘 크기 |
|--------|----------|------------|
| 1920×1080 | 80×96px | 64×64px |
| 1280×720 | 60×72px | 48×48px |

---

## 상태별 표시

### 사용 가능

| 요소 | 스타일 |
|-----|--------|
| 아이콘 | 원본 색상 |
| 테두리 | 밝은 테두리 |
| 비용 | 흰색 텍스트 |

### 코인 부족

| 요소 | 스타일 |
|-----|--------|
| 아이콘 | 회색조 (grayscale) |
| 테두리 | 어두운 테두리 |
| 비용 | 빨간색 텍스트 |

### 사용 횟수 소진

| 요소 | 스타일 |
|-----|--------|
| 아이콘 | 회색조 + 반투명 |
| 오버레이 | "X" 표시 또는 사선 |
| 비용 | 회색 텍스트 |

### 쿨다운 중

| 요소 | 스타일 |
|-----|--------|
| 아이콘 | 시계방향 쿨다운 오버레이 |
| 텍스트 | 남은 턴 수 표시 |
| 비용 | 회색 텍스트 |

---

## 비용 표시

```
┌──────┐
│  3   │  ← 코인 아이콘 + 숫자
└──────┘
```

- 위치: 슬롯 하단 중앙
- 아이콘: 작은 코인 이미지 (선택사항)
- 숫자: 코인 비용

---

## 호버 상태

마우스 오버 시:
- 슬롯 확대 (1.1배)
- 밝기 증가
- [툴팁 표시](./ui-tooltip.md)

---

## 드래그 인터랙션

### 드래그 시작
- 슬롯에서 아이콘 분리
- 원본 위치에 반투명 고스트 표시

### 드래그 중
- 커서 따라 아이콘 이동
- 유효 타겟 하이라이트

### 드롭
- 유효 타겟: 스킬 사용
- 무효 영역: 원위치 복귀
