# 스킬 바 UI

## 개요

전투 화면에서 스킬 아이콘을 표시하는 액션 바 레이아웃 정의

---

## 위치

- 코인 영역 바로 위 (전투 영역 내 하단)
- 수평 중앙 정렬
- 코인 영역과 8~16px 간격

---

## 레이아웃 구조

```
┌─────────────────────────────────────────────────────────┐
│                      전투 영역                            │
├─────────────────────────────────────────────────────────┤
│       [스킬][스킬][스킬][스킬]... (수평 중앙, 최대 2줄)      │  ← 스킬 바
├── 8~16px 간격 ──────────────────────────────────────────┤
│                      코인 영역                            │
└─────────────────────────────────────────────────────────┘
```

---

## 스킬 아이콘

| 속성 | 값 |
|-----|-----|
| 크기 | 80×80px (1920×1080 기준) |
| 배치 | 수평 일렬, 중앙 정렬 |
| 오버플로우 | 2줄로 자동 확장 |

---

## 아이콘 표시 정보

- 스킬 이미지 (아이콘)
- 코인 비용 (하단)
- 활성화/비활성화 상태

---

## 비활성화 상태

- 회색 처리 (grayscale)
- 조건: 코인 부족, 사용 횟수 소진, 쿨다운 중

---

## 툴팁 (마우스 호버)

- 스킬명
- 설명
- 코인 비용
- 턴당 사용 제한
- 쿨다운
- 타겟 타입
- 데미지/회복/방어 수치

---

## 스킬 드래그 & 프리뷰

- 스킬 사용: 드래그하여 타겟(플레이어/적)에 지정
- 프리뷰 시스템: [card-preview.md](../systems/card-preview.md) 참조
