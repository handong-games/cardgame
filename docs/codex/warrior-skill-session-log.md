# 전사 스킬 설계 기록 (세션 요약)

목적
- 전사 기본 스킬 3개를 확정하고, 설계 원칙을 문서화한다.
- 오늘 세션에서 결정된 기준을 남겨 다음 설계에 이어서 사용한다.

핵심 원칙
- 선택 강제 구조: 공격/방어는 필수지만, 상황별 최적이 달라지게 설계한다.
- 지루함 방지: 항상 2개만 쓰는 패턴을 막기 위해 상호 트레이드오프를 명확히 한다.
- 코인 실패 구제: 첫 턴 올-뒷면 등 최악 상황을 구제하는 유틸 스킬을 포함한다.

코스트/밸류 기준
- 상한만 고정: 1코인 ≤ 8 (하한은 자유)
- 기본 데미지는 5로 확정

전사 기본 스킬 3개 (의도 + 수치 확정)

1) 정의의 일격 (공격)
- 의도: 전사의 필수 공격 기준선
- 강점: 적 HP를 빠르게 줄여 전투 단축
- 약점: 방어가 부족해 큰 공격 턴에 취약
- 수치: 데미지 5

2) 성채의 방벽 (방어)
- 의도: 전사의 필수 생존 기준선
- 강점: 큰 공격 타이밍을 안정적으로 넘김
- 약점: 전투가 길어져 누적 리스크 증가
- 수치: 방어 7

3) 정의의 결심 (유틸, 뒷면 전용 구제)
- 의도: 코인 실패(올 뒷면) 상황의 안전장치
- 강점: 최악의 턴을 최소 손해로 넘기고 다음 턴 반등 제공
- 약점: 조건부라 자주 쓰이지 않음
- 조건: H=0일 때만 0코인 사용 가능
- 수치: 방어 3 + 다음 턴 앞면 1개 보장

결정 사항
- 기본 스킬은 초반에 선택 고민이 생기도록 설계한다.
- 보너스 효과는 현재 제거 상태로 유지한다.
- 추가 스킬(확장)은 다음 세션에서 설계한다.

오픈 항목 (다음 세션)

- 임계값/연속/보상 메커니즘을 일부 스킬에만 적용하는 기준
