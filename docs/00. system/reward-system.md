# 보상 시스템 기획서

## Part 1: 왜 만드는가 (Why)

### 1.1 기획 배경

몬스터를 처치한 후 보상을 제공함으로써 도전에 대한 성취감을 제공해야 한다. 보상 요소는 영혼(금전), 스킬, 전리품, 코인이 있으며, 몬스터 종류에 따라 보상 구성이 달라진다.

### 1.2 해결할 문제

| 문제 | 현재 상황 | 해결 방향 |
|------|----------|----------|
| **전투 보상 부재** | 몬스터를 처치해도 얻는 것이 없으면 동기 부족 | 영혼, 스킬, 전리품으로 성취감 제공 |
| **성장 수단 필요** | 스킬 시스템의 빌드 구성을 위한 스킬 획득 경로 필요 | 전투 보상으로 스킬 3개 중 1개 선택 |
| **재화 순환** | 상점 등 소비처를 위한 재화 획득 수단 필요 | 전투별 영혼 획득 |

> 핵심: 보상 시스템은 전투의 **성취감**을 제공하고, 스킬 획득과 재화 순환을 통해 **플레이어 성장의 경로**가 된다.

### 1.3 기대 효과 & 리스크

| 구분 | 내용 | 예시 |
|------|------|------|
| **기대 효과** | 새로운 스킬을 보며 전략 구상 및 기대감 | 보상에서 "앞면3 - 강타"를 발견하고 "코인이 5개로 늘면 이걸 안정적으로 쓸 수 있겠다"라고 기대하며 선택 |
| **기대 효과** | 스킬을 원하는 슬롯에 배치하며 사용 순서 전략 수립 | 1번 슬롯에 저코스트 버프, 2번에 메인 공격을 배치해 "버프 먼저 → 강화된 공격" 순서를 계획 |
| **기대 효과** | 슬롯이 꽉 찼을 때 교체 판단으로 전략의 깊이 심화 | 6슬롯이 꽉 찬 상태에서 "분노 방어"가 나왔을 때, 기존 "기본 방어"를 교체할지 고민하며 자신의 빌드가 공격 편향인지 방어 부족인지 점검 |

## Part 2: 무엇을 만드는가 (What)

### 2.1 한 줄 정의

> 보상 시스템은 **전투 승리 후 성취감 제공**을 담당하며, **스킬 선택과 슬롯 배치**를 통해 **전략을 구성하고 발전시키는 성장 경험**을 제공한다.

**예시:**
> 전사가 늑대 무리를 처치하고 영혼 30개를 획득한다. 보상 스킬 3개 중 "뒷면2 - 분노 방어"를 선택하여 4번 슬롯에 배치한다. 다음 전투에서 뒷면이 많이 나와도 방어와 공격력 버프를 동시에 얻을 수 있는 전략이 완성된다.

### 2.2 감정 설계

| 순간 | 목표 감정 | 설계 요소 | 예시 |
|------|----------|----------|------|
| 몬스터 처치 | **타격의 쾌감** | 몬스터 카드 소멸 이펙트 후 해당 위치에 영혼 드랍 | 늑대를 처치하자 카드가 소멸되며 영혼이 떨어진다 - "잡았다!" |
| 영혼 수집 | **납득 + 쾌감** | 모든 몬스터 처치 후 드랍된 영혼이 상단 자원 UI로 이동하며 수치 증가 | 엘리트 처치 후 영혼들이 위로 날아가며 40 → 90으로 증가 - "강한 적이었으니 이 정도는 받아야지" |
| 보상 화면 열림 | **성취감 → 기대감** | 영혼 수집 후 보상 화면 전환 | "승리!" 연출과 함께 보상 화면이 열리며 "뭐가 나올까" 기대 |
| 스킬 3개 확인 | **호기심 + 전략 구상** | 스킬 아이콘/효과/코스트/이름 표시, 현재 보유 스킬과 비교 가능 | "앞면3 강타"를 보고 "지금 버프 스킬이랑 조합하면 한 턴에 큰 데미지를 넣을 수 있겠다" |
| 스킬 선택 → 슬롯 배치 | **소유감 (내 선택)** | 원하는 슬롯 위치에 직접 배치 | "분노 방어"를 3번 슬롯에 배치 - "방어 스킬은 여기가 손이 편하니까" |
| 슬롯 꽉 참 → 교체 | **고민의 재미 (전략적 딜레마)** | "잃는다"가 아닌 "최적화한다"의 느낌. 교체 전후 비교 가능 | 6슬롯이 꽉 찬 상태에서 "약화 방어"를 "기본 방어" 자리에 교체 - "공격 약화가 더 도움되겠다" |
| 건너뛰기 | **판단의 확신** | 내 빌드에 불필요한 것을 과감히 넘기는 전략적 선택 | 앞면 위주 빌드인데 뒷면 스킬만 3개 나옴 - "지금 내 빌드엔 필요 없어" 과감히 패스 |
| 전리품 획득 | **희소성의 흥분** | 엘리트/보스에서만 등장, 차별화된 연출 | 엘리트 처치 후 "이끼 낀 부적" 등장 - "턴마다 방어력 +2면 후반에 크겠다" |
| 코인 획득 | **성장의 쾌감** | 보스에서만 등장, 코인 개수 증가 = 할 수 있는 행동 확장 | 보스 처치 후 코인 +1 획득 - "4개가 되면 앞면3 스킬을 더 안정적으로 쓸 수 있겠다" |

### 2.3 시스템 구조

```
전투 중 몬스터 처치
    |
[몬스터 소멸] 몬스터 카드 소멸 이펙트 → 해당 위치에 영혼 드랍
    |
모든 몬스터 처치 (전투 승리)
    |
[영혼 수집] 드랍된 영혼이 상단 영혼 자원 UI로 이동 → 수치 증가 표시
    |
[보상 화면] 전투 유형에 따라 구성이 다름
    |
    ├─ 일반 전투: 1단계
    |   └─ 영혼 획득량 + 스킬 3개 선택 (한 화면)
    |
    ├─ 엘리트 전투: 1단계
    |   └─ 영혼 획득량 + 전리품 자동 획득 표시 + 스킬 3개 선택 (한 화면)
    |
    └─ 보스 전투: 2단계
        ├─ 1단계: 영혼 획득량 + 코인 +1 + 전리품 3개 중 선택
        └─ 2단계: 스킬 3개 선택
    |
[스킬 선택 분기]
    ├─ 스킬 드래그 → 빈 슬롯 → 즉시 배치
    ├─ 스킬 드래그 → 기존 스킬 위 → 교체 확인 팝업 → 수락 시 교체
    ├─ 스킬 우클릭 → 빈 슬롯 있음 → 자동 배치
    ├─ 스킬 우클릭 → 빈 슬롯 없음 → 안내 문구 표시
    └─ 건너뛰기
    |
[보상 종료] 다음 단계로 진행
```

### 2.4 핵심 규칙

| 규칙 | 내용 | 예외 | 예시 |
|------|------|------|------|
| **영혼 드랍** | 몬스터 HP가 0이 되면 카드 소멸 이펙트 후 해당 위치에 영혼 드랍 | - | 늑대 처치 → 카드 소멸 → 영혼 15개 드랍 표시 |
| **영혼 수집** | 모든 몬스터 처치 후 드랍된 영혼이 상단 영혼 자원 UI로 이동하며 수치 증가 | - | 늑대 2마리 처치 → 영혼 15 + 15가 상단으로 이동 → 총 영혼 수치 30 증가 |
| **영혼 차등** | 몬스터별 영혼 차등 (일반 < 엘리트 < 보스) | - | 일반 늑대 → 영혼 15, 엘리트 고블린 → 영혼 40, 보스 트롤 → 영혼 80 |
| **스킬 선택** | 3개 중 1개 선택 또는 건너뛰기 | - | "강타", "분노 방어", "약화 베기" 3개가 제시되어 1개를 고르거나 패스 |
| **스킬 보상 풀** | 현재 클래스 스킬 + 공용 스킬, 현재 코인 수로 사용 가능한 스킬만 등장 | - | 코인 3개인 전사에게 앞면4 스킬은 나오지 않음. 전사에게 도적 전용 스킬은 나오지 않음 |
| **스킬 중복 불가** | 현재 보유 중이거나 이전에 획득했던 스킬은 보상에 나오지 않음 | - | "기본 공격"을 보유 중이면 제외. 이전에 "강타"를 획득 후 교체했더라도 다시 나오지 않음 |
| **스킬 배치 (드래그)** | 빈 슬롯에 드래그 → 즉시 배치. 기존 스킬 위에 드래그 → 교체 확인 팝업 | - | "강타"를 빈 4번 슬롯에 드래그 → 바로 배치. "기본 방어" 위에 드래그 → "기본 방어를 강타로 교체하시겠습니까?" 팝업 |
| **스킬 배치 (우클릭)** | 빈 슬롯 있음 → 자동 배치. 빈 슬롯 없음 → 교체 안내 문구 표시 | - | 5번 슬롯이 비어있을 때 우클릭 → 5번에 자동 배치. 6슬롯 모두 찼을 때 우클릭 → "슬롯이 가득 찼습니다. 교체할 스킬 위로 드래그하세요" |
| **스킬 교체 대가** | 없음 (무료 교체) | - | 어떤 스킬을 교체하든 영혼이나 코인 소모 없음 |
| **전리품 보상** | 엘리트: 1개 자동 획득 (아이콘/이름 표시, 호버 시 효과 확인). 보스: 3개 중 1개 선택 | - | 엘리트 처치 → "이끼 낀 부적" 자동 획득, 화면에 아이콘과 이름 표시, 호버하면 "턴 시작 시 방어력 +2" 확인. 보스 처치 → 3개 전리품 중 "숲의 가호" 선택 |
| **코인 보상** | 보스 전투에서만 코인 +1 | - | 보스 처치 후 코인 3개 → 4개로 증가, 다음 턴부터 앞면3 스킬을 37.5% 확률로 사용 가능 |
| **보상 화면 구성** | 일반: 영혼+스킬 한 화면. 엘리트: 영혼+전리품+스킬 한 화면. 보스: 1단계(영혼+코인+전리품 선택) → 2단계(스킬 선택) | - | 일반 전투: 영혼 15 + 스킬 3개가 한 화면에 표시. 보스 전투: 먼저 영혼 80 + 코인 +1 + 전리품 선택 → 다음 화면에서 스킬 선택 |

### 2.5 주요 수치 기준

| 항목 | 기준값 | 범위 | 비고 |
|------|--------|------|------|
| 스킬 선택지 | 3개 | - | - |
| 전리품 선택지 (보스) | 3개 중 1개 | - | 엘리트는 1개 자동 획득 |
| 코인 보상 (보스) | +1개 | - | - |
| 영혼 획득량 | 미정 | 일반 < 엘리트 < 보스 | 추후 밸런싱 |

## Part 3: 어떻게 확인하는가 (Verify)

### 3.1 관련 시스템

| 시스템 | 관계 | 영향 |
|--------|------|------|
| **동전 던지기** | 연동 | 코인 보상(보스)이 코인 개수 증가시킴 |
| **스킬 시스템** | 핵심 연동 | 보상 스킬 풀 제공, 슬롯 배치/교체 |
| **전투 시스템** | 의존 | 전투 승리가 보상 트리거, 몬스터 종류가 보상 구성 결정 |
| **상점** | 연동 | 영혼을 소비처로 연결 |

### 3.2 검증 기준

**구현 완료 기준:**
- [ ] 모든 전투 승리 시 영혼 획득량이 표시된다
- [ ] 스킬 3개가 표시되며, 현재 클래스 + 공용 스킬만 등장한다
- [ ] 현재 코인 수로 사용 불가능한 스킬은 보상에 나오지 않는다
- [ ] 보유 중이거나 이전에 획득했던 스킬이 보상에 나오지 않는다
- [ ] 스킬 드래그로 빈 슬롯 배치 및 기존 스킬 교체(확인 팝업)가 동작한다
- [ ] 스킬 우클릭 시 빈 슬롯 자동 배치, 꽉 찬 경우 안내 문구가 표시된다
- [ ] 엘리트 전투 시 전리품 1개 자동 획득 + 아이콘/이름/호버 효과가 표시된다
- [ ] 보스 전투 시 전리품 3개 중 1개 선택이 가능하다
- [ ] 보스 전투 시 코인 +1이 지급된다
- [ ] 보상 순서가 영혼 → 코인 → 전리품 → 스킬 순서로 진행된다

**플레이 테스트 기준:**
- [ ] 스킬 선택 시 플레이어가 3개를 비교하며 고민하는 상황이 발생한다
- [ ] 슬롯 교체 시 "어떤 걸 버릴까" 전략적 딜레마가 체감된다
- [ ] 건너뛰기가 합리적 선택이 되는 상황이 존재한다 (무조건 선택이 이득이 아님)
- [ ] 보스 전리품 선택 시 빌드에 따라 선호가 달라진다
