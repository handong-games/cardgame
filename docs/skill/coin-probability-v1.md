# 코인 확률 분석 v1

코인 토스 시스템에서 스킬 사용 가능 확률 분석

---

## 1. 기본 전제

- 턴 시작 시 코인 N개를 던짐
- 각 코인은 50% 확률로 앞면/뒷면
- 스킬은 **앞면 M개 이상** 또는 **뒷면 M개 이상**을 코스트로 요구

---

## 6. 설계 시사점

### 6.1 코스트별 안정성

- **코스트 1**: 거의 항상 사용 가능 (87.5%+)
- **코스트 2**: 절반 정도 사용 가능 (50%+)
- **코스트 3**: 운에 크게 의존 (12.5%~50%)

### 6.2 앞면/뒷면 포트폴리오

플레이어가 앞면 스킬만 가지고 있으면:
- 뒷면이 많이 나왔을 때 행동 불가

**권장**: 앞면 스킬 + 뒷면 스킬을 혼합하여 어떤 결과에서도 행동 가능하게 설계

### 6.3 코인 추가 스킬의 가치

코인 1개 추가 시 코스트 2 스킬 사용 확률:
- 3코인 → 4코인: 50% → 68.75% (+18.75%)
- 4코인 → 5코인: 68.75% → 81.25% (+12.5%)
