## 밸런스 이슈 패턴

### 공통 문제점

1. **버프 스택 가치 과대평가**
   - 버프 스택 가중치 3.0이 실제보다 높게 계산됨
   - 특히 제약 조건(턴당 1회 + 쿨다운)과 결합 시 과도한 밸류

2. **힐 스킬 약함**
   - 힐 가중치 1.4가 안정성을 충분히 반영하지 못함
   - 치유, 봉헌 등이 기준 미달

3. **조건부 효과 강함**
   - 처형(적 HP 조건)의 조건부 효과가 너무 강력
   - 발동 확률 대비 보상이 과도함

### 권장 조치

1. **버프 가중치 조정**: 3.0 → 2.5 검토
2. **힐 가중치 조정**: 1.4 → 1.2 검토
3. **조건부 효과 증폭 조정**: HP 조건 1.5 → 1.3 검토

## 검증 방법

```bash
# 전체 밸런스 검증 실행
cd dev/proto
npm run balance-check
```

스크립트는 모든 스킬의 밸류를 자동 계산하고 기준 범위와 비교합니다.

- **✅ OK**: 기준 범위 내
- **⚠️ WARNING**: 기준 범위 약간 초과 (±20%)
- **❌ FAIL**: 기준 범위 크게 초과
