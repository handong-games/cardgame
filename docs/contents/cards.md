# 카드 설계 문서

## 1. 기본 설계 원칙

### 전투 리소스
- **에너지**: 턴당 3 에너지
- **핸드 크기**: 턴당 5장 드로우 (기본값)
- **덱 순환**: 덱이 소진되면 버린 카드 더미를 섞어 새 덱 생성

### 카드 타입
| 타입 | 설명 | 예시 |
|------|------|------|
| **공격(Attack)** | 적에게 데미지 | 타격, 강타 |
| **스킬(Skill)** | 방어, 버프, 디버프 등 | 방어, 오라 |
| **파워(Power)** | 지속 효과 (전투 종료까지 유지) | 팔라딘 오라 |

## 2. 전사 기본 카드 (시작 덱)

### 2.1 타격 (Strike)
```yaml
이름: 타격
타입: 공격
비용: 1 에너지
효과: 6 데미지
희귀도: 기본
획득: 시작 덱
수량: 4장
```

**설계 의도**
- 가장 기본적인 공격 수단
- 에너지 대비 데미지 효율 6:1 (기준선)
- 턴당 3에너지로 3장 사용 가능

**밸런스 기준**
- 1 에너지 = 6 데미지 (기본 교환 비율)

---

### 2.2 방어 (Defend)
```yaml
이름: 방어
타입: 스킬
비용: 1 에너지
효과: 5 방어
희귀도: 기본
획득: 시작 덱
수량: 2장
```

**설계 의도**
- 기본 방어 수단
- 에너지 대비 방어 효율 5:1
- 데미지보다 약간 낮은 효율 (공격이 우선)

**밸런스 기준**
- 1 에너지 = 5 방어 (기본 교환 비율)
- 방어는 턴 종료 시 소멸

---

### 2.3 강타 (Power Strike)
```yaml
이름: 강타
타입: 공격
비용: 2 에너지
효과: 10 데미지
희귀도: 기본
획득: 시작 덱
수량: 1장
```

**설계 의도**
- 에너지를 몰아서 쓰는 강력한 한 방
- 2 에너지 = 10 데미지 (효율 5:1, 타격보다 낮음)
- 초반 엘리트/보스전 핵심 카드
- 턴당 3에너지 제약 때문에 1장만 사용 가능

**밸런스 기준**
- 높은 코스트는 효율이 약간 낮음 (유연성 감소 패널티)
- 프로토타입에서는 단순하게 효율만 조정

---

### 시작 덱 종합 분석
```
총 7장: 타격 x4 + 방어 x2 + 강타 x1

턴당 기대값 (핸드 5장 가정):
- 공격 카드 비율: 5/7 (71%) → 핸드에 평균 3-4장
- 방어 카드 비율: 2/7 (29%) → 핸드에 평균 1-2장

최대 화력 (3에너지):
- 타격 x3 = 18 데미지
- 타격 x1 + 강타 x1 = 16 데미지

최대 방어 (3에너지):
- 방어 x3 = 15 방어
```

**설계 의도**
- 공격 위주 플레이 유도
- 적절한 방어로 생존 가능
- 덱빌딩으로 전략 다양화

## 3. 팔라딘 전직 카드

### 3.0 오라 시스템 개요

**오라 규칙 (2026-01-11 변경)**
- **지속 시간**: 3턴 (이전: 전투 종료까지)
- **중첩 가능**: 여러 오라 동시 활성화 가능
- **갱신**: 같은 오라 재사용 시 지속 시간 갱신

### 3.1 헌신의 오라 (Aura of Devotion)

```yaml
이름: 헌신의 오라 (Aura of Devotion)
타입: 파워
비용: 1 에너지
효과:
  - (파워) 3턴간 지속
  - 매 턴 시작 시 2 방어 획득
  - 공격 카드 사용 시 +1 데미지
희귀도: 특급 (팔라딘 전용)
획득: 전직 시 자동 획득
수량: 1장
```

**설계 의도**
- **팔라딘 정체성**: "방어하면서 싸우는 성기사"
- **지속 효과**: 3턴간 유지되며 갱신 가능
- **시너지 구조**:
  1. 매 턴 자동 방어 (생존력 ↑)
  2. 공격 카드 강화 (화력 ↑)
  3. 오라 유지하며 싸우는 전략

**전략적 고려사항**
- 초반 1에너지 투자로 3턴간 이득
- 오라 먼저 켜고 → 공격 카드 연타
- 3턴마다 갱신 필요 (전략적 타이밍)
- 다른 오라와 중첩 가능

**밸런스**
```
오라 유지 시 (3턴 기준):
- 방어 수익: 2 x 3턴 = 6 방어
- 데미지 증가: 공격 카드당 +1 (턴당 2-3회 = 6-9 추가 데미지)
- 총 가치: 약 12-15 가치 (1 에너지 대비 효율적)
```

---

### 3.2 정의의 오라 (Justice Aura) - 신규

```yaml
이름: 정의의 오라 (Justice Aura)
타입: 파워
비용: 1 에너지
효과:
  - (파워) 3턴간 지속
  - 공격 카드 사용 시 +2 데미지
희귀도: 일반 (팔라딘 전용)
획득: 전직 후 보상 풀
수량: 무제한
```

**설계 의도**
- **공격 특화 오라**: 헌신의 오라보다 데미지에 집중
- **단순한 효과**: 공격 카드 강화에만 집중
- **중첩 시너지**: 헌신의 오라와 중첩 시 공격당 +3 데미지

**밸런스**
```
오라 유지 시 (3턴 기준):
- 데미지 증가: 공격 카드당 +2 (턴당 2-3회 = 12-18 추가 데미지)
- 헌신 + 정의 중첩: 공격당 +3 데미지
```

---

### 3.3 수호의 오라 (Guardian Aura) - 신규

```yaml
이름: 수호의 오라 (Guardian Aura)
타입: 파워
비용: 1 에너지
효과:
  - (파워) 3턴간 지속
  - 매 턴 시작 시 4 방어 획득
희귀도: 일반 (팔라딘 전용)
획득: 전직 후 보상 풀
수량: 무제한
```

**설계 의도**
- **방어 특화 오라**: 헌신의 오라보다 방어에 집중
- **탱커 플레이**: 방어에 집중하고 싶을 때 선택
- **중첩 시너지**: 헌신의 오라와 중첩 시 턴 시작 +6 방어

**밸런스**
```
오라 유지 시 (3턴 기준):
- 방어 수익: 4 x 3턴 = 12 방어
- 헌신 + 수호 중첩: 턴 시작 시 +6 방어
```

---

### 3.4 신성한 일격 (Holy Strike)

```yaml
이름: 신성한 일격 (Holy Strike)
타입: 공격
비용: 1 에너지
효과:
  - 8 데미지
  - 헌신의 오라 활성화 시: 추가로 3 방어 획득
희귀도: 일반 (팔라딘 전용)
획득: 전직 후 보상 풀에 추가
수량: 제한 없음
```

**설계 의도**
- **오라 시너지**: 오라 활성화 시 더 강력함
- **기본 성능**: 오라 없이도 타격(6 데미지)보다 우수
- **공-방 동시**: 공격하면서 방어 획득 (팔라딘 테마)

**전략적 고려사항**
- 오라 없이: 1에너지 = 8 데미지 (타격보다 33% 강함)
- 오라 활성화: 1에너지 = 8 데미지 + 3 방어 + 오라 보너스 (+1 데미지)
- 실질 가치: 9 데미지 + 3 방어 = 약 12 가치

**오라 시너지 계산**
```
헌신의 오라 활성화 상태에서:
1. 기본 데미지: 8
2. 오라 보너스: +1 (공격 카드 사용 시)
3. 추가 방어: 3
4. 총합: 9 데미지 + 3 방어

비교:
- 타격(6) + 오라(+1) = 7 데미지
- 신성한 일격(8) + 오라(+1) + 방어(3) = 9 데미지 + 3 방어
```

### 3.5 정화의 방패 (Shield of Purification)

```yaml
이름: 정화의 방패 (Shield of Purification)
타입: 스킬
비용: 2 에너지
효과:
  - 12 방어
  - 헌신의 오라 활성화 시: 다음 턴 드로우 +1
희귀도: 일반 (팔라딘 전용)
획득: 전직 후 보상 풀에 추가
수량: 제한 없음
```

**설계 의도**
- **강력한 방어**: 2 에너지로 높은 방어 제공
- **오라 시너지**: 드로우 보너스로 핸드 질 향상
- **전략적 선택**: 에너지 2개를 방어에 투자할 가치 제공

**전략적 고려사항**
- 오라 없이: 2에너지 = 12 방어 (방어 카드 2.4장 분량, 약간 효율적)
- 오라 활성화: 12 방어 + 다음 턴 카드 1장
- 드로우 가치: 더 많은 카드 = 더 좋은 선택지

**밸런스**
```
기본 효율:
- 방어 카드 2장 (2에너지) = 10 방어
- 정화의 방패 (2에너지) = 12 방어 (20% 효율 ↑)

오라 시너지:
- 12 방어 + 카드 1장 (5-6 가치)
- 총 가치: 약 17-18 (매우 효율적)
```

## 4. 팔라딘 시너지 구조

### 4.1 핵심 시너지 흐름

```
[1턴] 헌신의 오라 사용 (1에너지)
  ↓
[매 턴] 턴 시작 시 자동 방어 +2
  ↓
[공격] 신성한 일격 사용 (1에너지)
  → 8 데미지 + 오라 보너스(+1) = 9 데미지
  → 추가 방어 3 획득
  ↓
[방어] 정화의 방패 사용 (2에너지)
  → 12 방어
  → 다음 턴 드로우 +1
```

### 4.2 시너지 효과 정리

| 카드 조합 | 효과 | 전략적 가치 |
|---------|------|-----------|
| **오라 + 신성한 일격** | 9 데미지 + 3 방어 | 공-방 밸런스 |
| **오라 + 정화의 방패** | 12 방어 + 드로우 | 장기전 대비 |
| **오라 + 기본 공격** | 모든 공격 +1 데미지 | 화력 증폭 |

### 4.3 전직 전후 비교

**전직 전 (전사)**
```
턴당 기대값:
- 공격: 타격 x2 (12 데미지)
- 방어: 방어 x1 (5 방어)

전략: 단순 공방
```

**전직 후 (팔라딘)**
```
턴당 기대값 (오라 활성화):
- 공격: 신성한 일격 x2 (18 데미지 + 6 방어)
- 자동 방어: 2 (매 턴 시작)
- 총 방어: 8

전략: 지속 버프 + 공방 동시 + 카드 드로우
```

## 6. 설계 고려사항

### 6.1 밸런스 기준

| 리소스 | 교환 비율 | 비고 |
|--------|----------|------|
| 1 에너지 | 6 데미지 | 공격 기본 |
| 1 에너지 | 5 방어 | 방어 기본 |
| 파워 카드 | 장기적 이득 | 초반 투자 필요 |

### 6.2 프로토타입 검증 목표

**검증 질문**
1. 오라 사용이 전략적으로 의미있는가?
2. 전직 후 플레이가 확실히 달라지는가?
3. 팔라딘 카드가 기본 카드보다 매력적인가?
4. 덱빌딩 선택이 고민을 만드는가?

**성공 기준**
- 플레이어가 오라 활성화 여부에 따라 플레이 전략을 바꿈
- 전직 후 팔라딘 카드를 적극적으로 선택함
- "오라 중심" vs "하이브리드" 빌드 차이를 체감함

### 6.3 향후 확장 방향

**추가 가능 카드**
- 오라 충전 카드 (오라 효과 강화)
- 오라 소비 카드 (일회성 강력 효과)
- 다른 오라 종류 (공격형, 유틸형)

**다른 전직과의 차별화**
- 팔라딘: 지속 버프 + 공방 밸런스
- (미래) 버서커: 순수 화력 + 리스크
- (미래) 가디언: 방어 특화 + 반격

## 7. 카드 수량 관리

### 7.1 시작 덱 (전사)
- 총 7장
- 획득 방법: 게임 시작 시 자동 지급

### 7.2 팔라딘 전직 시
- 헌신의 오라 1장 자동 획득
- 전직 카드 중 1장 선택 획득 (신성한 일격 or 정화의 방패)

### 7.3 보상 카드 풀
**전직 전**: 전사 기본 카드만 등장
**전직 후**: 전사 기본 카드 + 팔라딘 전용 카드 등장
