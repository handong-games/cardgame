# 클래스 전직 시스템

## 개요

전투 중 특정 조건을 만족하면 상위 클래스로 전직할 수 있습니다.

---

## 전직 조건

### 전직 가능 시점
- 라운드 클리어 후 보상 선택 직후
- 덱에 해당 전직 카드가 일정 수 이상 있을 때

### 클래스별 전직 조건
| 기본 클래스 | 상위 클래스 | 조건 |
|------------|------------|------|
| 전사 | 팔라딘 | 팔라딘 카드 3장 이상 보유 |
| 전사 | 버서커 | (미구현) 버서커 카드 3장 이상 보유 |
| 전사 | 검사 | (미구현) 검사 카드 3장 이상 보유 |

---

## 전직 보상

### 전직 시 획득
1. **클래스 변경**: 해당 상위 클래스로 전환
2. **전용 카드 획득**: 전직 전용 카드 중 1장 선택

### 클래스별 전직 보상 카드
| 클래스 | 전직 보상 카드 |
|--------|----------------|
| 팔라딘 | 헌신의 오라, 신성한 일격, 정화의 방패 중 1장 |
| 버서커 | (미구현) |
| 검사 | (미구현) |

---

## 전직 후 보상 카드 풀

### 시스템 개요
전직 후에는 해당 클래스와 연관된 카드만 보상으로 등장합니다.

### 클래스별 보상 카드 풀

#### 전사 (기본 클래스)
공통 카드 풀에서 보상 카드 선택

| 카드명 | 전직 기여 |
|--------|----------|
| 전장의 의지 | 팔라딘, 버서커, 검사 |
| 강철 의지 | 팔라딘, 버서커 |
| 일섬 | 버서커, 검사 |
| 응수 | 팔라딘, 검사 |
| 신성한 집중 | 팔라딘 |
| 피의 대가 | 버서커 |
| 연환참 | 검사 |

#### 팔라딘 (전직 후)
팔라딘 전용 카드 풀에서 보상 카드 선택

| 카드명 | 타입 | 효과 |
|--------|------|------|
| 정의의 오라 | 파워 | 3턴간 공격 +2 데미지 |
| 수호의 오라 | 파워 | 3턴간 턴 시작 +4 방어 |
| 신성한 일격 | 공격 | 8 데미지, 오라 시 +3 방어 |
| 정화의 방패 | 스킬 | 12 방어, 오라 시 드로우 +1 |

> 헌신의 오라는 전직 보상으로만 획득 가능

---

## 전직 진행도 힌트

### UI 표시
보상 카드 선택 시 각 카드 아래에 전직 진행도 힌트가 표시됩니다.

```
[카드]
팔라딘: 2/3 장
```

### 힌트 표시 조건
- 해당 카드가 `advancesTo`에 전직 클래스를 포함할 때
- 현재 덱의 해당 전직 카드 수 / 필요 카드 수 표시
