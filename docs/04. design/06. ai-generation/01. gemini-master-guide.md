# Gemini 마스터 가이드

## 개요

이 문서는 Gemini 3.0 API를 사용하여 게임 아트 에셋을 생성하기 위한 종합 가이드입니다. 따뜻한 스토리북 스타일의 일관된 이미지를 생성하기 위한 방법론을 제공합니다.

---

## 프롬프트 레이어 시스템

### 4레이어 구조
```
프롬프트 아키텍처:
┌─────────────────────────────────────┐
│ Layer 1: 스타일 베이스 (필수)       │
│ - 손그림 스토리북 스타일            │
│ - 연필 스케치 + 수채화              │
└─────────────────────────────────────┘
          ↓
┌─────────────────────────────────────┐
│ Layer 2: 에셋 유형 (필수)           │
│ - 캐릭터/몬스터/배경/UI              │
│ - SD 비율, 볼 홍조 등               │
└─────────────────────────────────────┘
          ↓
┌─────────────────────────────────────┐
│ Layer 3: 세부 특성 (선택)           │
│ - 클래스/티어/지역 등               │
│ - 표정/포즈 수정자                  │
└─────────────────────────────────────┘
          ↓
┌─────────────────────────────────────┐
│ Layer 4: 네거티브 (필수)            │
│ - 어두운/무서운 스타일 제외         │
│ - 3D/사실적 스타일 제외             │
└─────────────────────────────────────┘
```

## Layer 1: 마스터 스타일 베이스

### 모든 에셋 공통 (복사하여 사용)
```
[MASTER STYLE - 모든 프롬프트 시작]
hand-drawn storybook illustration style,
pencil sketch with watercolor coloring,
warm cozy fairy tale atmosphere,
soft warm color palette,
simple wooden frame brown tones,
parchment paper texture background,
children's picture book illustration quality,
warm lighting from top-left
```

## 에셋 유형별 가이드

### 캐릭터 에셋

#### 기본 설정
| 항목 | 값 |
|------|-----|
| 비율 | 2:3 (세로) |
| 해상도 | 512 x 768 |
| 배경 | 양피지 + 나무 프레임 |
| 캐릭터 비율 | SD 2-2.5 등신 |

#### 캐릭터 프롬프트 템플릿
```
{MASTER_STYLE},
cute chibi SD character portrait,
big head small body 2-2.5 head ratio,
large expressive round eyes,
rosy blushing cheeks on both sides,
warm friendly smile expression,
{CLASS_DESCRIPTION},
simple wooden frame,
warm parchment background,
vertical portrait 2:3 ratio

{NEGATIVE_PROMPT}
```

### 몬스터 에셋

#### 몬스터도 귀엽게!
```
[MONSTER STYLE]
{MASTER_STYLE},
cute monster creature portrait,
adorable but mischievous appearance,
big expressive eyes,
round soft shapes,
warm friendly (or playfully menacing) expression,
{MONSTER_DESCRIPTION},
simple wooden frame,
vertical portrait 2:3 ratio

{NEGATIVE_PROMPT}
```

### 배경 에셋

#### 기본 설정
| 항목 | 값 |
|------|-----|
| 비율 | 16:9 (가로) |
| 해상도 | 1920 x 1080 |
| 분위기 | 따뜻하고 포근하게 |

#### 배경 프롬프트 템플릿
```
{MASTER_STYLE},
cozy fairy tale environment scene,
warm inviting atmosphere,
{REGION_DESCRIPTION},
soft golden sunlight,
layered depth composition,
central area empty for characters,
wide landscape 16:9 ratio,
no characters

{NEGATIVE_PROMPT}
```

### UI/아이콘 에셋

#### 기본 설정
| 항목 | 값 |
|------|-----|
| 비율 | 1:1 (정사각) |
| 해상도 | 128 x 128 또는 256 x 256 |
| 배경 | 투명 |

#### 아이콘 프롬프트 템플릿
```
{MASTER_STYLE},
simple cute game icon,
{ICON_SUBJECT},
warm friendly design,
bold readable silhouette,
soft colors,
isolated transparent background,
1:1 square format

{NEGATIVE_PROMPT}
```

## 네거티브 프롬프트 가이드

### 공통 네거티브 (항상 사용)
```
[NEGATIVE - 모든 에셋]
dark gothic horror scary creepy,
realistic photorealistic 3D render CGI,
cold colors blue purple dominant,
harsh shadows high contrast,
complex ornate decorations,
anime manga style sharp lines,
modern sci-fi futuristic,
sad depressing gloomy atmosphere,
realistic human proportions,
no blush on cheeks
```

### 에셋별 추가 네거티브

#### 캐릭터
```
+ADD: realistic proportions,
      scary fierce expression,
      complex detailed armor
```

#### 몬스터
```
+ADD: truly terrifying scary,
      grotesque disturbing,
      nightmare horror
```

#### 배경
```
+ADD: characters people figures,
      dark ominous atmosphere,
      gloomy depressing scene
```

## 일관성 유지 전략

### 핵심 요소 고정
```
항상 유지할 것:
├── 손그림 스케치 + 수채화 채색
├── 따뜻한 색상 팔레트
├── SD 비율 (캐릭터)
├── 볼 홍조 (모든 캐릭터)
├── 심플한 나무/금색 프레임
└── 양피지 배경 텍스처
```

### 색상 일관성
```
색상 규칙:
├── 배경: Warm Parchment #FFF5E6
├── 프레임: Warm Brown #8B4513
├── 테두리: Soft Gold #DAA520
├── 피부: Warm Peach #FFDAB9
├── 볼 홍조: Rosy Pink #FFB6C1
└── 검정 대신: Warm Brown #4A3728
```

## 문제 해결 가이드

### 흔한 문제와 해결책

#### 너무 사실적/3D
```
문제: 사실적이거나 3D 렌더 같음
해결:
+ "hand-drawn" "pencil sketch" "watercolor" 강조
+ "storybook illustration" 추가
+ 네거티브에 "realistic 3D render" 강화
```

#### 비율이 사실적
```
문제: SD 비율이 아닌 일반 비율
해결:
+ "chibi SD proportions" 강조
+ "big head small body 2-2.5 ratio" 명시
+ "large expressive eyes" 추가
```

#### 볼 홍조 없음
```
문제: 캐릭터에 볼 홍조가 없음
해결:
+ "rosy blushing cheeks" 명시적 추가
+ "pink blush on both cheeks" 추가
+ 네거티브에 "no blush" 추가
```

#### 어둡거나 무서운 분위기
```
문제: 어둡거나 무서운 결과물
해결:
+ "warm cozy friendly" 강조
+ "cute adorable charming" 추가
+ 네거티브에 "dark scary horror" 강화
```

#### 복잡한 프레임
```
문제: 프레임이 너무 화려함
해결:
+ "simple wooden frame" 강조
+ "minimal decoration" 추가
+ 네거티브에 "complex ornate" 강화
```

## 생성 워크플로우

### 권장 순서
```
1. 프롬프트 조립
   ├── Layer 1 (마스터 스타일) 복사
   ├── Layer 2 (에셋 유형) 선택
   ├── Layer 3 (세부 사항) 추가
   └── Layer 4 (네거티브) 포함

2. 체크리스트 확인
   ├── SD 비율 명시?
   ├── 볼 홍조 언급?
   └── 따뜻한 분위기 키워드?

3. 테스트 생성
   ├── 결과 확인
   └── 필요시 조정

4. 검증
   ├── 체크리스트 검토
   └── 스타일 일관성 확인
```

## API 활용 팁

### 프롬프트 최적화
```
효과적인 프롬프트:
├── 길이: 80-150 단어 권장
├── 구조: 콤마로 구분된 키워드
├── 순서: 스타일 → 주제 → 세부사항
├── 반복: 핵심 키워드 강조
└── 구체성: "cute" 보다 "cute chibi SD"
```

### 핵심 키워드 우선순위
```
1. hand-drawn storybook
2. pencil sketch watercolor
3. cute chibi SD big head
4. rosy blushing cheeks
5. warm cozy friendly
6. simple wooden frame
7. parchment background
```
