# 숲 지역 몬스터 스펙시트

## 개요

이 문서는 숲(Forest) 지역에 등장하는 모든 몬스터의 상세 스펙을 정의합니다. 모든 몬스터는 귀엽고 친근한 SD 스타일로 디자인됩니다.

---

## 지역 개요

### 숲 지역 테마
```
숲 지역 분위기:
├── 환경: 밝고 따뜻한 숲속
├── 색조: 녹색, 갈색, 황금빛 햇살
├── 느낌: 평화롭지만 약간의 모험심
├── 시간대: 주로 낮 (따뜻한 햇살)
└── 특징: 동화 속 숲, 귀여운 생물들
```

|------|--------|-----|--------|

## 공통 스타일 가이드

### 모든 몬스터 필수 요소
```
필수 디자인 요소:
├── SD 비율 (2~2.5 등신)
├── 큰 동그란 눈
├── 양쪽 볼 홍조 (#FFB6C1)
├── 부드러운 곡선 (각진 부분 없음)
├── 따뜻한 색상 팔레트
├── 손그림 스토리북 느낌
└── 위협적이지 않은 귀여운 외형
```

### 티어별 크기 비교
| 티어 | 크기 (캐릭터 대비) | 디테일 수준 |
|------|------------------|------------|
| Tier 1 | 45-70% | 단순, 깔끔 |
| Tier 2 | 80-120% | 중간, 악세서리 |
| Tier 3 | 130-180% | 복잡, 화려한 장식 |

### 상태 변화 공통
| 상태 | 시각 변화 |
|------|----------|
| 버프 | 녹색/금색 글로우 |
| 디버프 | 보라색 연기 (귀엽게) |
| 분노 | 빨간 김 + 뿔 이펙트 |
| 약화 | 채도 감소, 땀방울 |
