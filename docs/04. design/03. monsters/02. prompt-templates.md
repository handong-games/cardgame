# 몬스터 AI 프롬프트 템플릿

## 개요

이 문서는 Gemini API를 사용하여 몬스터 이미지를 생성하기 위한 검증된 프롬프트 템플릿을 제공합니다.

---

## 프롬프트 구조

### 4레이어 시스템
```
[Layer 1] 스타일 베이스 (모든 몬스터 공통)
    ↓
[Layer 2] 티어 특성 (Tier 1/2/3)
    ↓
[Layer 3] 지역 특성 (숲/던전/성)
    ↓
[Layer 4] 개체 특성 (구체적 몬스터)
    ↓
[Negative] 제외 요소
```

---

## Layer 1: 스타일 베이스

### 방향 규칙 (필수)
```
[DIRECTION - 모든 몬스터에 적용]
body facing left diagonal direction,
three-quarter view angled to left,
creature looking toward front-left
```

> **중요**: 모든 몬스터 카드는 몸이 좌측 대각선 방향을 향하도록 생성되어야 합니다.

### 모든 몬스터 공통
```
storybook illustration style monster creature,
dark fairytale medieval fantasy atmosphere,
hand-painted watercolor and gouache texture,
menacing yet characterful expression,
clear readable silhouette shape,
exaggerated features for recognition,
body facing left diagonal direction,
three-quarter view angled to left,
warm ambient light with deep cast shadows,
aged parchment texture background vignette,
single creature centered composition,
card game enemy portrait artwork,
vertical portrait format 2:3 aspect ratio
```

## Layer 2: 티어 특성

### Tier 1: 일반 몬스터
```
[TIER_1]
small to medium sized creature,
natural muted earthy color palette,
simple straightforward design,
minimal ornate details,
feral mischievous wild expression,
common expendable enemy appearance,
low immediate threat level,
basic creature anatomy
```

### Tier 2: 정예 몬스터
```
[TIER_2]
medium to large sized creature,
darker saturated unnatural hues,
moderate complexity design,
notable ornate armor or markings,
intelligent calculating expression,
dangerous elite enemy appearance,
significant threat level,
enhanced mutated anatomy
```

### Tier 3: 보스 몬스터
```
[TIER_3]
large imposing massive sized creature,
dramatic high contrast color scheme,
complex intricate detailed design,
elaborate ornate decorations,
overwhelming dominating expression,
legendary boss enemy appearance,
extreme catastrophic threat level,
unique fantastical anatomy,
magical aura particle effects,
dramatic backlighting rim light
```

## 변형 생성 팁

### 같은 종류 다른 개체
```
기본 프롬프트에 추가:
- "variant design" (변형 디자인)
- "different pose angle" (다른 각도)
- "alternate weapon" (다른 무기)
- "damaged battle-worn" (전투 손상)
```

### 강화/약화 버전
```
강화 (엘리트 버전):
+ "enhanced" "empowered" "alpha variant"
+ "glowing magical runes"
+ "larger more imposing"

약화 (일반 버전):
+ "lesser" "common variant"
+ "simpler design"
+ "smaller stature"
```
