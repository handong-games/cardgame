# 몬스터 디자인 가이드

## 개요

이 문서는 게임 내 적 몬스터의 시각적 디자인 원칙을 정의합니다. 몬스터도 캐릭터와 마찬가지로 귀여운 SD 비율의 카드 형태로 표현됩니다.

---

## 몬스터 디자인 철학

### 핵심 원칙
```
1. 귀여움: 무섭지 않고 귀여운 SD 비율의 몬스터
2. 세계관 통합: 따뜻한 동화 분위기에 맞는 디자인
3. 가독성: 전투 중 빠른 인식 가능
4. 개성: 각 몬스터만의 귀여운 특징
```

### 귀여운 몬스터 접근
```
특징:
├── 동그랗고 부드러운: 날카로운 부분 최소화
├── SD 비율: 큰 머리, 작은 몸
├── 표정: 장난스럽거나 귀여운 표정 (볼 홍조!)
└── 세계관 소속감: 따뜻한 동화 세계의 친구들
```

## 몬스터 티어 시스템

### 티어별 귀여움 표현

#### Tier 1: 일반 몬스터 (Normal)
| 요소 | 특징 |
|------|------|
| 크기 | 캐릭터보다 작음 |
| 색상 | 따뜻한 자연색 (갈색, 녹색) |
| 디테일 | 단순하고 귀여움 |
| 실루엣 | 동그랗고 친숙한 형태 |
| 표정 | 장난스럽고 귀여움 |

```
예시: 귀여운 고블린, 동글 슬라임, 아기 늑대
색상: Warm Green #90EE90, Warm Brown #DEB887
```

#### Tier 2: 정예 몬스터 (Elite)
| 요소 | 특징 |
|------|------|
| 크기 | 캐릭터와 비슷한 크기 |
| 색상 | 조금 더 진한 따뜻한 색조 |
| 디테일 | 중간, 귀여운 장식 요소 |
| 실루엣 | 특징적이지만 부드러운 형태 |
| 표정 | 자신감 있지만 귀여움 |

```
예시: 뚱뚱한 오크, 통통 골렘, 귀여운 유령
색상: Warm Purple #9370DB, Warm Teal #20B2AA
```

#### Tier 3: 보스 몬스터 (Boss)
| 요소 | 특징 |
|------|------|
| 크기 | 캐릭터보다 큼 (하지만 귀엽게) |
| 색상 | 화려하고 따뜻한 색상 |
| 디테일 | 귀여운 장식, 왕관 등 |
| 실루엣 | 인상적이지만 동그란 형태 |
| 표정 | 뽐내는 듯한 귀여운 표정 |

```
예시: 아기 드래곤, 귀여운 마왕, 통통한 기사
색상: Warm Red #CD5C5C, Warm Gold #FFD700, Soft Purple #DDA0DD
```

## 지역별 몬스터 테마

### 숲 지역 (Forest)

#### 디자인 테마
```
숲 몬스터 특징:
├── 귀여운 숲속 동물/생물 기반
├── 밝은 녹색/갈색 팔레트
├── 잎, 꽃, 버섯 장식
├── 장난스럽고 귀여운 느낌
└── 숨바꼭질하듯 등장
```

#### 색상 팔레트
| 요소 | 색상 | HEX |
|------|------|-----|
| 주색 | Warm Green | #90EE90 |
| 보조색 | Warm Brown | #DEB887 |
| 악센트 | Honey | #F0E68C (눈) |
| 볼 홍조 | Rosy Pink | #FFB6C1 |

#### 대표 몬스터
```
Tier 1: 귀여운 고블린, 동글 슬라임, 통통 다람쥐
Tier 2: 나무 요정, 포근한 늑대, 숲의 요정
Tier 3: 귀여운 숲의 왕, 아기 드래곤
```


## 몬스터 카드 구조

### 카드 레이아웃 (빈티지 양피지 스타일)
```
   ╭───────────────────────╮  ← 둥근 사각형
  ╱│                       │╲ ← 닳고 해진 모서리
 │ │   질감 있는 베이지    │ │
 │ │   수채화 종이 결      │ │
 │ │   ┌─────────┐        │ │
 │ │   │ SD 몬스터 │        │ │ ← 몬스터 일러스트
 │ │   │ (큰머리)  │        │ │    (볼 홍조!)
 │ │   │ (볼홍조)  │        │ │
 │ │   └─────────┘        │ │
 │ ├───────────────────────┤ │ ← 얇은 갈색 구분선
 │ │  ❤️❤️❤️  [의도]      │ │ ← HP, 의도 (하단)
  ╲│                       │╱ ← 닳고 해진 모서리
   ╰───────────────────────╯  ← 둥근 사각형
```

### 프레임/배경 스타일 (몬스터 전용)
| 요소 | 스타일 |
|------|--------|
| 테두리 | 낡은 양피지 테두리 + 덩굴/나뭇잎 장식 |
| 테두리 색상 | 어두운 갈색 (#6B5642) + 녹색 포인트 |
| 형태 | 둥근 사각형 |
| 모서리 장식 | 덩굴/나뭇잎 무늬 (캐릭터: 우아한 곡선) |
| 모서리 | 빈티지한 닳고 해진 종이 |
| 배경색 | 약간 어두운 베이지색 |
| 배경 질감 | 수채화 종이 결, 숲속 느낌 |
| 분위기 | 오래된 종이, 자연적/야생적 느낌 |

### 캐릭터 vs 몬스터 테두리 차이
| 요소 | 캐릭터 카드 | 몬스터 카드 |
|------|------------|------------|
| 모서리 장식 | 우아한 곡선 스크롤 | 덩굴/나뭇잎 무늬 |
| 테두리 톤 | 밝은 갈색 | 어두운 갈색+녹색 |
| 전체 느낌 | 영웅/모험가 | 숲속/야생 생물 |

### 의도 표시 시스템
```
공격 의도:
├── 아이콘: 귀여운 검 (따뜻한 빨강)
├── 데미지 숫자 표시
└── 작은 별 반짝임 효과

방어 의도:
├── 아이콘: 귀여운 방패 (하늘색)
├── 블록 숫자 표시
└── 부드러운 빛 효과

버프 의도:
├── 아이콘: 귀여운 화살표 (초록)
├── 버프 아이콘 미리보기
└── 반짝이는 별 효과

특수 의도:
├── 아이콘: 귀여운 물음표 (보라)
├── 알 수 없는 행동
└── 신비로운 반짝임
```

## 표현 가이드라인

### 몬스터 표정/동작
```
일반 몬스터:
├── 장난스러운 표정
├── 놀라는 듯한 포즈
├── 볼에 홍조 필수!
└── 귀여운 행동

정예 몬스터:
├── 자신감 있는 표정
├── 뽐내는 포즈
├── 볼에 홍조 필수!
└── 약간 거만한 귀여움

보스 몬스터:
├── 위풍당당한 표정 (하지만 귀여움)
├── 왕처럼 포즈
├── 볼에 홍조 필수!
└── 뿌듯한 귀여움
```

### 크기 대비
```
화면 내 상대적 크기:
├── Tier 1: 플레이어 카드의 80-100%
├── Tier 2: 플레이어 카드의 100-120%
└── Tier 3: 플레이어 카드의 120-150%
```

## 검증 체크리스트

| 항목 | 확인 사항 | Pass/Fail |
|------|-----------|-----------|
| SD 비율 | 큰 머리, 작은 몸? | □ |
| 볼 홍조 | 양 볼에 홍조 있음? | □ |
| 표정 | 귀엽고 친근한 표정? | □ |
| 지역 | 해당 지역 테마와 일치? | □ |
| 색상 | 따뜻한 색상 팔레트? | □ |
| 스타일 | 스토리북 느낌 유지? | □ |
| 테두리 | 낡은 양피지 테두리? | □ |
| 형태 | 둥근 사각형? | □ |
| 모서리 | 빈티지 닳은 모서리? | □ |
| 배경 | 질감 있는 베이지 종이? | □ |
| 분위기 | 귀엽고 따뜻한 느낌? | □ |
