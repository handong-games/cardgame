# 파티클 시스템 가이드

## 개요

이 문서는 게임 내 파티클 효과의 디자인 원칙과 규격을 정의합니다. 모든 파티클은 따뜻한 스토리북 스타일을 유지하며, 귀엽고 아기자기한 느낌을 줍니다.

---

## 파티클 기본 원칙

### 스타일 일관성
```
따뜻한 파티클 원칙:
├── 둥글고 부드러운 형태 (각진 모양 금지)
├── 따뜻한 색상 팔레트 유지
├── 반짝이는 하이라이트 포함
├── 자연스러운 움직임 (급격한 변화 금지)
└── 동화책 일러스트와 어울리는 느낌
```

### 색상 원칙
| 파티클 유형 | 기본 색상 | 보조 색상 | 금지 색상 |
|-------------|-----------|-----------|-----------|
| 긍정적 효과 | 금색, 분홍색 | 노란색, 연두색 | 검정, 진회색 |
| 도전적 효과 | 은색, 연보라 | 하늘색, 연분홍 | 진빨강, 진보라 |
| 환경 파티클 | 크림색, 베이지 | 연두색, 연노랑 | 차가운 파랑 |

## 파티클 유형별 규격

### 1. 별 파티클 (Star)
```
용도: 성공, 레벨업, 보상, 긍정적 피드백
├── 기본 형태: 4각 또는 6각 별
├── 크기: 8px ~ 24px
├── 지속시간: 400ms ~ 800ms
├── 개수: 5 ~ 15개
├── 색상: Warm Gold (#FFD700), Soft Gold (#DAA520)
├── 움직임: 중심에서 바깥으로 퍼지며 페이드 아웃
└── 회전: 자연스럽게 회전 (180° ~ 360°)
```

**AI 생성 프롬프트**
```
cute hand-drawn star sparkle, warm golden color,
soft glowing edges, storybook illustration style,
watercolor texture, no sharp edges
```

### 2. 먼지/반짝임 파티클 (Dust/Sparkle)
```
용도: 배경 앰비언트, 마법 효과, 힐링
├── 기본 형태: 원형 또는 부드러운 점
├── 크기: 2px ~ 8px
├── 지속시간: 800ms ~ 1500ms
├── 개수: 10 ~ 30개
├── 색상: Cream (#FFFDD0), Light Gold (#FFF8DC)
├── 움직임: 천천히 떠다니며 깜빡임
└── 투명도: 30% ~ 70% 변화
```

**AI 생성 프롬프트**
```
soft magical dust particles, warm cream color,
gentle floating sparkles, fairy tale atmosphere,
glowing dots, watercolor soft edges
```

### 3. 불꽃 파티클 (Flame/Fire)
```
용도: 공격 효과, 화염 마법, 분노 상태
├── 기본 형태: 물방울 뒤집은 형태
├── 크기: 12px ~ 32px
├── 지속시간: 300ms ~ 600ms
├── 개수: 8 ~ 20개
├── 색상: Warm Orange (#FFA500), Soft Coral (#FF6B6B)
├── 움직임: 위로 올라가며 흔들림
└── 페이드: 위로 갈수록 투명해짐
```

**주의**: 불꽃도 따뜻하고 귀여운 느낌 유지 (무섭지 않게)

**AI 생성 프롬프트**
```
cute warm flame particles, orange and coral colors,
soft watercolor fire, friendly fire effect,
storybook illustration, not scary or aggressive
```

### 4. 잎/꽃잎 파티클 (Leaf/Petal)
```
용도: 숲 지역, 자연 마법, 힐링 효과
├── 기본 형태: 둥근 잎 또는 꽃잎
├── 크기: 10px ~ 20px
├── 지속시간: 1000ms ~ 2000ms
├── 개수: 5 ~ 12개
├── 색상: Soft Green (#90EE90), Rose Pink (#FFB6C1)
├── 움직임: 좌우로 흔들리며 천천히 낙하
└── 회전: 자연스러운 회전 (좌우 왔다갔다)
```

**AI 생성 프롬프트**
```
cute falling leaves and flower petals,
soft green and pink colors, gentle floating motion,
hand-drawn storybook style, watercolor texture
```

### 5. 물방울 파티클 (Water Drop)
```
용도: 물 마법, 비, 슬픔 표현 (귀엽게)
├── 기본 형태: 둥근 물방울
├── 크기: 6px ~ 16px
├── 지속시간: 400ms ~ 800ms
├── 개수: 8 ~ 15개
├── 색상: Sky Blue (#87CEEB), Light Blue (#ADD8E6)
├── 움직임: 낙하 후 작은 튀김 효과
└── 반짝임: 하이라이트 포함
```

**AI 생성 프롬프트**
```
cute water droplets, soft sky blue color,
gentle rain drops, sparkling highlights,
storybook illustration, not sad atmosphere
```

### 6. 연기/구름 파티클 (Smoke/Cloud)
```
용도: 소환 효과, 사라짐, 미스터리
├── 기본 형태: 부드러운 구름 형태
├── 크기: 24px ~ 64px
├── 지속시간: 600ms ~ 1200ms
├── 개수: 3 ~ 8개
├── 색상: Cream (#FFFDD0), Soft Lavender (#E6E6FA)
├── 움직임: 팽창하며 페이드 아웃
└── 투명도: 시작 60% → 끝 0%
```

**AI 생성 프롬프트**
```
soft puffy smoke clouds, warm cream and lavender,
magical poof effect, cute cloud shapes,
fairy tale atmosphere, gentle dissipating
```

## 이미터 설정 가이드

### 방출 패턴
| 패턴 | 설명 | 사용처 |
|------|------|--------|
| 점 방출 (Point) | 한 점에서 모든 방향으로 | 폭발, 임팩트 |
| 원형 방출 (Circle) | 원형 테두리에서 바깥으로 | 버프, 힐링 오라 |
| 선형 방출 (Line) | 선을 따라 방출 | 검기, 이동 궤적 |
| 영역 방출 (Area) | 사각 영역 내 랜덤 | 배경 앰비언트 |

### 중력 및 물리
```
파티클 물리 규칙:
├── 중력 (Gravity): -100 ~ +100 (음수: 위로)
│   ├── 별/반짝임: -20 ~ -50 (위로 뜨는 느낌)
│   ├── 잎/꽃잎: +30 ~ +50 (천천히 낙하)
│   └── 불꽃: -80 ~ -100 (빠르게 상승)
├── 바람 (Wind): -50 ~ +50 (좌우 흔들림)
└── 감속 (Drag): 0.9 ~ 0.98 (자연스러운 감속)
```

### 수명 곡선
```
파티클 수명 단계:
├── 생성 (0-10%): 0% → 100% 크기, 투명도 증가
├── 활성 (10-80%): 최대 크기 유지, 이동
├── 소멸 (80-100%): 크기 감소, 투명도 감소
└── 완료 (100%): 제거
```

## 파티클 조합 예시

### 레벨업 이펙트
```
레벨업 파티클 조합:
├── 1차: 원형 빛 확장 (Glow, 0ms)
├── 2차: 별 파티클 x15 (200ms)
├── 3차: 먼지 반짝임 x20 (400ms)
└── 4차: 잔여 빛 페이드 (800ms)
총 지속시간: 1200ms
```

### 힐링 이펙트
```
힐링 파티클 조합:
├── 1차: 녹색 글로우 (0ms)
├── 2차: 잎/꽃잎 x8 상승 (100ms)
├── 3차: 반짝임 x10 (300ms)
└── 4차: 숫자 팝업 (+HP)
총 지속시간: 800ms
```

### 데미지 이펙트
```
데미지 파티클 조합:
├── 1차: 임팩트 플래시 (0ms, 50ms)
├── 2차: 작은 별 x5 (100ms)
├── 3차: 숫자 팝업 (-HP)
└── 4차: 잔여 반짝임 (300ms)
총 지속시간: 500ms
```

## 성능 고려사항

### 최대 파티클 수
| 상황 | 최대 동시 파티클 |
|------|------------------|
| 일반 전투 | 50개 |
| 강한 스킬 | 100개 |
| 보스 연출 | 150개 |
| 전체 화면 | 200개 |

### 최적화 규칙
```
성능 최적화:
├── 화면 밖 파티클 즉시 제거
├── 작은 파티클(4px 미만)은 단순 원으로
├── 오래된 파티클부터 우선 제거
├── LOD: 거리에 따라 파티클 수 조절
└── 배치 렌더링으로 드로우 콜 최소화
```

## 검증 체크리스트

### 파티클 스타일 검증
| 항목 | 확인 사항 | Pass/Fail |
|------|-----------|-----------|
| 형태 | 둥글고 부드러운가? | □ |
| 색상 | 따뜻한 색상 팔레트인가? | □ |
| 움직임 | 자연스럽고 부드러운가? | □ |
| 분위기 | 귀엽고 아기자기한가? | □ |
| 일관성 | 다른 에셋과 어울리는가? | □ |

### 불합격 사유 예시
- 각진 형태의 파티클
- 차가운 색상 (순수 파랑, 진회색)
- 급격한 움직임
- 무섭거나 위협적인 느낌
- 너무 사실적인 효과
