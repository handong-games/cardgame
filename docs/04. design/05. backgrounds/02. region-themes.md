# 지역별 테마

## 개요

이 문서는 게임 내 각 지역의 시각적 테마를 정의합니다. 각 지역은 고유한 따뜻하고 포근한 분위기, 색상, 환경 요소를 가집니다.

---

## 지역 구조

### 지역 개요
```
게임 지역:
├── 숲 (Forest) - 시작 지역
│   ├── 햇살 숲 (Sunny Forest)
│   ├── 버섯 숲 (Mushroom Forest)
│   └── 꽃 숲 (Flower Forest)
├── 던전 (Dungeon) - 중반 지역
│   ├── 모험 동굴 (Adventure Cave)
│   ├── 보물 창고 (Treasure Room)
│   └── 비밀 통로 (Secret Passage)
└── 성 (Castle) - 후반 지역
    ├── 성 정원 (Castle Garden)
    ├── 성 내부 (Castle Interior)
    └── 왕좌의 방 (Throne Room)
```

---

## 숲 지역 (Forest)

### 전체 테마
```
숲 지역 컨셉:
├── 분위기: 따뜻하고 포근한 동화 숲
├── 시간대: 주로 맑은 낮 (황금빛 햇살)
├── 느낌: 친근한 숲 속 친구들
└── 감정: 설렘 + 모험심
```

### 색상 팔레트
| 요소 | 색상 | HEX | 용도 |
|------|------|-----|------|
| 주색 | Warm Green | #90EE90 | 나무, 잎 |
| 보조색 | Warm Brown | #DEB887 | 나무 줄기, 땅 |
| 악센트 | Golden Light | #FFD700 | 빛줄기 |
| 꽃색 | Soft Pink | #FFB6C1 | 꽃, 장식 |
| 하늘 | Warm Blue | #87CEEB | 하늘 |

### 환경 요소
```
필수 요소:
├── 귀여운 나무: 동그랗고 친근한 형태
├── 꽃과 잎: 파스텔 색상으로 장식
├── 귀여운 버섯: 빨강/분홍색 점박이
├── 부드러운 빛: 황금빛 햇살
└── 작은 꽃들: 여기저기 피어있음

분위기 요소:
├── 나비/새: 친근한 생물
├── 반딧불: 밤 장면 (따뜻한 노란빛)
├── 귀여운 달: 미소 짓는 달
├── 꽃잎: 날리는 꽃잎
└── 무지개: 가끔 배경에
```

### 하위 지역

#### 햇살 숲 (Sunny Forest)
```
특징:
├── 밝고 따뜻한 분위기
├── 황금빛 햇살 가득
├── 푸른 잎, 다채로운 꽃
└── 귀여운 숲 몬스터들

프롬프트 수정자:
"sunny bright forest, golden sunbeams,
warm welcoming cozy atmosphere,
colorful flowers, cute mushrooms,
friendly inviting environment"
```

#### 버섯 숲 (Mushroom Forest)
```
특징:
├── 귀여운 거대 버섯들
├── 신비로운 분위기 (하지만 따뜻하게)
├── 반딧불 빛나는 밤
└── 귀여운 버섯 몬스터들

프롬프트 수정자:
"cute mushroom forest, giant colorful mushrooms,
warm magical glow, fireflies twinkling,
cozy fairy tale atmosphere"
```

#### 꽃 숲 (Flower Forest)
```
특징:
├── 형형색색의 꽃들
├── 꽃잎 날리는 분위기
├── 나비와 벌들
└── 꽃 요정 몬스터들

프롬프트 수정자:
"beautiful flower forest, colorful blooms,
petals floating in air, butterflies dancing,
warm spring feeling, friendly atmosphere"
```

## 던전 지역 (Dungeon)

### 전체 테마
```
던전 지역 컨셉:
├── 분위기: 따뜻한 횃불빛의 모험 공간
├── 시간대: 지하 (따뜻한 횃불 조명)
├── 느낌: 신나는 보물찾기
└── 감정: 호기심 + 탐험
```

### 색상 팔레트
| 요소 | 색상 | HEX | 용도 |
|------|------|-----|------|
| 주색 | Warm Gray | #A9A9A9 | 돌벽 |
| 보조색 | Cream | #FFFDD0 | 하이라이트 |
| 악센트 | Warm Orange | #FFA500 | 횃불 빛 |
| 보물 | Soft Gold | #FFD700 | 보물상자, 코인 |
| 마법 | Soft Purple | #DDA0DD | 마법 빛 |

### 환경 요소
```
필수 요소:
├── 따뜻한 돌벽: 갈색 톤의 귀여운 벽돌
├── 횃불: 따뜻한 오렌지빛
├── 보물상자: 반짝이는 금화
├── 열쇠: 여기저기 있는 귀여운 열쇠
└── 마법 빛: 부드러운 보라빛

분위기 요소:
├── 귀여운 박쥐: 무섭지 않게
├── 반짝이는 보석: 벽에 박힌
├── 물방울: 천장에서 똑똑
├── 고양이: 던전의 안내자
└── 지도/나침반: 모험 느낌
```

### 하위 지역

#### 모험 동굴 (Adventure Cave)
```
특징:
├── 반짝이는 광물들
├── 따뜻한 횃불 조명
├── 탐험 장비들
└── 귀여운 동굴 몬스터들

프롬프트 수정자:
"adventure cave, warm torchlight glow,
sparkling crystals and gems,
exploration equipment, treasure maps,
exciting discovery atmosphere"
```

#### 보물 창고 (Treasure Room)
```
특징:
├── 금화가 가득!
├── 보물상자들
├── 반짝반짝 빛나는
└── 보물 지키는 귀여운 몬스터

프롬프트 수정자:
"treasure room, piles of gold coins,
glowing treasure chests,
sparkling jewels, warm golden light,
exciting discovery atmosphere"
```

#### 비밀 통로 (Secret Passage)
```
특징:
├── 신비로운 분위기
├── 숨겨진 문과 레버
├── 마법 룬 (따뜻한 색상)
└── 비밀을 지키는 친구들

프롬프트 수정자:
"secret passage, hidden doors,
warm magical runes glowing,
mysterious but friendly atmosphere,
discovery adventure feeling"
```

## 성 지역 (Castle)

### 전체 테마
```
성 지역 컨셉:
├── 분위기: 따뜻하고 화려한 동화 속 성
├── 시간대: 따뜻한 촛불/스테인드글라스 빛
├── 느낌: 멋진 왕국의 모험
└── 감정: 설렘 + 웅장함
```

### 색상 팔레트
| 요소 | 색상 | HEX | 용도 |
|------|------|-----|------|
| 주색 | Warm Red | #CD5C5C | 카펫, 커튼 |
| 보조색 | Soft Gold | #FFD700 | 장식, 프레임 |
| 악센트 | Soft Purple | #DDA0DD | 마법, 장식 |
| 하이라이트 | Warm Cream | #FFFDD0 | 촛불 빛 |
| 배경 | Warm Beige | #F5DEB3 | 벽, 바닥 |

### 환경 요소
```
필수 요소:
├── 아름다운 아치/기둥: 화려하지만 따뜻하게
├── 샹들리에: 반짝이는 촛불
├── 초상화: 귀여운 왕/여왕
├── 붉은 카펫/커튼: 따뜻한 색상
└── 스테인드글라스: 무지개빛

분위기 요소:
├── 귀여운 갑옷: 복도에 서 있음
├── 꽃병의 꽃: 아름다운 꽃들
├── 깃발/배너: 화려한 문장
├── 왕관/보석: 반짝반짝
└── 귀여운 하인: 분주히 움직임
```

### 하위 지역

#### 성 정원 (Castle Garden)
```
특징:
├── 아름다운 꽃들
├── 분수대
├── 미로 정원
└── 정원 몬스터들

프롬프트 수정자:
"castle garden, beautiful flowers blooming,
fountain sparkling, hedge maze,
warm sunny day, magical atmosphere,
fairytale garden feeling"
```

#### 성 내부 (Castle Interior)
```
특징:
├── 화려한 복도/홀
├── 초상화 갤러리
├── 촛불 조명
└── 성의 하인 몬스터들

프롬프트 수정자:
"castle interior corridor hall,
warm candlelit atmosphere,
beautiful portraits, red carpets,
elegant cozy feeling"
```

#### 왕좌의 방 (Throne Room)
```
특징:
├── 멋진 왕좌
├── 스테인드글라스
├── 화려한 조명
└── 귀여운 왕/여왕 보스

프롬프트 수정자:
"grand throne room, magnificent throne,
stained glass rainbow light,
golden decorations, red carpet,
majestic but warm atmosphere"
```

## 지역 전환 시각 효과

### 전환 연출
```
지역 간 이동 시:
├── 부드러운 페이드 (1초)
├── 새 지역 배경 로드
├── 밝아지며 페이드 인 (1초)
└── 따뜻한 분위기 사운드 전환
```

### 보스 방 연출
```
보스 방 진입 시:
├── 문 열림 애니메이션
├── 반짝이는 빛 효과
├── 배경음악 전환
└── 귀여운 보스 등장 연출
```
