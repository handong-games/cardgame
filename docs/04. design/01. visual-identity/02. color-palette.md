## 감정 색상 시스템

### 따뜻한/긍정적 (Warm/Positive)
```
따뜻한 스펙트럼:
├── 희망: Sunny Yellow #FFD93D - 밝고 희망찬
├── 용기: Warm Orange #FFA500 - 따뜻한 용기
├── 사랑: Rose Pink #FFB6C1 - 포근한 사랑
└── 치유: Soft Green #98FB98 - 부드러운 치유
```

### 도전/주의 (Challenge/Caution)
```
도전 스펙트럼 (너무 어둡지 않게):
├── 주의: Warm Amber #FFBF00 - 주의 필요
├── 긴장: Soft Orange #FF8C00 - 살짝 긴장
├── 위험: Coral Red #FF6B6B - 위험 (부드럽게)
└── 도전: Dusty Purple #9370DB - 미지의 도전
```

## 색상 조합 규칙

### 따뜻한 분위기 유지
```
기본 원칙:
├── 차가운 색상 최소화 (순수 파랑, 보라 지양)
├── 모든 색상에 따뜻한 톤 추가
├── 검정 대신 따뜻한 갈색 사용
└── 흰색 대신 크림/아이보리 사용
```

### 채도 규칙
- **주요 색상**: 채도 60-80% (밝고 따뜻하게)
- **배경 색상**: 채도 20-40% (부드럽게)
- **그림자**: 채도 40-60% (따뜻한 그림자)

### 금지 색상
```
사용 금지:
├── 순수 검정 (#000000) → 따뜻한 갈색 사용
├── 순수 흰색 (#FFFFFF) → 크림/아이보리 사용
├── 네온 색상 → 부드러운 파스텔로 대체
├── 차가운 회색 → 따뜻한 베이지/갈색 사용
└── 어두운 보라/파랑 → 라벤더/하늘색 사용
```
