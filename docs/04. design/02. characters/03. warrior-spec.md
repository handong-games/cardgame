# 전사 (Warrior) 캐릭터 스펙시트

## 개요

전사는 기본 시작 클래스로, 용감하고 당당한 근접 전투형 캐릭터입니다. 강인하지만 귀엽고 친근한 느낌을 주는 것이 핵심입니다.

---

## 캐릭터 정체성

### 기본 정보
| 항목 | 내용 |
|------|------|
| 클래스명 | 전사 (Warrior) |
| 역할 | 근접 물리 딜러/탱커 |
| 성격 | 용감, 정직, 약간 단순, 열정적 |
| 나이대 느낌 | 청년 (귀엽게 표현) |
| 테마 | 용기, 힘, 보호 |

### 캐릭터 컨셉
```
전사의 핵심:
├── 용감하지만 순수한 마음
├── 몸으로 부딪치는 직진형
├── 친구를 지키려는 따뜻한 마음
├── 약간 덜렁대지만 믿음직한
└── 강해 보이지만 귀여운 갭
```

## SD 비율 스펙

### 기본 비율
```
전사 SD 비율:
├── 전체 등신: 2.5 등신
├── 머리 비율: 전체의 40%
├── 몸통 비율: 전체의 30%
├── 다리 비율: 전체의 30%
├── 어깨: 머리 너비의 1.5배 (넓게)
└── 손/발: 작고 둥글게
```

### 실제 치수 (512x768 기준)
| 부위 | 높이 | 너비 | 비고 |
|------|------|------|------|
| 머리 | 280px | 240px | 동그란 형태 |
| 몸통 | 220px | 200px | 어깨 넓게 |
| 다리 | 180px | 160px | 짧고 통통 |
| 무기 | 350px | 100px | 몸보다 큼 |

## 얼굴 디자인

### 기본 얼굴 특징
```
귀여운 전사 얼굴:
├── 눈: 크고 동그랗게, 얼굴의 40%
│   ├── 눈동자: 갈색 또는 적갈색
│   ├── 하이라이트: 크게 2개
│   └── 표정: 자신감 있지만 선량한
├── 코: 매우 작게, 점 또는 U자
├── 입: 작고 귀엽게
│   └── 기본: 자신만만한 미소
├── 볼 홍조: 양쪽 필수 (#FFB6C1)
├── 눈썹: 두껍고 짧게, 결의에 찬 모양
└── 머리카락: 짧고 뾰족, 빨강/갈색
```

### 표정 가이드
| 표정 | 눈 | 입 | 볼 홍조 | 용도 |
|------|-----|-----|---------|------|
| 기본 | 밝고 자신감 | 미소 | 연하게 | 기본 상태 |
| 전투 | 결의에 찬 | 다문 입 | 보통 | 공격 시 |
| 기쁨 | 눈웃음 | 활짝 웃음 | 진하게 | 승리, 보상 |
| 피격 | X자 또는 소용돌이 | 벌린 입 | 진하게 | 데미지 |
| 당황 | 동그랗게 | 벌린 입 | 진하게 | 실수, 놀람 |

## 의상 & 장비 디자인

### 갑옷 (Armor) - 허름한 초보 모험가 느낌

#### 상체 갑옷
```
전사 갑옷 (상체):
├── 형태: 낡고 허름한 가죽+금속 조합
├── 소재 느낌: 닳고 해진 금속, 기워진 흔적
├── 주 색상: Dusty Red (#B85450)
├── 보조 색상: Worn Gray (#6B7B8B)
├── 디테일:
│   ├── 금색 장식 없음 (장식 제거!)
│   ├── 찌그러진 부분, 긁힌 자국 표현
│   ├── 가죽 끈으로 고정된 어깨 보호대
│   └── 천 패드 일부 노출
└── 중요: 귀엽지만 초보 모험가 느낌
```

#### 하체 갑옷
```
전사 갑옷 (하체):
├── 형태: 낡은 가죽 벨트 + 천 치마
├── 허벅지 보호대: 낡은 가죽, 기워진 자국
├── 색상: 탈색된 빨강 + 회색
├── 가죽 벨트: 낡은 갈색, 단순한 버클 (금색 X)
└── 부츠: 낡은 가죽 부츠, 둥근 코, 먼지 묻은 느낌
```

### 망토 (Cape) - 선택 사항
```
전사 망토:
├── 형태: 짧은 반 망토 (어깨에서 등까지)
├── 색상: 탈색된 빨강 (#B85450)
├── 테두리: 해진 천, 올 풀림 표현
├── 움직임: 바람에 살짝 펄럭임
└── 용도: 상위 등급에서 추가 (기본 등급은 없음)
```

### 무기 (Weapon)

#### 기본 검 (Sword)
```
전사 검 디자인:
├── 크기: 캐릭터보다 살짝 큼 (귀여운 과장)
├── 형태: 낡고 닳은 넓은 검
├── 색상:
│   ├── 칼날: Worn Gray (#6B7B8B), 녹슨 부분 있음
│   ├── 손잡이: 낡은 갈색 가죽 (#8B4513)
│   └── 가드: 닳은 철 (#5B6B7B), 금색 없음!
├── 디테일: 찌그러진 곳, 칼날에 작은 이빨 자국
└── 중요: 귀엽지만 오래 사용한 느낌
```

#### 방패 (Shield) - 선택 사항
```
전사 방패:
├── 크기: 캐릭터 몸통과 비슷
├── 형태: 둥근 원형 또는 나뭇잎형
├── 색상: Steel Gray + Warm Red
├── 문양: 심플한 방패/별 마크
└── 질감: 나무와 금속 조합
```

## 포즈 바리에이션

### 1. 기본 대기 포즈 (Idle)
```
기본 포즈:
├── 자세: 정면, 살짝 비스듬히 서있음
├── 검: 한 손에 들고 어깨에 기댐
├── 다른 손: 허리에 또는 주먹 불끈
├── 표정: 자신만만한 미소
├── 시선: 정면 응시
└── 분위기: 당당하지만 귀여운
```

```
전투 포즈:

├── 검: 양손으로 앞에 들기
├── 다리: 어깨 너비, 살짝 굽힘
├── 표정: 결의에 찬, 눈빛 강렬
├── 시선: 적을 향해
└── 분위기: 진지하지만 귀여운
```

**AI 프롬프트**
```
cute chibi warrior battle stance, determined expression,
holding sword with both hands forward, knees slightly bent,
warm red armor, steel gray details, soft gold accents,
big eyes with fighting spirit, rosy cheeks,
SD proportions, hand-drawn storybook illustration,
pencil sketch watercolor style, brave but adorable
```

### 3. 공격 포즈 (Attack)
```
공격 포즈:
├── 자세: 검을 크게 휘두르는 순간
├── 검: 머리 위에서 아래로 스윙
├── 몸: 역동적으로 기울임
├── 표정: 힘차게 외치는 얼굴
├── 효과: 검 궤적 라인 (선택)
└── 분위기: 역동적, 용감한
```

**AI 프롬프트**
```
cute chibi warrior attacking pose, sword swing motion,
dynamic action pose, yelling expression,
warm red armor flying motion, gold trim sparkle,
big expressive eyes, rosy blushing cheeks,
SD proportions, storybook illustration style,
watercolor action lines, energetic and cute
```

### 4. 승리 포즈 (Victory)
```
승리 포즈:
├── 자세: 팔을 들어올리며 기뻐함
├── 검: 하늘 높이 들기
├── 표정: 환하게 웃는 얼굴, 눈웃음
├── 효과: 반짝이는 별 파티클
├── 시선: 살짝 위를 향함
└── 분위기: 기쁨, 자랑스러움
```

**AI 프롬프트**
```
cute chibi warrior victory pose, sword raised high,
celebrating happy expression, eyes closed smiling,
warm red armor, gold trim shining, sparkle effects,
rosy blushing cheeks bright, jumping slightly,
SD proportions 2.5 head ratio, storybook illustration,
watercolor style, joyful and adorable atmosphere
```

## 등급별 시각 변화

### 기본 (Basic) - Lv. 1~10
```
기본 전사:
├── 갑옷: 단순한 가죽+금속 조합
├── 색상: 채도 80%
├── 장식: 최소 (금색 테두리만)
├── 무기: 기본 철검
├── 오라: 없음
└── 파티클: 없음
```

### 숙련 (Advanced) - Lv. 11~20
```
숙련 전사:
├── 갑옷: 더 정교한 디자인
├── 색상: 채도 90%
├── 장식: 어깨 장식 추가
├── 무기: 문양이 새겨진 검
├── 오라: 미세한 붉은 기운
└── 파티클: 가끔 반짝임
```

### 마스터 (Master) - Lv. 21~30
```
마스터 전사:
├── 갑옷: 정교한 문양, 망토 추가
├── 색상: 채도 100%, 더 선명
├── 장식: 금색 장식 증가
├── 무기: 빛나는 검
├── 오라: 붉은 오라 상시
└── 파티클: 주변 반짝임
```

### 전설 (Legendary) - Lv. 31+
```
전설 전사:
├── 갑옷: 화려하지만 우아한 디자인
├── 색상: 채도 100% + 빛나는 느낌
├── 장식: 보석, 정교한 금 장식
├── 무기: 불꽃 이펙트 검
├── 오라: 강한 붉은 오라 + 불꽃
└── 파티클: 상시 불꽃/별 파티클
```

## 상태별 시각 변화

### 버프 상태
```
버프 받은 전사:
├── 오라: 녹색/금색 글로우
├── 아이콘: 머리 위 화살표↑
├── 색상: 약간 밝아짐
└── 파티클: 위로 올라가는 반짝임
```

### 디버프 상태
```
디버프 전사:
├── 오라: 보라색 연기
├── 아이콘: 머리 위 화살표↓
├── 색상: 약간 어두워짐 (차갑지 않게)
└── 표정: 살짝 힘들어 보이는
```

### 부상 상태
```
부상 전사:
├── 붕대: 머리나 팔에 작은 붕대
├── 표정: 살짝 지친 표정
├── 색상: 약간 탈색
└── 효과: 귀여운 별/새가 빙빙
```

## 전직 프리뷰

### 전사 → 버서커
```
버서커 특징 (미리보기):
├── 테마: 분노, 야성, 강력한 힘
├── 색상: 더 진한 빨강 + 검정 포인트
├── 갑옷: 거칠고 야성적 (가죽+금속)
├── 무기: 거대한 양손 도끼
├── 특징: 머리카락 더 뾰족, 이빨 살짝 보임
└── 오라: 붉은 불꽃 오라
```

### 전사 → 검사
```
검사 특징 (미리보기):
├── 테마: 기사도, 명예, 검술
├── 색상: 은색 + 파란색 포인트
├── 갑옷: 정돈된 기사 갑옷
├── 무기: 우아한 장검
├── 특징: 깔끔한 헤어, 우아한 자세
└── 오라: 은빛 검기
```
