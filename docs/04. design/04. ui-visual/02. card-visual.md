# 카드 비주얼

## 개요

이 문서는 게임 내 카드의 시각적 디자인을 정의합니다. 연필 스케치 + 수채화 스타일의 따뜻한 카드 디자인입니다.

---

## 카드 디자인 컨셉

### 핵심 스타일
```
카드 디자인 키워드:
├── 연필 스케치 기반 라인
├── 수채화 채색
├── 양피지 배경
├── 심플한 금색 테두리
├── 귀여운 SD 비율 캐릭터
├── 따뜻한 표정, 볼 홍조
└── 나무 프레임 (심플)
```

---

## 카드 유형

### 1. 손패 카드 (Hand Card)
플레이어가 사용하는 액션 카드

### 2. 캐릭터 카드 (Character Card)
플레이어/적 캐릭터를 표현하는 카드

---

## 손패 카드 디자인

### 카드 크기
| 상태 | 크기 | 비고 |
|------|------|------|
| 기본 | 100 x 140 px | 손패 기본 |
| 호버 | 120 x 168 px | 1.2배 확대 |
| 상세보기 | 200 x 280 px | 2배 확대 |

### 카드 구조
```
┌─────────────────────────┐
│ ┌───┐     ═══════════  │ ← 금색 테두리
│ │ 2 │     양피지 배경   │ ← 에너지 (좌상단)
│ └───┘                   │
│ ┌───────────────────┐   │
│ │    ~~~           │   │
│ │   연필 스케치     │   │ ← 일러스트 영역
│ │   + 수채화        │   │
│ │    ~~~           │   │
│ └───────────────────┘   │
│ ┌───────────────────┐   │
│ │    카드 이름      │   │ ← 이름 (갈색 텍스트)
│ └───────────────────┘   │
│ ┌───────────────────┐   │
│ │  효과 텍스트      │   │ ← 설명 영역
│ └───────────────────┘   │
│ ═══════════════════════ │ ← 금색 테두리
└─────────────────────────┘
```

### 프레임 스타일
```
심플한 금색 테두리:
├── 두께: 2-3px
├── 색상: Soft Gold #DAA520
├── 모서리: 부드럽게 둥글게 (8px radius)
├── 배경: Warm Parchment #FFF5E6
└── 장식: 최소한 (과하지 않게)
```

### 카드 유형별 악센트
| 유형 | 악센트 색상 | HEX | 표시 위치 |
|------|-------------|-----|-----------|
| 공격 (Attack) | Coral Red | #FF6B6B | 상단 띠 |
| 스킬 (Skill) | Sky Blue | #87CEEB | 상단 띠 |
| 파워 (Power) | Warm Gold | #FFD700 | 상단 띠 |

## 캐릭터 카드 디자인

### 카드 크기
| 종류 | 크기 | 비고 |
|------|------|------|
| 플레이어 캐릭터 | 180 x 240 px | 화면 하단 |
| 일반 적 | 150 x 200 px | 화면 상단 |
| 보스 | 200 x 267 px | 더 큼 |

### 캐릭터 카드 구조
```
┌─────────────────────────┐
│ ╔═══════════════════╗  │ ← 나무 프레임 (상단)
│ ║   양피지 배경     ║  │
│ ║                   ║  │
│ ║   ┌─────────┐    ║  │
│ ║   │ SD 캐릭터 │    ║  │ ← 캐릭터 일러스트
│ ║   │ (큰머리)  │    ║  │    (볼 홍조!)
│ ║   │ (볼홍조)  │    ║  │
│ ║   └─────────┘    ║  │
│ ╠═══════════════════╣  │ ← 금색 구분선
│ ║ ❤️❤️❤️    🛡️5   ║  │ ← HP, 블록 (하단)
│ ╚═══════════════════╝  │ ← 나무 프레임 (하단)
└─────────────────────────┘
```

### 프레임 스타일
```
심플한 나무 프레임:
├── 재질: 따뜻한 나무 질감
├── 색상: Warm Brown #8B4513
├── 두께: 8-10px
├── 모서리: 부드럽게 둥글게
├── 하이라이트: Light Wood #DEB887
└── 그림자: Dark Wood #5D3A1A
```

### HP 영역 디자인
```
하단 HP 표시:
├── 배경: 살짝 어두운 양피지
├── 하트 아이콘: 귀여운 하트 ❤️
├── 색상:
│   ├── HP 높음: Warm Red #CD5C5C
│   ├── HP 중간: Warm Orange #FFA500
│   └── HP 낮음: Coral Red #FF6B6B
├── 블록: Sky Blue #87CEEB 방패
└── 폰트: 따뜻한 갈색, 굵게
```

## 일러스트 스타일

### 연필 스케치 + 수채화
```
채색 단계:
1. 연필 스케치 라인 (따뜻한 갈색 #4A3728)
2. 수채화 베이스 컬러 (투명하게)
3. 수채화 그림자 (부드럽게)
4. 하이라이트 (흰색 남겨두기)
5. 볼 홍조 추가 (필수!)
```

### 일러스트 규칙
| 요소 | 규칙 |
|------|------|
| 라인 | 연필 스케치 느낌, 완벽하지 않게 |
| 채색 | 수채화 투명함, 겹침 표현 |
| 볼 홍조 | 모든 캐릭터에 필수 |
| 표정 | 따뜻하고 친근하게 |
| 배경 | 양피지 텍스처 |

---

## 카드 상태 표시

### 호버 상태
```
호버 시:
├── 크기: 1.2배 확대
├── 위치: 위로 30px 상승
├── 효과: 부드러운 황금빛 글로우
└── 애니메이션: 200ms ease-out
```

### 사용 가능/불가능
| 상태 | 시각 효과 |
|------|-----------|
| 사용 가능 | 기본 밝기 |
| 에너지 부족 | 살짝 어둡게 (30% overlay) |
| 선택됨 | 금색 글로우 |

---

## 애니메이션

### 손패 애니메이션
| 동작 | 시간 | 설명 |
|------|------|------|
| 드로우 | 300ms | 덱에서 부채꼴로 |
| 호버 | 150ms | 위로 상승 + 확대 |
| 사용 | 200ms | 대상으로 날아감 |
| 버림 | 200ms | 축소 + 페이드 |

### 캐릭터 카드 애니메이션
| 동작 | 시간 | 설명 |
|------|------|------|
| 대기 | 2000ms 루프 | 미세한 호흡 |
| 공격 | 400ms | 앞으로 살짝 + 복귀 |
| 피격 | 200ms | 흔들림 (살짝) |
| 방어 | 300ms | 블록 반짝임 |

## AI 생성 프롬프트

### 손패 카드 일러스트
```
[HAND CARD PROMPT]
hand-drawn pencil sketch with watercolor coloring,
single item or action illustration,
warm parchment paper background,
simple gold border frame,
storybook illustration style,
soft warm colors, cozy atmosphere,
vertical portrait 2:3 ratio,
children's book art quality

[NEGATIVE]
3D render, photorealistic, dark scary,
anime style, complex border, cold colors
```

### 캐릭터 카드 일러스트
```
[CHARACTER CARD PROMPT]
hand-drawn pencil sketch with watercolor coloring,
cute chibi SD character, big head small body,
rosy blushing cheeks on both sides,
warm friendly smile expression,
warm parchment paper background,
simple wooden frame brown tones,
soft gold trim accent,
storybook illustration style,
vertical portrait 2:3 ratio

[NEGATIVE]
realistic proportions, dark gothic scary,
3D render, anime sharp style,
cold colors, no blush, complex frame
```
