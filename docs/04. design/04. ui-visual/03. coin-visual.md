# 코인 시스템 비주얼

## 개요

코인 뒤집기는 이 게임의 핵심 메커니즘입니다. 이 문서는 코인의 시각적 디자인과 애니메이션 연출을 정의합니다.

---

## 코인의 역할

### 게임플레이에서의 의미
```
코인 = 운명의 선택
├── 앞면 (Heads): 유리한 결과, 희망, 기회
├── 뒷면 (Tails): 불리한 결과, 도전, 역경
└── 뒤집기: 운명의 순간, 긴장의 절정
```

### 감정적 목표
| 단계 | 목표 감정 | 시각적 지원 |
|------|-----------|-------------|
| 뒤집기 전 | 기대와 불안 | 코인 강조, 배경 어두워짐 |
| 회전 중 | 고조되는 긴장 | 빠른 회전, 빛 반사 |
| 결과 직전 | 절정의 서스펜스 | 슬로모션, 드라마틱 조명 |
| 결과 확정 | 희열 또는 긴장 | 색상 폭발, 피드백 효과 |

---

## 코인 기본 디자인

### 크기 및 형태
| 상태 | 크기 | 비고 |
|------|------|------|
| 기본 (손패 옆) | 48 x 48 px | 인터페이스에서 기본 |
| 뒤집기 시작 | 64 x 64 px | 약간 확대 |
| 회전 중 | 96 x 96 px | 화면 중앙, 확대 |
| 결과 표시 | 128 x 128 px | 최대 확대, 강조 |

### 코인 두께감
```
측면 표현 (회전 시 보이는 두께):
├── 두께: 직경의 8-10%
├── 색상: 앞면/뒷면의 중간 톤
├── 질감: 동전 테두리의 톱니 무늬
└── 광택: 금속 하이라이트
```

---

## 앞면 (Heads) 디자인

### 시각적 테마: 태양 / 희망 / 황금

### 색상
| 요소 | 색상 | HEX | 설명 |
|------|------|-----|------|
| 기본면 | Royal Gold | #FFD700 | 따뜻한 황금색 |
| 하이라이트 | Bright Gold | #FFF8DC | 빛 반사 영역 |
| 그림자 | Deep Gold | #B8860B | 입체감 |
| 테두리 | Dark Gold | #8B7500 | 가장자리 |
| 심볼 | Sun Orange | #FF8C00 | 태양 문양 |

### 앞면 심볼: 태양
```
태양 디자인:
├── 중심: 원형 (코인의 40%)
├── 광선: 8개의 귀여운 광선
├── 표현: 따뜻하고 친근한 느낌
├── 스타일: 손그림 스토리북 일러스트
└── 디테일: 귀여운 미소 (선택적)
```

### 앞면 텍스처
- 손그림 금화 느낌
- 부드러운 수채화 표면
- 따뜻한 황금빛 그라데이션

---

## 뒷면 (Tails) 디자인

### 시각적 테마: 달 / 도전 / 따뜻한 은빛

### 색상
| 요소 | 색상 | HEX | 설명 |
|------|------|-----|------|
| 기본면 | Warm Silver | #C0C0C0 | 따뜻한 은회색 |
| 하이라이트 | Light Silver | #D3D3D3 | 부드러운 반사 |
| 그림자 | Warm Gray | #A9A9A9 | 따뜻한 그림자 |
| 테두리 | Soft Gray | #808080 | 가장자리 |
| 심볼 | Soft Blue | #87CEEB | 달 문양 |

### 뒷면 심볼: 달
```
달 디자인:
├── 형태: 귀여운 초승달
├── 위치: 약간 오프셋 (중앙이 아닌)
├── 표현: 신비롭고 따뜻한 느낌
├── 스타일: 손그림 스토리북 일러스트
└── 디테일: 귀여운 별 2-3개 추가
```

### 뒷면 텍스처
- 손그림 은화 느낌
- 부드러운 수채화 표면
- 따뜻한 톤 유지

## 코인 뒤집기 애니메이션

### 전체 타이밍 (총 1.7초)
```
Phase 1: 상승 (0.3s)
├── 코인이 화면 중앙으로 이동
├── 크기 48px → 96px
└── 배경 어두워짐 시작

Phase 2: 회전 (0.8s)
├── Y축 기준 빠른 회전 (8-10회전)
├── 앞면/뒷면 빠르게 교차
├── 모션 블러 효과
└── 빛 반사 파티클

Phase 3: 감속 (0.4s)
├── 회전 속도 급격히 감소
├── 마지막 1-2회전 슬로모션
├── 드라마틱 일시정지 (0.1s)
└── 결과 면이 보이기 시작

Phase 4: 착지 (0.2s)
├── 결과에 따른 색상 폭발
├── 크기 96px → 128px (강조)
├── 충격파 효과
└── 배경 밝아짐 (희망) 또는 유지 (도전)
```

### 회전 중 시각 효과
```
모션 블러:
├── 회전 속도에 비례한 블러 강도
├── 최대 회전 시: 앞뒷면 구분 불가
└── 감속 시: 점차 선명해짐

빛 파티클:
├── 황금/은색 작은 빛 입자
├── 코인 주변을 맴돎
└── 결과 확정 시 방사형 확산
```

### 결과별 피드백 효과

#### 앞면 (Heads) 결과
```
색상: 황금빛 폭발
├── 중심에서 방사형 황금 광선
├── 따뜻한 오렌지/골드 파티클
├── 화면 전체 밝기 +20%
└── 짧은 승리 사운드 연상

텍스트: "HEADS!" 황금색 글씨
└── 위로 떠오르며 페이드아웃
```

#### 뒷면 (Tails) 결과
```
색상: 은빛 펄스
├── 중심에서 부드러운 은색 파동
├── 따뜻한 은빛/하늘색 파티클
├── 화면 살짝 신비로운 분위기
└── 도전적인 사운드 연상

텍스트: "TAILS" 은색 글씨
└── 아래로 내려가며 페이드아웃
```

## 손패 연동 시각화

### 코인 결과에 따른 손패 변화
```
앞면 (Heads) 시:
├── 해당 카드들 황금빛 글로우
├── "유리한 효과 발동" 표시
├── 카드 약간 위로 점프
└── 강화 수치 강조 (초록색)

뒷면 (Tails) 시:
├── 해당 카드들 은색 글로우
├── "도전 효과 발동" 표시
├── 카드 약간 흔들림
└── 페널티 수치 표시 (빨간색)
```

### 코인 의존 카드 표시
```
카드에 코인 아이콘 표시:
├── 위치: 카드 우상단
├── 크기: 16x16px
├── 표시: 미니 코인 아이콘
└── 호버 시: "코인 결과에 따라 효과 변화" 툴팁
```

## 다중 코인 시스템

### 여러 코인 뒤집기 (미래 확장)
```
2개 코인:
├── 나란히 회전
├── 동시에 또는 연속 착지
└── 조합 결과 표시

3개 이상:
├── 부채꼴 배치
├── 순차적 또는 동시 회전
└── 총 결과 집계 표시
```
